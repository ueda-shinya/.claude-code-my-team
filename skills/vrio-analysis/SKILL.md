# vrio-analysis

VRIO分析でリソース毎に4観点（V/R/I/O）でA/B/C評価し、競争優位レベル（持続的優位／一時的優位／競争均衡／競争劣位）を判定して、強化・投資すべきリソースを特定する。「主観評価」を避けるため、必ず評価基準と根拠を明文化する。

## 適用条件

- 使う場面：
  - 自社のどのリソースに投資すべきかを判断したい
  - 持続的な競争優位の源泉を特定したい
  - M&A・買収候補の評価
  - 新サービス立ち上げで「自社の何を活かすか」を見極めたい
- 使わない場面：
  - 業界の競争構造分析 → 5フォース分析（`five-forces-analysis`）
  - 強み弱み機会脅威の整理 → SWOT分析（`swot-analysis`）
  - 顧客・競合・自社の3視点 → 3C分析（`3c-analysis`）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 分析対象 | 必須 | 会社・事業・サービス単位 | 「自社のSEO支援事業」「新サービスX」 |
| 分析の目的 | 必須 | 投資判断／優位性把握／買収評価 | 「どのリソースに投資すべきか判断」 |
| 既知のリソース | 任意 | 評価候補となるリソースの仮リスト | 「ブランド／顧客基盤／技術力／チーム」 |

## 実行ステップ

### ステップ1：評価対象のリソース・能力を洗い出す

- 入力：分析対象、既知のリソース
- 処理：以下3カテゴリでリソースを洗い出す
  - **有形資産**: 設備、不動産、在庫、現金、契約済み顧客リスト
  - **無形資産**: ブランド、特許、商標、ノウハウ、データ、認証
  - **組織的能力**: 顧客対応力、開発力、営業力、文化、チームの専門性
- 各カテゴリで3〜5個ずつ、合計10個前後を候補に挙げる
- 最終的に深掘り評価する5〜8個に絞る（多すぎると評価が浅くなる）
- 出力：評価対象リソース 5〜8個

### ステップ2：V（Value 経済的価値）を評価する

- 入力：評価対象リソース
- 処理：各リソースについて以下を確認する
  - **顧客への価値**: 顧客がそのリソース（が生み出すサービス）にお金を払う意思があるか
  - **収益貢献**: そのリソースが売上・利益に直接結びついているか
  - **機会捕捉**: そのリソースで外部の機会を捉えられるか
  - **脅威対応**: そのリソースで外部の脅威に対応できるか
- A/B/Cで評価：
  - **A**: 顧客価値高 × 収益直結 × 機会・脅威に強く対応
  - **B**: 価値はあるが改善余地あり
  - **C**: 価値が低い／収益に直結していない
- 評価には根拠（受注事例、顧客の声、価格プレミアム等）を必ず添える
- 出力：V評価（リソース×A/B/C＋根拠）

### ステップ3：R（Rarity 希少性）を評価する

- 入力：V評価結果
- 処理：各リソースについて以下を確認する
  - **競合保有率**: そのリソースを競合何社が持っているか
  - **業界平均**: 業界全体に普及しているリソースか
  - **独自性**: 他社では再現できない要素を含むか
- A/B/Cで評価：
  - **A**: 競合がほぼ持っていない／業界唯一
  - **B**: 一部競合は持っているが過半数は未保有
  - **C**: 業界に広く普及している（コモディティ）
- 出力：R評価（リソース×A/B/C＋競合保有状況）

### ステップ4：I（Imitability 模倣困難性）を評価する

- 入力：V評価・R評価結果
- 処理：各リソースについて以下を確認する
  - **模倣コスト**: 競合が真似する場合のコスト（金額・時間・労力）
  - **歴史的固有性**: そのリソースの形成に長い歴史・経路依存性があるか
  - **因果関係の不明性**: 「なぜそうなっているか」が外部から見えにくいか
  - **社会的複雑性**: 文化・人間関係等、複製困難な要素を含むか
- A/B/Cで評価：
  - **A**: 模倣に5年以上かかる／そもそも模倣不可能
  - **B**: 模倣可能だが時間とコストがかかる
  - **C**: 短期間で模倣可能
- 出力：I評価（リソース×A/B/C＋模倣困難性の根拠）

### ステップ5：O（Organization 組織）を評価する

- 入力：V/R/I評価結果
- 処理：各リソースについて以下を確認する
  - **活用体制**: そのリソースを活用する組織体制が整っているか
  - **プロセス**: 活用するための業務プロセス・KPIが定義されているか
  - **権限と責任**: 活用責任者が明確か
  - **インセンティブ**: 活用を促す評価制度・報酬制度があるか
- A/B/Cで評価：
  - **A**: リソース活用体制が完全に整い、最大限引き出せている
  - **B**: 活用しているが、もっと引き出せる余地がある
  - **C**: リソースはあるが活用できていない／組織が追いついていない
- 出力：O評価（リソース×A/B/C＋活用状況）

### ステップ6：競争優位レベルを判定する

- 入力：ステップ2〜5の全評価
- 処理：4観点の組み合わせで競争優位レベルを判定する

| V | R | I | O | 競争優位レベル |
|---|---|---|---|---|
| C | - | - | - | **競争劣位**（このリソースは武器にならない） |
| A/B | C | - | - | **競争均衡**（持っていて当然のリソース） |
| A/B | A/B | C | - | **一時的競争優位**（いずれ模倣される） |
| A/B | A/B | A/B | C | **持続的競争優位（未活用）**（組織課題で機会損失中） |
| A/B | A/B | A/B | A/B | **持続的競争優位**（最強の戦略資産） |

- 各リソースに上記4レベルのいずれかを付与
- 出力：リソース毎の競争優位レベル

### ステップ7：強化・投資すべきリソースと戦略を特定する

- 入力：ステップ6の判定結果
- 処理：以下のロジックで戦略を導出する
  - **持続的競争優位（未活用）リソース**: 最優先で組織体制を強化（O改善）
  - **一時的競争優位リソース**: 模倣困難性（I）を高める投資（特許・ノウハウ秘匿・参入障壁構築）
  - **競争均衡リソース**: 希少性（R）を高める方向で差別化を検討
  - **競争劣位リソース**: そのリソースに投資しない／撤退も視野
- 各戦略に「KPI」「優先順位」「投資額・期間」を必ず付ける
- 短期（〜3ヶ月）／中期（3〜12ヶ月）／長期（1年以上）に振り分ける
- 出力：強化戦略 2〜3個（KPI・優先順位・期間付き）

### ステップ8：VRIO分析シートを完成させる

- 入力：ステップ1〜7の全成果物
- 処理：以下のテンプレートに統合する
- 出力：VRIO分析完成シート

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 評価対象リソースが10個以上 | 5〜8個に絞る | 多すぎると評価が浅くなる |
| A評価ばかり | 自己評価バイアス。基準を厳格にして再評価 | 楽観バイアスは戦略を狂わせる |
| C評価ばかり | 過度な自己卑下。長所を再点検 | 過度な悲観も戦略を狂わせる |
| 評価根拠が「感覚」「主観」 | 数値・事例・第三者の声で必ず裏付ける | 主観のVRIOは形骸化 |
| 持続的競争優位（未活用）リソースがある | 組織体制強化を最優先タスク | 機会損失コストが大きい |
| 持続的競争優位リソースが複数 | 競合への防御策（特許・契約等）を強化 | 競合が必ず狙ってくる |
| すべて競争均衡・競争劣位 | 新規リソース獲得・育成を最優先 | 現状リソースでは戦えない |
| 評価が組織内で割れる | 複数評価者で評価し平均を取る | 主観性は複数視点で減らす |
| SWOT分析の強みと矛盾 | SWOT・3C分析と併用して整合性確認 | 単独評価は偏る |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| すべてA評価で自己満足 | 厳格な基準で再評価（競合と並べて比較） | 楽観バイアスは戦略を狂わせる |
| 評価根拠が「感覚」 | 数値・事例・受注事例・第三者の声で裏付け | 主観VRIOは形骸化 |
| Vだけで判定して終わる | 必ず4観点（VRIO）すべてを評価する | Vのみは「持っている」だけで戦略にならない |
| O（組織）を省略 | 必ずOを評価する。Oが弱いと他が強くても活かせない | Oは多くの企業の盲点 |
| 持続的優位リソースを発見して満足 | 防御策（特許・契約・契約期間延長）を必ず立てる | 持続的優位は競合が必ず狙ってくる |
| 単独実施 | SWOT・3C・5フォースと併用 | 内部リソース単独では戦略にならない |
| 1度作って放置 | 半年〜1年で再評価 | 模倣・組織・市場は変化する |
| 競合と比較せずに評価 | 必ず競合の同種リソースと並べて比較 | 比較なしの希少性評価は楽観バイアスの温床 |
