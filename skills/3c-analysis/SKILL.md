# 3c-analysis

3C（Customer / Competitor / Company）の3視点で市場環境を分析し、クロス3CでKSF（Key Success Factor）と戦略仮説を導出する。「現状把握止まり」を防ぐため、必ず戦略仮説の言語化まで進める。

## 適用条件

- 使う場面：
  - 新規事業の参入領域を決める前
  - 既存事業の戦略見直し・差別化ポイントの再確認
  - ホワイトスペース（競合不在領域）を探したい
  - KSF（重要成功要因）を明確化したい
- 使わない場面：
  - 内部の強み弱みと外部の機会脅威の整理 → SWOT分析（`swot-analysis`）
  - マクロ外部環境のみ → PEST分析（`pest-analysis`）
  - 競争構造の深掘り → 5フォース分析（`five-forces-analysis`）
  - 価値提供の言語化のみ → バリュープロポジション（`value-proposition`）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 分析対象 | 必須 | 事業・サービス単位 | 「自社のコールセンター事業」「新規SaaS」 |
| 分析の目的 | 必須 | 新規参入／戦略見直し／差別化 | 「KSFを発見して差別化戦略を作る」 |
| ターゲット候補 | 任意 | 顧客セグメント仮説 | 「中小企業 / 地方企業」 |
| 既知の競合情報 | 任意 | 競合一覧と特徴 | 「競合A社=価格訴求、B社=AI訴求」 |

## 実行ステップ

### ステップ1：Customer（市場・顧客）を分析する

- 入力：分析対象、ターゲット候補
- 処理：以下4項目を整理する（最初に「顧客」から始めるのがセオリー。自社や競合から始めると主観に偏る）
  - **ターゲット市場**: 市場規模・成長率・トレンドを数字で記述
  - **顧客のニーズ**: 顕在ニーズ（言語化されているもの）と潜在ニーズ（行動から読めるもの）を分けて記述
  - **顧客の行動・購買動機**: なぜその商品を買うか／どこで知るか／意思決定者は誰か
  - **未解決のニーズ**: 既存サービスでは満たされていないもの（ホワイトスペース候補）
- データソース：市場調査レポート、自社の受注ヒアリング、競合のレビュー（Amazon・Google等）
- 出力：顧客分析シート（ニーズ5〜8項目）

### ステップ2：Competitor（競合）を分析する

- 入力：既知の競合情報、ステップ1の顧客分析
- 処理：競合3〜5社を選び、各社で以下を整理する
  - **戦略**: その会社が顧客に何を訴求しているか（1文）
  - **強み**: 価格／品質／ブランド／チャネル／規模等
  - **弱み**: 顧客のレビューや失注理由から逆算
  - **市場シェア**: 概算でよい（圧倒的1位／中位／ニッチ）
  - **訴求軸**: 顧客に対して打ち出している主要メッセージ（5項目以内）
- 競合選定ルール：
  - 直接競合（同じ市場・同じ顧客）3社必須
  - 代替競合（顧客のニーズを別の手段で満たす）2社推奨
- 出力：競合分析シート（3〜5社×訴求軸）

### ステップ3：Company（自社）を分析する

- 入力：ステップ1〜2の結果
- 処理：以下3項目を「顧客の喜びにどう結びつくか」の視点で整理する
  - **強み**: 顧客が選ぶ理由（受注時のヒアリング・営業現場の声を引用）。「自慢の工場」「すごい技術」だけでは×。それが顧客にとって何の価値かまで書く
  - **弱み**: 失注理由・社内不満から逆算
  - **競争優位性**: 競合が真似できない／真似しにくい資産
- **創造的な視点**: 強みが見えない場合、「今は存在しないが、自社だけが今後創り出せるもの」も検討する
- **重要な注意**: 「すごい技術がある」止まりにせず、「その技術がどう消費者の喜びに結びつくか」まで書く
- 出力：自社分析シート（強み3〜5項目、根拠つき）

### ステップ4：クロス3Cで4要素を導出する

- 入力：ステップ1〜3の結果
- 処理：3つの輪を重ね合わせて以下を抽出する
  - **市場×自社（ターゲット設定）**: 自社の強みが顧客のニーズに刺さる領域はどこか
    - 出力例：「中小企業の電話対応外部委託ニーズ × 自社の柔軟料金体系 → 月商500万円以下の中小企業向け」
  - **市場×競合（競合の提供価値を知る）**: 競合は顧客に何を提供しているか
    - 出力例：「大規模コールセンターの安定性・低価格・AI自動応答」
  - **自社×競合（ホワイトスペースを探す）**: 自社が提供できて競合が提供していない領域
    - 出力例：「業界専門オペレーター × 24時間365日対応 × 当日サービス開始 → 即応性×専門性のホワイトスペース」
  - **3者交点（KSF）**: 3つの輪の中心に来る、事業成功に不可欠な要因
    - 出力例：「業界別カスタマイズ対応 × トライアル提供 × 当日対応 → KSF = 即応性×専門性×低参入障壁」
- 出力：クロス3C ベン図（4要素を整理）

### ステップ5：戦略仮説を3つに絞り込む

- 入力：ステップ4のクロス3C
- 処理：KSFを満たす戦略仮説を3つ立案する
  - **戦略仮説1（攻め）**: 自社の強みを活かしたホワイトスペースへの参入戦略
  - **戦略仮説2（差別化）**: 競合と差別化するための具体施策（価格・サービス・チャネル等）
  - **戦略仮説3（拡大）**: 既存顧客の深耕／新規セグメント開拓／代理店経由等の拡大戦略
- 各仮説に「KPI」「期間軸」「リソース概算」を必ず付ける
- 出力：戦略仮説3つ（KPI・期間軸・リソース付き）

### ステップ6：3C分析シートを完成させる

- 入力：ステップ1〜5の全成果物
- 処理：以下のテンプレートに統合する
- 出力：3C分析完成シート

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 自社分析から始めたくなる | 必ずCustomer（顧客）から始める | 自社・競合から始めると主観バイアスが強くなる |
| 自社の強みに「すごい技術」「自慢の工場」 | 「それが顧客の何の喜びに結びつくか」まで書く | 顧客視点なき強みは強みではない |
| 競合が「自社のみ」と感じる | 代替競合（別手段で同じニーズを満たすもの）を必ず2社挙げる | 競合は同業だけではない |
| 顧客のニーズが「価格が安いこと」だけ | 顕在ニーズ（言語化）と潜在ニーズ（行動）を分けて再分析 | 価格訴求は最終手段。差別化が薄くなる |
| KSF が3者交点に来ない | 3つの分析を再点検する | KSF不在の戦略は失敗する |
| ホワイトスペースが見つからない | 顧客のセグメントを切り直す（業種別・規模別・地域別） | ホワイトスペースは必ず存在する。見えないだけ |
| 戦略仮説が「がんばる」「強化する」 | 動詞＋目的語＋数値で書き直す | 抽象戦略は実行不能 |
| 競合情報がない／少ない | レビュー（Amazon/Google/X）と価格表を必ず確認 | 「分からない」で済ませない |
| 自社の強みが本当に見えない | 「今後、自社だけが創り出せるもの」として未来形で記述 | 強み不在は分析不足 |

### 記入例（コールセンターサービスの例）

```
## 1. Customer
- 顕在ニーズ：電話対応の品質向上・業務効率化、本業集中、コスト削減、夜間休日対応、繁忙期対応
- 未解決ニーズ：業界知識を持った専門オペレーターへのニーズが既存サービスでは満たされにくい

## 2. Competitor
- 大手A社：大規模・低価格・AI自動応答／弱み：業界特化対応薄い
- 中堅B社：多言語対応・実績豊富／弱み：当日対応不可

## 3. Company
- 強み：2週間無料トライアル、月1.5万円+従量課金、当日サービス開始、業界専門オペレーター、24h365日

## 4. クロス3C
- 市場×自社：中小企業の品質訴求ニーズ × 専門オペレーター → 業界特化型コールセンター
- 自社×競合：トライアル+柔軟料金+当日対応 → 即応性×専門性のホワイトスペース
- KSF：「即応性×専門性×低参入障壁」

## 5. 戦略仮説
1. 攻め：トライアルファーストで参入障壁低減、3ヶ月で30社獲得
2. 差別化：業界別カスタマイズ対応で高単価維持、平均月単価+30%
3. 拡大：代理店ホワイトラベル提供で6ヶ月以内に代理店5社契約
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 自社分析から始める | Customer（顧客）から始める | 主観バイアスを避ける |
| 「すごい技術がある」止まり | 顧客の何の喜びに結びつくかまで書く | 顧客視点なき強みは無価値 |
| 競合を同業他社に限定 | 代替競合（別手段）も必ず含める | 真の競合は別業界にもいる |
| ニーズを「価格が安いこと」だけにする | 顕在ニーズと潜在ニーズを分ける | 価格依存の戦略は脆い |
| KSFを書かない／3者交点と無関係 | 3つの分析の重なりからKSFを必ず抽出 | KSFなき戦略は方向性を失う |
| ホワイトスペースが見つからないと諦める | 顧客セグメントを切り直す | ホワイトスペースは必ず存在する |
| 戦略仮説が「がんばる」 | 動詞+目的語+数値で書く | 抽象戦略は実行不能 |
| 「強みが見えない」で停止する | 「未来形」で創造的に記述 | 強み不在は分析不足 |
| 一度作ったら見直さない | 環境変化があるたびに再作成 | 市場・競合は動き続ける |
