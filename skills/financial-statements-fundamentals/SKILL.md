# financial-statements-fundamentals スキル

財務三表（BS／PL／CF）の基本構造・関係性・活用方法を体系的に理解し、自社の財務状況を診断する。事業戦略スクール「知足」が定義する「中小企業経営者の死活問題＝資金繰り」の観点を中核に据え、BS/PL/CF表を一気通貫で読み、運転資金の先読みと財務体質改善まで一貫支援する。

## 適用条件

- このスキルを使う場面：
  - BS／PL／CFの基本構造と読み方を理解したい
  - 自社の財務状況（健全性・収益性・資金繰り）を診断したい
  - 運転資金が足りているか・何ヶ月分あるかを把握したい
  - 黒字倒産のリスクをチェックしたい
  - 財務体質を改善したい（売上・利益率・固定費）
  - 労働生産性・労働分配率を診断したい
- このスキルを使わない場面：
  - M&A戦略の検討 → `ma-strategy-basics` を使う
  - デットファイナンス／エクイティファイナンス選択 → `financing-strategy` を使う
  - 個別プロジェクトの収支管理・契約 → `project-finance-contract` を使う
  - KGI/KPIの設計 → `kgi-kpi-kai-design` を使う
  - 売上シミュレーション（YonY）→ `yony-sales-simulation` を使う

## 実行ステップ

### ステップ1：BS（貸借対照表）の構造を確認する

**BS = ある一定時点の企業の財務状況を表すもの**

| 区分 | 内訳 | 説明 |
|---|---|---|
| 資産 | 流動資産 | 1年以内に現金化できる資産 |
| 資産 | 固定資産 | 長期間にわたり使用する資産 |
| 資産 | 繰延資産 | 支出済みで将来効果がある費用 |
| 負債 | 流動負債 | 1年以内に支払う必要がある負債 |
| 負債 | 固定負債 | 1年を超えて到来する負債 |
| 純資産 | 純資産 | 資産から負債を差し引いた自己資本 |

**確認ポイント**：
- 資産と負債のバランス（健全な財務状態か）
- 流動資産と流動負債の比率（短期的な支払い能力＝流動比率）

### ステップ2：PL（損益計算書）の4つの利益を確認する

**PL = 企業の一定期間における収益と費用を記録し、利益を示すもの**

**基本式**：収益 − 費用 = 利益

**4つの利益**：

| # | 利益 | 計算式 | 意味 |
|---|---|---|---|
| ① | 売上総利益（粗利益） | 売上高 − 売上原価 | 事業の基本的な収益力を示す |
| ② | 営業利益 | 売上総利益 − 販管費 | 本業の収益性を示す（マーケや運営の効率性が影響） |
| ③ | 経常利益 | 営業利益 + 営業外損益 | 本業と本業以外を含めた通常の収益力 |
| ④ | 当期純利益 | 経常利益 + 特別損益 − 法人税等 | 最終的に企業に残る利益（配当・内部留保の源泉） |

**3つの経営成績の階層**：
- 会社の本業の経営成績（売上総利益・営業利益）
- 会社の通常の経営成績（経常利益）
- 最終的な経営成績（当期純利益）

### ステップ3：CF（キャッシュフロー計算書）の3区分を確認する

**CF = 一定期間における現金の流れを記録し、企業の資金繰りを明らかにするもの**

**基本式**：期首の現金残高 + 各キャッシュフロー = 期末の現金残高

**3大キャッシュフロー**：

| 区分 | 内容 | 例 |
|---|---|---|
| 営業活動によるCF（Operating CF） | 本業での現金収支 | 売上現金 − 仕入支出 |
| 投資活動によるCF（Investing CF） | 設備投資や資産購入・売却による現金収支 | 設備購入支出、有価証券売却収入 |
| 財務活動によるCF（Financing CF） | 資金調達や借入返済、配当金支払いによる現金収支 | 借入金収入、配当金支出 |

**確認ポイント**：
- 営業活動でプラスのキャッシュフローが維持されているか
- 設備投資や資金調達の健全性
- 3大キャッシュフローは必ず見る

### ステップ4：BS／PL／CFの関係性を理解する

**3表の連携**：
- **PLがBSに反映される**：PLの純利益は、BSの「純資産」に加算される
- **CFがBSの現金残高を補完する**：CF計算書の現金収支は、BSの現金残高と一致する

**活用方法**：
| 表 | 何を見るか |
|---|---|
| BS | 資産や負債の状況を把握し、財務の安定性を評価 |
| PL | 売上や利益率を分析し、収益性を評価 |
| CF | 現金の流れを確認し、資金繰りの健全性をチェック |

### ステップ5：資金繰り8チェックリストで診断する

**中小企業の社長が毎月直面するのが「キャッシュフロー問題」。望ましいのは運転資金が6か月以上、絶対安定を意図するなら18ヵ月以上（1年半）。**

| # | チェック項目 | 確認内容 |
|---|---|---|
| 1 | いくら販管費を使うか？ | 人件費／原価／販促費／オフィス賃料・水道光熱費／借入利息を「固定費」と「変動費」に分けて把握 |
| 2 | いくら売上をたてられるか？ | 「楽観」「悲観」両面の売上予測。月商ベース、シーズン要因、契約継続率を加味した複数シナリオ。BtoBは受注〜入金リードタイム注意 |
| 3 | 売上の支払いサイトは？ | 現金商売は即時、請求書払い（BtoB）は30日後・60日後も当たり前。サイトが長いと黒字でも資金ショートリスク |
| 4 | 毎月の返済額は？ | 借入金（日本政策金融公庫・銀行等）の元本＋利息合計。複数借入は月ごとキャッシュアウト一覧化。リスケは信用毀損リスクあり |
| 5 | 毎月の社会保険はいくら？ | 健康保険・厚生年金・雇用保険の会社負担分。給与総額の約15〜17%が目安。従業員数増で固定費比重増 |
| 6 | 法人税はいくらになる見込みか？ | 法人税・住民税・事業税を事前試算。黒字転換直後や2期目以降は税負担を見落としがち。中間納税スケジュール注意 |
| 7 | 減価償却対象資産の実際のキャッシュアウトはいつ・いくら？ | 減価償却は会計上の費用計上で現金は過去に出ている。新たな設備投資は未来のキャッシュアウト予測必要 |
| 8 | 毎月いくら残るか？（または、いくら赤字になるか？） | 月ごとの「収入−支出＝純CF」がプラスかマイナスか。赤字月は何ヶ月先までに資金が尽きるか試算し、調達・コスト削減の打ち手検討 |

**重要ポイント**：
- POINT01：キャッシュフローで見るべきは、売上と原価・経費だけではない
- POINT02：PLで見れば利益が出ていても、銀行口座にお金がなく、債務不履行になれば会社は倒産する（黒字倒産）

### ステップ6：支払いサイトと運転資金の仕組みを理解する

**支払いサイト** = 入金までの期間と、支払いまでの期間の期間差

**例（広告代理店）**：運用する広告費が先出し、運用利益は当月請求・翌月末支払い → 約2ヶ月入金が後になる

**運転資金の仕組み**：
```
仕入れ → 棚卸資産 → 売掛金 → 受取手形 → 現金（売掛側）
仕入れ → 買掛債務 → 支払手形 → 現金（買掛側）
```

- **棚卸資産回転期間**：仕入から販売まで
- **売掛債権回転期間**：販売から現金回収まで
- **買掛債務回転期間**：仕入から現金支払いまで
- **必要運転資金サイトの差**：売掛回収サイト − 買掛支払いサイト

### ステップ7：財務体質改善の3要素を導入する

**収益構造の改善＝資金繰り調整より優先で取り組むべき経営者・事業責任者の最優先事項**

PLの基本構造：
```
売上高 − 変動費 = 限界利益（付加価値） − 固定費 = 経常利益
```

**改善3要素**：
| 要素 | 内容 |
|---|---|
| 売上高の拡大 | 既存事業の伸長／新規事業／新商品 |
| 限界利益率の向上 | 変動費を下げる（仕入交渉・原価率改善） |
| 固定費の削減・変動費化 | 固定費削減（家賃・人件費見直し）／変動費化（業務委託・外注活用） |

### ステップ8：労働生産性・労働分配率を診断する

| 指標 | 計算式 | 意味 |
|---|---|---|
| 労働生産性 | 付加価値 ÷ 従業員 | 一人当たりが生み出す付加価値 |
| 労働分配率 | 人件費 ÷ 付加価値 | 付加価値に対する人件費の割合 |

**判断基準**：
- 労働生産性が低い → 一人当たりの稼ぐ力を上げる施策（教育・自動化・付加価値の高い業務へシフト）
- 労働分配率が高すぎる → 人件費負担が重い（生産性向上か人員適正化）
- 労働分配率が低すぎる → 人件費投資不足の可能性（採用・処遇改善検討）

### ステップ9：4課題分析STEPで自社財務を点検する

| STEP | 確認項目 |
|---|---|
| STEP01 BS | 資産と負債のバランスは健全か？流動負債に対する流動資産の割合は適切か？ |
| STEP02 PL | 売上や利益が継続的に成長しているか？原価や人件費の割合は適正か？ |
| STEP03 CF | 営業活動によるキャッシュフローがプラスか？投資や借入金の動きは企業戦略と一致しているか？ |
| STEP04 資金繰り | 運転資金は6ヶ月以上あるか？ 18ヶ月（1年半）に近づけられるか？ |

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 運転資金が6ヶ月未満 | 即時調達検討（融資／資本／売掛回収早期化） | 中小企業の死活ライン |
| 運転資金が18ヶ月以上 | 絶対安定圏。投資・成長投資余力あり | 業界変動・景気悪化に耐えられる水準 |
| PL黒字だがCF営業活動マイナス | 黒字倒産リスク。売掛回収・在庫管理を即見直し | 利益と現金の乖離が事業継続の最大リスク |
| 流動比率（流動資産÷流動負債）<100% | 短期支払い能力に問題あり、即時改善必要 | 流動負債を返せない状態 |
| 営業利益がマイナス | 本業の収益構造に問題あり、再設計が最優先 | 営業外で補填しても持続不可能 |
| 限界利益率が業界平均を下回る | 変動費（仕入・原価）見直し優先 | 売上拡大しても利益が残らない構造 |
| 固定費比率が高すぎる | 変動費化（業務委託・外注）検討 | 売上変動への耐性を上げる |
| 労働分配率が業種平均超過 | 人件費負担が重く利益圧迫 → 生産性向上 or 人員見直し（業種別目安：小売・サービス50-60%／製造業40-50%／IT・コンサル60-70%／飲食60-65%） | 業種により最適水準が大きく異なるため、必ず自社業種の平均と比較 |
| 労働分配率が業種平均を大きく下回る（目安：業種平均-15%以下） | 人件費投資不足の可能性 → 採用・処遇改善検討 | 優秀人材流出リスク |
| 借入返済を1社だけ見て安心している | 全借入を月ごとキャッシュアウト一覧化 | 複数借入の重畳でショート発生 |
| 「うちは現金商売だから資金繰り問題ない」 | 社会保険・法人税・固定費を含めた資金繰りチェック | 現金商売でも法人税中間納税で資金ショート例多数 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| PLだけ見て「黒字だから安心」と判断 | BS／PL／CFを必ず3表セットで確認 | 黒字倒産（PL黒字でCFマイナス）の最大原因 |
| 売上目標だけ立てて、支払いサイトを考慮しない | 売上回収サイトと支払いサイトの差を必ず計算 | 売上立っても入金が遅いと資金ショート |
| 1社からの借入だけ見て返済計画を立てる | 全借入を月ごとキャッシュアウト一覧化 | 複数借入の重畳でショート発生 |
| 法人税を事前試算せず納税月に慌てる | 黒字転換期から法人税を月割で予算化 | 黒字転換直後の倒産パターン頻出 |
| 減価償却を「現金が出ていく費用」と勘違い | 減価償却＝過去に出た現金の費用化（混同しないこと） | 投資判断の基本誤解 |
| 「運転資金は3ヶ月あれば十分」と楽観判断 | 6ヶ月以上を目標、絶対安定は18ヶ月以上 | 業界変動・景気悪化耐性が不足する |
| 売上拡大だけで利益改善しようとする | 売上拡大＋限界利益率向上＋固定費削減の3要素同時 | 売上拡大しても利益率低いと労働だけ増えて儲からない |
| 労働分配率を見ずに人件費を増やし続ける | 労働分配率と労働生産性を同時に追跡 | 人件費投資が成果につながらないと経営圧迫 |
| 業種特性を考慮せず一般論で資金繰りを判断 | 広告代理店2ヶ月／製造業3ヶ月等、業種別の支払いサイト前提 | 業種特有の資金繰り構造を見落とす |

- `yony-sales-simulation`：売上シミュレーション（YonY）で予測精度を上げる
- `growth-phase-strategy`：企業成長フェーズ別の財務戦略選択
- `medium-term-business-plan`：中長期計画における財務予測
- `meeting-cadence-design`：月次・四半期の財務レビュー会議体設計

## 5状態出力契約

このスキルは外部API・サブプロセス実行・外部設定依存のいずれも持たないため、CLAUDE.md「スキル出力の5状態契約」の対象判定基準には該当しない。ただし以下のケースで明示マーカーを使うこと：

- `[OK:0件]`：自社の財務諸表データが提供されず、仕組み解説のみ実施した場合
- `[SKIP]`：診断に必要な前提（最低でもBS/PLいずれか）が欠落しており、フレームワーク説明のみで終了した場合

外部API・サブプロセスを持たないため、`[FAIL]`（試行後の失敗）は通常発生しない。
