# pmf-journey

フィットジャーニー5段階で自社事業の現フィット段階を判定し、次フィットを達成するための具体的アクションを設計する。「PMFを目指す＝1段階の議論」になりがちな所を、5段階に分解して順次達成することで失敗リスクを下げる。

## 適用条件

- 使う場面：
  - 新規事業がPMFに到達したか診断したい
  - 「成長が鈍化している」「離脱率が高い」原因を構造的に分析したい
  - スケール投資すべきタイミングか判断したい
  - 「次にやるべきこと」が見えなくなっているとき
- 使わない場面：
  - 普及プロセスの分類 → イノベーター理論（`innovator-theory`）
  - 価値提案の言語化 → バリュープロポジション（`value-proposition`）
  - 顧客セグメントの設計 → STP分析（`stp-analysis`）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 分析対象 | 必須 | プロダクト・サービス | 「自社のSaaS」「新規BtoBサービス」 |
| 分析の目的 | 必須 | フェーズ判定／原因分析／投資判断 | 「PMF到達したか診断」 |
| 既知の数値 | 任意 | リテンション／NPS／成長率／LTV/CAC等 | 「3ヶ月継続率60%」「LTV/CAC=2.0」 |

## 実行ステップ

### ステップ1：5つのフィット段階を順番に診断する

- 入力：分析対象、既知の数値
- 処理：以下5段階を順番に診断する。各段階の達成サイン（数値指標）を満たしているか確認する

#### STEP 1: Problem-Solution Fit（PSF：解決すべき課題が本当に存在するか？）
- 達成サイン：
  - ユーザーが「これは本当に困っている」と感じている
  - 課題を解決することにお金を払う意欲がある
- 検証方法：顧客インタビュー20件以上、ペルソナ明確化、市場調査

#### STEP 2: Product-Solution Fit（課題を解決できるプロダクトが作れるか？）
- 達成サイン：
  - ユーザーがプロダクトを繰り返し使っている
  - 「このサービスがなくなると困る」と答える割合 **40%以上**（40%ルール）
- 検証方法：MVP開発、テストユーザーへのリリース、継続率測定

#### STEP 3: Product-Market Fit（PMF：プロダクトが市場に受け入れられているか？）
- 達成サイン：
  - 口コミでのユーザー獲得が増える（**紹介率 30%以上**）
  - **3ヶ月後の継続率 70%以上**
  - CAC（顧客獲得コスト）がLTV（顧客生涯価値）に対して健全な比率
  - NPS スコア **50以上**
- 検証方法：有料ユーザー増加、自然成長、価格上げても離脱しないか

#### STEP 4: Go-to-Market Fit（GTMフィット：マーケと販売戦略が機能しているか？）
- 達成サイン：
  - 広告なしでも一定数の顧客が自然流入
  - **LTV/CAC比が 3倍以上**
  - マーケティングROIがプラス
- 検証方法：最適なチャネル特定（広告/SEO/SNS等）、営業プロセス標準化

#### STEP 5: Scale Fit（スケールフィット：事業を大きく拡大できるか？）
- 達成サイン：
  - **ARR（年間売上成長率）20%以上**
  - **口コミ経由の成長 50%以上**
  - 競争優位性が確立、市場ポジションが明確
- 検証方法：組織体制、新規市場開拓、ネットワーク効果活用

- 各段階で「達成済」「未達成」「判定不能（データなし）」を判定する
- 出力：5段階の達成判定表

### ステップ2：現在の段階を特定する

- 入力：ステップ1の判定結果
- 処理：以下のロジックで現フェーズを特定する
  - **直前まで全て達成済 + 次の段階が未達成** → その「未達成」の段階が現フェーズ
  - **複数飛ばして達成済の段階がある** → 最初の未達成段階が現フェーズ（飛ばし達成は錯覚の可能性大）
  - **全て達成済** → スケールフェーズ後の継続改善
- 例：PSF達成、Product-Solution Fit達成、PMF未達 → 現フェーズ = PMF獲得期

### ステップ3：PMF未達のサインをチェックする

- 入力：分析対象の現状
- 処理：以下5つのPMF未達サインに該当するか確認する
  - **顧客の解約率が高い**: 試用後すぐ解約。ニーズに合っていない
  - **成長が鈍化**: 新規獲得が難しい。CAC > LTV
  - **顧客フィードバックがネガティブ**: 「なくなると残念」と感じない
  - **市場需要が薄い**: 関心低下、競合がより良い製品を提供
  - **ユーザーエンゲージメントが低い**: 頻度が低い、友人に勧めない
- 該当サインがあれば、現フェーズで取り組むべき課題を特定
- 出力：PMF未達サインのチェック結果

### ステップ4：次にやるべきアクションを特定する

- 入力：ステップ2の現フェーズ判定、ステップ3のサイン
- 処理：現フェーズに応じた具体アクションを以下から選定する

| 現フェーズ | 次にやるべきアクション |
|---|---|
| PSF未達 | 顧客インタビュー＋ペルソナ設定 → 課題仮説の検証 |
| Product-Solution Fit未達 | MVP開発 → ユーザー行動データで改善（40%ルール達成まで） |
| PMF未達 | 継続率/解約率分析 → 価格テスト → 紹介率改善 |
| GTMフィット未達 | チャネル最適化 → 営業プロセス標準化 → LTV/CAC改善 |
| スケールフィット未達 | 組織体制構築 → 新規市場開拓 → バイラル設計 |

- 各アクションに「具体施策」「KPI」「期間」「優先順位」を付ける
- 出力：次フィット達成アクションプラン

### ステップ5：典型的ミスを回避する

- 入力：現フェーズ、アクションプラン
- 処理：以下の典型的ミスに該当しないか確認する
  - **PMF未達のままスケール投資**: 漏れたバケツに水を注ぐ。PMF達成後にスケール投資する
  - **PSF未達のままMVP開発**: 解決すべき課題が間違っているとMVPで何を作っても無駄
  - **数値指標を見ずに「感覚」でPMF達成と判断**: 40%ルール、紹介率、継続率を必ず数値で確認
  - **1段階達成後に過去段階の指標が悪化**: 段階を進めても過去段階の指標を維持する必要がある
- 該当する場合は警告フラグを立て、優先タスクを再設計
- 出力：典型ミスチェック結果

### ステップ6：PMF/フィットジャーニーシートを完成させる

- 入力：ステップ1〜5の全成果物
- 処理：以下のテンプレートに統合する
- 出力：PMF/フィットジャーニー完成シート

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 40%ルール未達（「なくなると困る」<40%） | Product-Solution Fit未達。MVP改善継続 | 40%ルールはPMFの最重要指標 |
| 3ヶ月継続率<70% | PMF未達。継続率改善が最優先 | 継続なきPMFはあり得ない |
| LTV/CAC < 1 | ビジネスモデル破綻。ピボット検討 | 顧客獲得するほど赤字 |
| LTV/CAC = 1〜3 | GTMフィット未達。CAC削減 or LTV改善 | 健全な比率は3倍以上 |
| LTV/CAC > 3 | GTMフィット達成済。スケール投資検討 | スケールフィット段階に進める |
| ARR < 20% | スケールフィット未達。成長加速施策 | スケール段階の最低成長率 |
| 「PMF達成感」あるが数値見ていない | 必ず40%ルール・紹介率・継続率を測定 | 主観のPMF判定は危険 |
| 1段階飛ばして達成済 | 飛ばし達成は錯覚。前段階を再点検 | フィットは順番が重要 |
| PSF未達でMVP開発に進む | 課題仮説の検証に戻る | 課題が間違っているとMVP無駄 |

### 記入例（BtoB SaaSの例・要約）

```
判定フェーズ：PMF未達（Product-Solution Fit達成済、PMF獲得期）

5段階診断：
- PSF：達成済（顧客100社インタビューで課題確認）
- Product-Solution Fit：達成済（40%ルール=45%、3ヶ月継続=60%）
- PMF：未達（紹介率15%、3ヶ月継続60%、NPS=35）← 現在ここ
- GTMフィット：未達（LTV/CAC=1.5）
- スケールフィット：未達

PMF未達サイン：紹介率<30%、NPS<50→「友人に勧めない」エンゲージメント低い

次フィット達成アクション：
1. 紹介機能・Refer-a-friend強化（紹介率30%目標、3ヶ月）
2. オンボーディング改善で3ヶ月継続70%へ（LTV改善、3ヶ月）
3. NPS改善のためカスタマーサクセス強化（スコア50目標、6ヶ月）

典型ミスチェック：「LTV/CAC=1.5でスケール検討」 → 警告：PMFと GTM両方未達でスケール投資はNG
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 全フィットを「PMFを目指す」と1段階で議論 | 5段階に分解して順次達成 | 一気のPMF達成は失敗確率が高い |
| PMF未達でスケール投資 | PMF達成（40%ルール+継続率+紹介率）後にスケール | 漏れたバケツに水 |
| 「感覚」でPMF達成と判断 | 数値指標で必ず判定（40%/30%/70%/3倍） | 主観のPMF判定は危険 |
| 1段階飛ばして達成済と勘違い | 全段階を順番に確認 | 飛ばし達成は錯覚 |
| 1度PMF達成したら永遠と思う | 環境変化で再評価必要 | 市場・競合は変わる |
| 達成サイン指標を覚えない | 40%/30%/70%/3倍/20% を覚える | 数値が判定の基準 |
| ピボットを検討しない | LTV/CAC<1ならピボット検討必須 | ビジネスモデル破綻 |
| 過去段階の指標悪化を放置 | 全段階の指標を継続監視 | 後退すると次に進めない |
