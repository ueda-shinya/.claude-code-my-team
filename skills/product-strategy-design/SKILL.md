# product-strategy-design

コトラーのProduct 3要素（コア／形態／付随機能）と3つの分類軸で、自社製品の戦略上の重点ポイントを構造化する。マーケティングミックス4PのうちProduct領域に特化。

## 適用条件

- 使う場面：
  - 自社製品・サービスの構成要素を体系的に分解したい
  - 製品タイプ別（耐久財／サービス／消費財／専門品等）に重点戦略を変えたい
  - 商品カテゴリマトリクスで「機能訴求 vs 情緒訴求」のどちらに振るか判断
  - 新製品立ち上げ時のプロダクト戦略設計
- 使わない場面：
  - 4P全体を俯瞰したい → `marketing-mix-4p4c`
  - 価格設定 → `pricing-strategy`
  - プロモーション設計 → `promotion-strategy`
  - 製品ライフサイクル分析 → `product-life-cycle`
  - バリュープロポジション設計 → `value-proposition`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 製品・サービス名 | 必須 | 分析対象 | 「会議室予約SaaS」「高級腕時計」 |
| 業界・市場 | 必須 | BtoB／BtoC、業界 | 「BtoB SaaS」「BtoC消費財」 |
| 既知の特徴 | 任意 | 品質・価格・ブランド情報 | 「価格は競合の80%、3年保証付き」 |

## 実行ステップ

### ステップ1：Product 3要素（コア／形態／付随機能）に分解する

- 入力：製品・サービス名、業界、既知の特徴
- 処理：コトラーの3要素モデルで分解
  - **製品・サービスのコア（ベネフィットサービス）**：製品が提供する基本的な価値や機能
    - 例：自動車のコア＝「移動手段」、コーヒーのコア＝「カフェイン摂取」
  - **製品・サービスの形態（パッケージ／特徴／スタイル）**：実際の製品のデザイン、仕様、品質、ブランド
    - 例：自動車の形態＝デザイン／燃費／ブランド、コーヒーの形態＝豆の種類／焙煎度
  - **製品・サービスの付随機能（設定／設置／配達／クレジット等）**：サポート、アフターサービス、保証、付属品
    - 例：自動車の付随機能＝保証／メンテナンス、コーヒーの付随機能＝Wi-Fi／空間体験
- 出力：3要素の内容＋自社の重点重要度（コア／形態／付随機能のうちどこが最重要か）

### ステップ2：物理的特性で分類する（非耐久財／耐久財／サービス）

- 入力：製品・サービス名
- 処理：以下の3分類のどれに該当するか判定
  - **非耐久財**（短期消費・使用回数限定）：食品、飲料、消耗品 → 重点：リピート購入、ブランド認知、流通最適化
  - **耐久財**（長期間使用の物理製品）：自動車、家具、家電、機械設備 → 重点：品質、保証、人的販売
  - **サービス**（形がなく価値を生む）：コンサル、ホテル、修理メンテ → 重点：信頼性、顧客体験、柔軟な対応
- 出力：分類判定＋分類別の重点戦略

### ステップ3：消費財／生産財／カテゴリ細分化

- 入力：ステップ2の分類結果（非耐久財・耐久財の場合）
- 処理：物理製品なら以下を判定
  - **消費財（個人・家庭が日常使用）の細分化**：
    - **最寄品**（頻繁購入・日常品）：食品・飲料・生活必需品 → 重点：手軽さ、利便性、流通最適化
    - **買回品**（品質・価格を比較購入）：衣類・家具・家電 → 重点：情報提供、選択肢の比較
    - **専門品**（ブランド・品質こだわり）：高級時計・スポーツカー・ブランドバッグ → 重点：ブランド価値、限定感、高級感あるプロモ
  - **生産財（企業・工場用）の細分化**：
    - **材料・部品**：原材料（鉄鋼・木材・石油）、部品（ネジ・ギア）
    - **資本財**：製造装置、工場ライン、工場・倉庫・店舗・オフィスビル
    - **備品・サービス**：パソコン・コピー機・オフィス家具、コンサル・広告・マーケ支援
- 出力：細分カテゴリ＋カテゴリ別の販売戦略の重点

### ステップ4：商品カテゴリマトリクス（理性×情緒）で位置づける

- 入力：製品・サービス名、ステップ3の分類
- 処理：以下4象限のどこに位置するか判定（出典：池田紀行『マーケティング「つながる」思考術』）
  - **左上：機能・必要性で選ぶ大きな買い物**（理性×買回品/専門品）：自動車・家電・住宅・保険・ビジネスホテル → スペック訴求・比較表
  - **右上：好み・ブランドで選ぶ大きな買い物**（情緒×買回品/専門品）：趣味のバイク・高級家具・コスメ・音楽映画・高価格レストラン → 世界観訴求・ブランドストーリー
  - **左下：生活に必要だから買う日常品**（理性×最寄品）：食品・飲料・日用品・風邪薬・ガソリン・安い飲食店 → 価格・利便性・流通網
  - **右下：その時の気分で選ぶ日常品**（情緒×最寄品）：シャンプー・お菓子・アイスクリーム・柔軟剤・エナジードリンク → パッケージ・キャンペーン・SNS
- 出力：4象限のどこか＋象限別の訴求方針

### ステップ5：プロダクト戦略の重点ポイントをまとめる

- 入力：ステップ1〜4の結果
- 処理：以下を統合してプロダクト戦略を1〜3個の重点ポイントに集約
  - 3要素のうちどこに重点投資するか（コア改善／形態磨き込み／付随機能拡充）
  - 物理的特性に応じた重点（リピート促進／保証／顧客体験）
  - 細分カテゴリに応じた重点（手軽さ／情報提供／ブランド）
  - 4象限に応じた訴求（スペック訴求／ブランドストーリー／価格訴求／パッケージ）
- 出力：プロダクト戦略 重点3項目＋実行アクション

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 製品が成熟期で差別化困難 | 付随機能（保証・サポート）に重点 | コア・形態だけでは差がつかないため |
| 新規参入で認知度低い | コア（基本価値）の明確化を最優先 | 顧客が「何の便益か」を理解できないと買わない |
| BtoB SaaSプロダクト | サービス分類＋付随機能（カスタマーサクセス） | 解約率が成否を決めるため |
| 高級ブランド商品 | 専門品＋情緒象限で世界観訴求 | スペック訴求は逆効果 |
| 日用消費財 | 最寄品＋（理性or情緒）で流通最適化 | 棚取りと価格訴求が勝敗を決める |

## 連携スキル

- **`marketing-mix-4p4c`**：4P全体の文脈で位置付け
- **`pricing-strategy`**：商品カテゴリ確定後の価格戦略
- **`promotion-strategy`**：象限別の訴求方針を実装
- **`product-life-cycle`**：プロダクトのフェーズ判定（導入／成長／成熟／衰退）
- **`value-proposition`**：コア＋付随機能をバリュープロポジションに変換
- **`stp-analysis`**：ターゲット明確化と組み合わせ
