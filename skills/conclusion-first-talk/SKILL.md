# conclusion-first-talk

ダラダラした商品説明を、結論から話す6ステップ構成に再編集し、聞き手が集中モードに入るトーク台本を生成する。

**商談フェーズ位置**：提案（商品説明・差別化訴求）

## 適用条件

- 使う場面：商品説明、提案プレゼン、差別化ポイントの訴求、商談中盤
- 使わない場面：雑談、ヒアリング、クロージング直前（別スキルを使う）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 商材/サービス名 | 必須 | 説明対象 | 外壁塗装 |
| 一番伝えたい結論 | 必須 | 1文で言い切れる売り | 耐久年数が20年 |
| 理由 | 必須 | 結論の裏付けとなる仕組み | 先端コーティング技術 |
| 証拠/データ | 必須 | 数値・実験・事例 | 20年間剥がれなかった実験結果 |
| 追加メリット | 任意 | 安心ポイント・付帯価値 | アフターフォロー即対応 |
| ターゲット | 任意 | 誰に話すトークか | 戸建オーナー |

## 実行ステップ

### ステップ1：前振り文を生成する
- 入力：商材、ターゲット
- 処理：
  - 「これは〇〇の中で、なぜ〇〇に選ばれているか」型の一文を作る
  - 聞き手の脳に「これから何の話か」を予告し、集中スペースを作る
- 出力：前振り文

### ステップ2：結論文を1文で生成する
- 入力：一番伝えたい結論
- 処理：
  - 「実は〇〇なんです」型で1文に圧縮する
  - 数値・極端な言葉（驚異の/圧倒的に/実は）を含める
  - 2文に分けない
- 出力：結論文

### ステップ3：理由文を生成する
- 入力：理由
- 処理：
  - 「なぜなら〇〇だからです」型で1〜2文にする
  - 技術名や仕組み名を1つ入れる
- 出力：理由文

### ステップ4：証拠/データ文を生成する
- 入力：証拠/データ
- 処理：
  - 数値・実験・第三者機関・事例のいずれかを必ず1つ入れる
  - 「実験結果でも〇〇」「〇〇社の導入実績で〇〇」型
- 出力：証拠文

### ステップ5：追加メリット文を生成する
- 入力：追加メリット
- 処理：
  - 「しかも〇〇」で始め、安心ポイントを追加する
  - 追加メリットが無ければ保証・アフターで埋める
- 出力：追加メリット文

### ステップ6：再結論（まとめ）文を生成する
- 入力：ステップ1〜5
- 処理：
  - 「だから〇〇は当然のことなのです」型で締める
  - 冒頭の結論を別の言い方で繰り返す
- 出力：再結論文

### ステップ7：6ステップを結合しトーク台本として出力する
- 入力：ステップ1〜6の全文
- 処理：順序通りに結合し、総尺を90〜120秒（日本語700〜900字）に収める。超過する場合は理由・証拠を削る
- 出力：完成トーク台本

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 証拠データが弱い | 事例に置換するか「お客様の声」型に変更 | 数字が弱いと結論が浮く |
| 追加メリット無し | 保証・サポートで代替 | 5ステップ目が空だと失速する |
| 競合が強い市場 | 前振りに「100社ある中で」など比較母数を入れる | 選ばれる理由の強調に効く |
| 時間が30秒しかない | ステップ1省略、結論→理由→再結論の3ステップに圧縮 | 最重要は結論・理由・再結論 |

## 出力フォーマット

```
【1. 前振り】
〇〇

【2. 結論】
〇〇

【3. 理由】
〇〇

【4. 証拠/データ】
〇〇

【5. 追加メリット】
〇〇

【6. 再結論】
〇〇
```

### 記入例

```
【1. 前振り】
これは100社あるリフォーム会社の中で、なぜ当社が選ばれているかという話なんですけど。

【2. 結論】
実はうちの塗装は耐久年数が驚異の20年間と非常に長いんです。

【3. 理由】
なぜなら先端コーティング技術を使用しているからです。

【4. 証拠/データ】
実験結果でも20年近く剥がれないことが確認されています。

【5. 追加メリット】
しかもアフターフォローもしっかりしており、何かあればすぐに駆けつけます。

【6. 再結論】
だから、100社の中からお客様が当社を選ばれるのは当然のことなのです。
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 前振りなしで結論からいきなり | 前振り1文で予告 | 脳が集中スペースを作れない |
| 結論が2文以上 | 結論は必ず1文 | 要点がぼやける |
| 数値なしの証拠 | 必ず数値・実験・事例のいずれか | 結論が浮く |
| 再結論を省略 | 必ず再結論で締める | 納得感が定着しない |
