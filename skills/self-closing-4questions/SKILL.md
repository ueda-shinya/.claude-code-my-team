# self-closing-4questions

顧客の口から購入理由を言語化させる4質問を商材に合わせて生成する。

**商談フェーズ位置**：クロージング（提案完了後〜成約）

## 適用条件

- 使う場面：高関与・高単価商材、押しに弱い顧客、顧客にじっくり言語化させたい場面、複数回商談でも段階運用可
- 使わない場面：冒頭アプローチ、クレーム対応
- 棲み分け：短時間・即決型の場面では `sokketsu-closing` を使う。本スキルは「顧客自身に4段階で言語化させる」深掘り用途に特化

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 商材/サービス名 | 必須 | クロージング対象 | パーソナルジム |
| 主要ベネフィット | 必須 | 顧客が得られる変化 | 3ヶ月で-8kg |
| 顧客の属性 | 任意 | BtoC個人、BtoB等 | BtoC個人 |
| 想定される良かったポイント | 任意 | 事前仮説（参考用） | マンツーマン／食事指導 |

## 実行ステップ

### ステップ1：質問1（どこが良かったか）を生成する
- 入力：商材
- 処理：
  - 型「本日お話を聞いていただいて、どこのポイントが良かったと思っていただけましたか？」
  - 商材名で少しカスタマイズする
  - 営業は自分の想定を絶対に先に言わない指示を注記する
- 出力：質問1セリフ

### ステップ2：質問2（なぜ良いと思ったか）を生成する
- 入力：質問1の想定回答
- 処理：
  - 型「なぜそのポイントが良いと思われたんですか？」
  - 相手の回答をそのままキーワードとして挿入する指示を注記
  - 「なるほど」「そうなんですね」以外の反応を挟まない
- 出力：質問2セリフ

### ステップ3：質問3（生活がどう変わるか）を生成する
- 入力：主要ベネフィット
- 処理：
  - 型「それが実現したら、〇〇様の生活は具体的にどう変わりそうですか？」
  - 「具体的に」を必ず含める（抽象回答を避けるため）
  - 想像を促す時制（未来形）を使う
- 出力：質問3セリフ

### ステップ4：質問4（気分はどうか）を生成する
- 入力：質問3の想定回答
- 処理：
  - 型「そうなったら、〇〇様の気分はどうですか？」
  - 感情語を引き出す質問に寄せる
  - 「嬉しいですか？」など誘導は使わない（オープン質問）
- 出力：質問4セリフ

### ステップ5：質問間の待ち時間・反応ルールを指定する
- 入力：4質問
- 処理：
  - 各質問後、沈黙を最低3秒待つ指示を明記
  - 相手の回答を遮らない・要約しない・言い換えない
  - 回答を必ずメモして、次の質問で参照する
- 出力：反応ルールブロック

### ステップ6：4質問を台本として結合する
- 入力：ステップ1〜5
- 処理：質問1→2→3→4の順に並べ、各間に「沈黙3秒」「メモ」の注記を挿入する
- 出力：完成4質問台本

### ステップ7：最終クロージング接続文を生成する
- 入力：想定される顧客回答
- 処理：
  - 型「今お話いただいた〇〇（相手の言葉）が実現するのが、まさにこのサービスです。ぜひスタートしましょう」
  - プレースホルダー〇〇は顧客発言を差し込む指示を明記
- 出力：最終クロージング接続文

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 質問1で相手が「特に…」と濁す | 提案内容に戻って1つポイントを再説明する | 良かった点がないと4質問が成立しない |
| 質問3で「分からない」回答 | 質問3aを追加：「もし〇〇ができたら、何を最初にやりたいですか？」 | 想像が苦手な相手への補助線 |
| 質問4で感情語が出ない | 「嬉しい/安心/自信、どれに近いですか？」と3択で聞く | 感情のラベリングが苦手な相手向け |
| BtoBの場合 | 「〇〇様の生活」→「御社の業務」に置換 | 法人文脈に合わせる |

```
〇〇
（沈黙3秒・メモ）

【質問2】なぜ良いと思ったか
〇〇
（沈黙3秒・メモ）

【質問3】生活がどう変わるか
〇〇
（沈黙3秒・メモ）

【質問4】気分はどうか
〇〇
（沈黙3秒・メモ）

【最終クロージング接続】
「今お話いただいた【相手の言葉】が実現するのが、まさにこのサービスです。ぜひスタートしましょう」
```

### 記入例

※記入例は「相手が質問1に『マンツーマンが良かった』と答えた後の流れ」を示したもの。質問2以降は相手の言葉をそのまま差し込む。

```
【質問1】
「本日パーソナルジムのお話を聞いていただいて、どこのポイントが良かったと思っていただけましたか？」
（沈黙3秒・メモ）

【質問2】
「なぜマンツーマンのポイントが良いと思われたんですか？」
（沈黙3秒・メモ）

【質問3】
「それが実現したら、田中様の生活は具体的にどう変わりそうですか？」
（沈黙3秒・メモ）

【質問4】
「3ヶ月後に-8kgを達成できたら、田中様の気分はどうですか？」
（沈黙3秒・メモ）

【最終クロージング接続】
「今お話いただいた【自分に自信が持てる】が実現するのが、まさにこのサービスです。ぜひスタートしましょう。」
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 「どうでしたか？」と漠然と聞く | 「どこのポイントが良かったですか？」と具体化 | 漠然質問は無難な答えしか出ない |
| 営業が先に「〇〇が良かったですよね」と代弁 | 沈黙して待つ | 自分で言わせないと刺さらない |
| 相手の回答を要約し直す | そのままの言葉でメモする | 言い換えで熱量が落ちる |
| 質問3を省略 | 必ず4質問全て使う | 感情の揺さぶりが弱くなる |
