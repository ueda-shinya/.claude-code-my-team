# financing-strategy スキル

資金調達戦略の選択肢として、デットファイナンス（借入金）とエクイティファイナンス（株式発行）を体系的に比較し、自社の成長段階・財務状況・成長戦略に応じた最適な調達手段を選定・組み合わせる。事業戦略スクール「知足」フレームワーク準拠。

## 適用条件

- このスキルを使う場面：
  - 資金調達手段（デット／エクイティ）の選択を検討したい
  - 銀行借入の交渉順序・進め方を知りたい
  - 株式発行（公募増資／第三者割当増資／IPO）の判断材料が欲しい
  - 自社の成長段階に応じた調達戦略を設計したい
  - デットとエクイティの組み合わせバランスを最適化したい
- このスキルを使わない場面：
  - 財務三表（BS/PL/CF）の基礎理解 → `financial-statements-fundamentals` を使う
  - M&A戦略 → `ma-strategy-basics` を使う
  - 個別プロジェクトの収支管理 → `project-finance-contract` を使う
  - 融資・契約の法務実務 → 専門家（弁護士・税理士）に委任

## 実行ステップ

### ステップ1：デットファイナンスの基本を確認する

**デットファイナンス** = 借入金によって資金を調達する方法
**返済義務**：元本の返済と利息の支払いが必要

**代表的な2方法**：
| 方法 | 内容 |
|---|---|
| 銀行借入 | 銀行からの融資による資金調達 |
| 社債発行 | 投資家に社債を発行し資金調達 |

**メリット・デメリット**：
| メリット | デメリット |
|---|---|
| 株主の持分が希薄化しない | 毎月の利息支払いが発生 |
| 支払利息は税控除可能 | 売上が不振でも元本返済が必要 |
| 資金調達が迅速 | 借入金が多いと財務リスクが増大 |

### ステップ2：中小企業の銀行借入交渉順序を理解する

**スタートアップでない限り、多くのケースで融資での資金調達となる**

**まず交渉すべき銀行（中小企業）**：
| 機関 | 特徴 |
|---|---|
| 信用金庫 | 地域密着、中小向けに柔軟 |
| 政策金融公庫 | 創業融資・無担保融資の選択肢豊富 |
| 商工組合中央金庫 | 中堅・中小企業向け、金利安定 |

→ **初期は保証協会の保証ありで融資を受ける**

**次に交渉すべき銀行（実績をつくってから）**：
| 機関 |
|---|
| 地方銀行 |
| メガバンク |
| 会社の近くに支店がある銀行 |

→ **実績をつくってから、プロパー融資を狙う**

**金融機関から融資をうける際の2ポイント**：
1. 綿密な事業計画の策定
2. 実績と人間関係での信頼構築

### ステップ3：エクイティファイナンスの基本を確認する

**エクイティファイナンス** = 株式を発行して資金を調達する方法
**返済義務なし**：投資家からの資金提供は返済不要

**代表的な3方法**：
| 方法 | 内容 |
|---|---|
| 公募増資 | 新規株式を市場で販売して資金調達 |
| 第三者割当増資 | 特定の投資家に株式を発行し資金調達 |
| IPO（株式公開） | 株式市場に上場して多額の資金調達 |

**メリット・デメリット**：
| メリット | デメリット |
|---|---|
| 資金返済の必要がないため財務負担が少ない | 株式が増加すると既存株主の持分が希薄化 |
| 長期的な資金調達が可能 | コントロールの分散（投資家の意見が意思決定に影響する） |
| 投資家の支援やネットワークを活用できる | 市場環境の影響を受けやすい |

### ステップ4：6比較軸でデット／エクイティを使い分ける

| 比較軸 | デットファイナンス | エクイティファイナンス |
|---|---|---|
| 返済義務 | あり | なし |
| 所有権の変化 | なし | 株式発行により変化 |
| 資金コスト | 支払利息 | 株主配当や株式価値の希薄化 |
| 調達スピード | 比較的速い | 条件や市場環境による |
| リスク | 債務超過リスク | コントロールの分散 |
| 適合する成長段階 | 安定成長期 | 創業期・成長初期 |

**使い分けのポイント**：
- デットファイナンスは安定収益が見込める場合に適している
- エクイティファイナンスは高リスク事業や創業初期の資金調達に適している

### ステップ5：成長段階別の組み合わせ活用例を選ぶ

**両者のメリットを活用し、リスクとリターンを最適化**

| 成長段階 | 推奨アプローチ |
|---|---|
| 成長初期 | エクイティファイナンスで事業拡大資金を確保 |
| 安定期 | デットファイナンスで低コストの資金を調達 |
| 大規模投資時 | デットとエクイティを組み合わせ、資金調達の柔軟性を高める |

**実際の事例**：
- スタートアップ企業：創業初期にエクイティで資金調達し、成長後にデットで資金運用を効率化
- 大企業：設備投資はデット、新規事業開発にはエクイティを活用

### ステップ6：4課題分析STEPで自社の調達戦略を点検する

| STEP | 確認項目 |
|---|---|
| STEP01 財務状況 | 借入金の割合や財務リスクは適切か？ |
| STEP02 成長段階 | 企業の成長段階に適した調達方法を採用しているか？ |
| STEP03 所有権の影響 | 株式希薄化やコントロール分散の影響をどう管理するか？ |
| STEP04 コストとリスク | 調達コストと返済リスクを総合的に評価できているか？ |

### ステップ7：3つの基本方針でまとめる

| 方針 | 内容 |
|---|---|
| ① 資金調達の基本方針 | デットは安定キャッシュフロー企業／エクイティは成長可能性の高い企業 |
| ② メリットとリスクのバランス | 両者の特徴を理解し、最適な組み合わせを検討 |
| ③ 企業成長段階に応じた調達戦略 | 創業初期から成長段階、安定期まで、段階に応じて柔軟に使い分ける |

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 中小企業で初めての銀行借入 | 政策金融公庫／商工中金／信用金庫から保証協会付きで開始 | 中小企業の融資定石、地銀・メガバンクは実績後 |
| 安定収益・実績あり、設備投資資金が必要 | デットファイナンス（プロパー融資狙い） | 低コストかつ持分希薄化なし |
| 創業期で実績ない、高成長を目指す | エクイティファイナンス（VC・エンジェル投資家） | 返済義務なし、長期成長資金として最適 |
| 成長期（売上拡大局面・PMF達成後・拡大投資が必要） | デット＋エクイティの併用検討（事業拡大規模に応じて主軸を選択） | 急成長期は資金需要が大きく単一手段では不足、リスク分散と柔軟性確保 |
| 既存株主のコントロール維持を最優先 | デットファイナンス優先 | エクイティは持分希薄化リスク |
| 売上変動が大きい・キャッシュフロー不安定 | エクイティ優先（デットの元本返済リスクを避ける） | 売上不振でも元本返済義務がデットの最大リスク |
| 大規模投資（IPO直前など） | デット＋エクイティの組み合わせ | 柔軟性最大化 |
| 借入金比率が自己資本の3倍超 | デットの追加調達は慎重に、エクイティ検討 | 財務リスク（債務超過）の閾値 |
| 株主構成を整えてからIPOを目指す | 第三者割当増資→IPO の段階設計 | 一足飛びのIPOは準備不足リスク |
| 「うちは独自事業で投資家がつきやすい」と楽観 | `competitive-absence-audit` で検証 | 市場価値の過大評価リスク |
| 1機関のみと交渉し条件比較していない | 必ず複数機関で並行交渉 | 条件比較なしは交渉力低下 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 中小企業がいきなりメガバンクに融資相談 | 政策金融公庫／信用金庫から開始、実績後にメガバンクへ | 中小企業の融資交渉の定石 |
| 1機関とのみ交渉して条件比較しない | 複数機関で並行交渉 | 条件比較なしは交渉力低下、不利な条件で契約 |
| 借入比率が高い状態でさらに借入 | 借入金が自己資本の3倍超なら慎重判断 | 債務超過リスクの閾値 |
| エクイティで安易に持分を50%以下にする | 創業者比率は議決権に留意、株主間契約で調整 | コントロール権喪失リスク |
| 事業計画なしで融資交渉 | 綿密な事業計画書（資金使途・返済計画・売上予測）を持参 | 銀行融資の最重要評価項目 |
| 「返済不要だから」だけでエクイティ選択 | 持分希薄化・コントロール分散リスクも評価 | エクイティのデメリット軽視は失敗パターン |
| 売上変動が大きいのにデット中心 | 元本返済義務リスクを評価、エクイティ併用検討 | 売上不振時の元本返済が経営圧迫の主因 |
| 「独自事業で投資家がつきやすい」と楽観判断 | `competitive-absence-audit` で需要・市場検証 | 市場価値の過大評価で資金調達失敗 |
| 創業初期にデット中心で財務硬直化 | 創業期はエクイティ優先 | 創業期は売上不安定、デットの元本返済が致命傷 |
| IPOを最終目標にせず「いつかやる」と先延ばし | 第三者割当→グロース市場→プライム市場の段階設計 | IPO準備は5年以上必要 |

## 連携スキル

- `financial-statements-fundamentals`：BS/PL/CF基礎で自社財務体力を把握
- `ma-strategy-basics`：M&Aの資金調達手段としての位置づけ
- `medium-term-business-plan`：中長期事業計画と資金調達計画の整合
- `growth-phase-strategy`：企業成長フェーズに応じた財務戦略
- `business-model-canvas`：投資家プレゼン用のビジネスモデル整理
- `pmf-journey`：エクイティ調達のタイミング判断（PMF達成有無）
- `competitive-absence-audit`：市場価値・需要の楽観判断検証
- `project-finance-contract`：個別プロジェクトの収支・契約管理

## 5状態出力契約

このスキルは外部API・サブプロセス実行・外部設定依存のいずれも持たないため、CLAUDE.md「スキル出力の5状態契約」の対象判定基準には該当しない。ただし以下のケースで明示マーカーを使うこと：

- `[OK:0件]`：自社の成長段階・必要資金が不明瞭で、フレームワーク説明のみで終了した場合
- `[SKIP]`：競合不在の楽観判断が出た場合は `competitive-absence-audit` への誘導でスキップ／詳細実務は専門家委任が必要として手前で停止

外部API・サブプロセスを持たないため、`[FAIL]`（試行後の失敗）は通常発生しない。
