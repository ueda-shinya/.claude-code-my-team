# branding スキル

ブランディング = 企業や製品の「アイデンティティ」を形成し、認知や信頼を構築する活動。事業戦略スクール「知足」のフレームワーク準拠。

## ブランディングとマーケティング（プロモーション）の違い

| 観点 | マーケティング（プロモーション） | ブランディング |
|---|---|---|
| **定義** | 製品・サービスを市場に届け顧客獲得 | 企業・製品の「アイデンティティ」形成 |
| **目的** | 売上向上・顧客獲得 | ブランド価値向上・長期的信頼関係 |
| **焦点** | 短〜中期（キャンペーン・広告） | 長期（企業イメージ・価値観・文化） |
| **アプローチ** | 顧客ニーズに応える製品提供 | 「自社はどう在るべきか」を定義し一貫保持 |
| **成果物** | 広告・SEO・メールマーケ | ロゴ・スローガン・ガイドライン・トーン&マナー |
| **ターゲット** | 特定顧客層・市場セグメント | 顧客全体・社会一般 |
| **測定** | 売上・CV率・リード数 | 認知度・ロイヤルティ・信頼度・好感度 |
| **感情の影響** | 購買行動を刺激する戦術 | 心に残る共感・つながり |

※広義にはブランディングもマーケティングの一部。本スキルは**プロモーションとの対比**で扱う。

## 適用条件

- このスキルを使う場面：
  - ブランドエクイティ（ブランド資産価値）を体系的に設計・測定したい
  - 既存ブランドの認知度・ロイヤルティ・連想を強化したい
  - 競合と差別化された一貫したブランドメッセージを設計したい
  - スターバックス・Apple・Nike のような事例に学んでブランド戦略を作りたい
- このスキルを使わない場面：
  - MVV（経営理念）の言語化 → `mvv-design`
  - ブランドビジュアル規定（ロゴ・カラー・タイポグラフィ・スタイルガイド） → `brand-guidelines`
  - キャッチコピー・スローガン文言の作成 → コト（copywriter）
  - プロモーション戦略・広告ミックス → `promotion-strategy`
  - マーケ戦略全体 → `marketing-mix-4p4c`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 会社・ブランド情報 | 必須 | 企業名／ブランド名／業種／規模 | 「カフェチェーン20店舗・地方発」 |
| 既存ブランド要素 | 任意 | 既存のロゴ／スローガン／ガイドライン | 「ロゴあり、スローガンなし」 |
| 想定顧客 | 任意 | ターゲット顧客像 | 「20-40代の都市部在住女性」 |
| 競合 | 任意 | 主要競合3〜5社 | 「スターバックス、ドトール、サードウェーブ系」 |
| MVV | 任意 | 確定したミッション・ビジョン・バリュー | 「Mission：地域に第三の居場所を提供」 |
| 設計の目的 | 任意 | 新規構築／既存強化／リブランディング | 「既存ブランドの認知度を上げたい」 |

## ブランドエクイティの2モデル

ブランドエクイティ = ブランドが持つ資産価値。消費者の認知度・信頼感・ロイヤルティなどが形成する。

### A. アーカーモデル（David A. Aaker提唱）— 5要素

| 要素 | 内容 | 例 |
|---|---|---|
| **ブランド認知度（Brand Awareness）** | 消費者がブランドをどれだけ知っているか | Nike、Coca-Cola は世界的高認知度 |
| **ブランドロイヤルティ（Brand Loyalty）** | ブランドに対する消費者の忠誠心 | 常に同じブランドの商品を選ぶ顧客 |
| **ブランド連想（Brand Association）** | ブランドに対するイメージ | BMW = 高級車・性能の良さ |
| **知覚価値（Perceived Quality）** | 製品・サービスの品質に対する評価 | Dyson = 革新性・高性能 |
| **その他のブランド資産** | 特許・商標・独自チャネル等 | 独自販売網・知財ポートフォリオ |

### B. ケラーモデル（Kevin Lane Keller提唱）— CBBE ブランドピラミッド

下から上に積み上がる4階層6要素のピラミッド：

| 階層 | 要素 | 内容 | ブランディング目標 | 顧客行動 |
|---|---|---|---|---|
| **第4階層（頂点）** | レゾナンス（共感） | ブランドの心理的な絆の強さと深さ | 強く積極的なロイヤルティ | 絆 |
| **第3階層** | ジャッジメント（評価）／フィーリング（感情） | 顧客の製品評価／感情的反応・評価 | ポジティブで好意的な反応 | 信頼・愛着／満足・納得 |
| **第2階層** | パフォーマンス（性能）／イメージ（印象） | 機能的特性／抽象的イメージで心理的ニーズ充足 | 差別化ポイント／類似化ポイント | （第3階層へ昇る） |
| **第1階層（基礎）** | セイリエンス（再認・再生） | ブランドに対する深くて広い認知 | 深くて広いブランド認知 | 興味・関心 |

**両側の柱**：左＝機能的価値（パフォーマンス→ジャッジメント）、右＝情緒的価値（イメージ→フィーリング）。最終的に頂点のレゾナンス（共感）でブランドコミットメントが最大化する。

## ブランド価値向上のメリット（4段階の影響）

強いブランドは消費者の心情・行動に影響を与え、ビジネスに経済的恩恵をもたらす。

| 段階 | 消費者行動への影響 | ビジネスへの恩恵 |
|---|---|---|
| **① 知覚品質を上げる** | 商品/サービスに期待する価値やイメージが豊富。多少高くても購入／同パフォーマンスでもより価値を感じる | **価格プレミアム実現／製品機能の差別化** |
| **② 選択と推奨** | 他商品より瞬時に期待価値・情報が豊富。比較せず購入／推奨度合いが高まる | **顧客獲得コストの低減** |
| **③ ロイヤリティを高める** | 継続購入や利益増加に伴うブランドへの絆／ロイヤリティ醸成 | **顧客の参入障壁の構築** |
| **④ 自己表現** | ロイヤリティ強化に伴うブランドへの絆の醸成・強化 | **ブランド拡張機会の増加** |

## ブランドエクイティを高める5つの方法

| 方法 | 内容 | 事例 |
|---|---|---|
| **① 一貫性のあるブランドメッセージ** | ブランドストーリーを一貫して発信する | Patagonia は「環境保護」をテーマに統一メッセージ発信 |
| **② 高品質な製品・サービス** | 信頼を維持するために品質を高め続ける | （業界によらず必須） |
| **③ 顧客体験の最適化** | 購入からアフターサービスまでの顧客体験を最適化 | Amazon の迅速な配送とカスタマーサポート |
| **④ 感情的なつながりの構築** | 消費者の価値観や感情に訴える | Disney は「夢」や「魔法」を提供するブランド |
| **⑤ ブランドロイヤルティプログラム** | ポイント制度や特典で顧客のロイヤルティを育む | Starbucks のリワードプログラム |

## ブランドエクイティの測定方法

| 測定軸 | 内容 |
|---|---|
| **売上や市場シェア** | ブランドの競争力を数値化 |
| **ロイヤルティ率** | 繰り返し購入する顧客の割合（リピート率／NPS） |
| **ブランド認知度調査** | 消費者がブランドをどれだけ知っているかを調査（純粋想起／助成想起） |
| **ブランドイメージ調査** | ブランドに対する消費者の印象やイメージを把握（連想ワード調査） |

## ブランドを定義する2フレームワーク

### フレームA：ブランディング・サーベイ（タナベ・コンサルティング社提唱）

ブランドを調査・定義するため、社員・顧客にアンケートで自社ブランドの要素を抽出する3円交差モデル：

| 円 | 視点 | 内容 |
|---|---|---|
| **Brand Loyalty（ブランド力の総合判断指標）** | 社員＋顧客視点 | 1. ブランド理解度（よく知ってもらうこと）／2. ブランド好感度（好まれ、選ばれること）／3. ブランド優先度（ナンバーワンになること） |
| **Brand Deployment（ブランディングの取り組みへの評価）** | 社員視点 | 1. ブランドコンセプト（理念）／2. ブランドターゲット（顧客）／3. ブランドバリュー（商品・サービス）／4. インナーブランディング（社員教育）／5. アウターブランディング（プロモーション） |
| **Brand Image（顧客・社員から見た企業のイメージ）** | 顧客視点 | 1. 会社はどのような社会価値があるか／2. 会社はどのような顧客価値があるか／3. 会社の人材はどのような価値があるか |

3円の交差で総合的なブランド評価が見える。

### フレームB：ブランド・アイデンティティ・プリズム（Kapferer提唱）

顧客のイメージと自社ブランドのアイデンティティを同時に整理する6面構造：

| 区分 | 要素 | 説明 | スターバックスの事例 |
|---|---|---|---|
| **自社（Picture of Sender）外的（Externalization）** | Physique（物理的特徴） | ブランドの物理的特徴 | シアトル初のコーヒーカフェ／コーヒー豆のこだわり／落ち着きのある空間／フレンドリーなバリスタ |
| **自社 内的（Internalization）** | Personality（パーソナリティ） | ブランドの人格 | 上質／フレンドリー／洗練／正統的／誠実 |
| **両者の中間 外的** | Relationship（関係性） | 顧客とブランドの関係性 | 家でも職場でもない「サードプレイス」 |
| **両者の中間 内的** | Culture（文化） | ブランドの文化・価値観 | 世界中の人々の心に活力と栄養を与え、環境をより良い場所に変える |
| **顧客（Picture of Receiver）外的** | Reflection（反映） | ブランドのターゲット | 忙しい毎日をすごす都会的な男女／知的でおしゃれ／非喫煙者 |
| **顧客 内的** | Self-image（セルフイメージ） | 顧客の自己像 | 最高のコーヒー体験／心身ともに落ち着いたリラックス体験／都会的に洗練された自分としての自己表現 |

中央に Brand Symbol（ブランドシンボル＝ロゴ等）を配置。

## 実行ステップ

### ステップ1：現状診断とMVV確認
- 入力：会社・ブランド情報、既存ブランド要素、MVV
- 処理：
  1. MVVが未確定なら `mvv-design` への接続を提案
  2. 既存ブランド要素（ロゴ・スローガン・ガイドライン・トーン&マナー）を一覧化
  3. ブランディングとマーケティングの違い9観点で「現状どちらに偏っているか」を診断
- 出力：現状診断レポート＋MVV連携状況

### ステップ2：ブランドエクイティ現状評価
- 入力：会社情報＋顧客アンケート/売上データ（あれば）
- 処理：
  1. **アーカーモデル5要素**で現状をA/B/C評価する
     - ブランド認知度／ロイヤルティ／連想／知覚価値／その他資産
  2. **ケラーCBBEブランドピラミッド**で現在の到達階層を判定
     - 第1（認知）／第2（差別化）／第3（評価・感情）／第4（共感・絆）
  3. データ不足項目は「測定が必要」として残課題に
- 出力：エクイティ現状評価表＋ピラミッド到達階層判定

### ステップ3：ブランディング・サーベイ設計（現状把握ツール作成）
- 入力：ステップ2の現状評価結果＋会社情報
- 処理：
  1. 3円（Loyalty／Deployment／Image）の各設問を設計
  2. 社員アンケート＋顧客アンケートのテンプレを作成
  3. 想定回答数・回収方法・分析軸を設計
  4. ステップ2のA/B/C評価のうち「データ不足」項目をサーベイで埋められる設問配置に
- 出力：ブランディング・サーベイ用アンケート設計書（次ステップのプリズム作成の根拠データを取得する道具）

### ステップ4：ブランド・アイデンティティ・プリズム作成（仮説構築）
- 入力：会社情報＋MVV＋顧客像＋（あれば）サーベイ実施済み結果
- 処理：
  1. 6面（Physique／Personality／Relationship／Culture／Reflection／Self-image）を1つずつ言語化
  2. 自社視点 vs 顧客視点で乖離がないかチェック（サーベイ未実施段階では「仮説」と明記）
  3. 中央のブランドシンボルを明示（既存ロゴ／コアメッセージ）
- 出力：ブランド・アイデンティティ・プリズム図（仮説 or 検証済みを明記）

### ステップ5：ブランドエクイティ強化施策の設計
- 入力：現状評価＋プリズム＋サーベイ結果（仮）
- 処理：以下の5方法から該当する施策を3〜5つ選定
  1. **一貫性のあるブランドメッセージ**：ブランドストーリーの言語化＋発信媒体統一
  2. **高品質な製品・サービス**：品質基準の文書化＋継続改善ルール
  3. **顧客体験の最適化**：カスタマージャーニーマップ作成（→ `customer-journey` 連携）
  4. **感情的なつながりの構築**：価値観・感情にフックする訴求設計
  5. **ブランドロイヤルティプログラム**：リピート促進プログラム設計
- 出力：強化施策セット（3〜5施策／実行優先度付き）

### ステップ6：測定設計と運用
- 入力：強化施策セット
- 処理：
  1. 測定4軸（売上シェア／ロイヤルティ率／認知度調査／イメージ調査）から該当軸を選定
  2. 測定頻度（四半期／半期／年次）と担当を決定
  3. 改善サイクルの運用フローを設計（→ `pdca-cycle` または `ooda-loop` 連携）
- 出力：測定運用計画＋改善サイクル設計

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| MVVが未確定 | `mvv-design` を先行実施 | ブランディングの土台はMVV |
| マーケティング（プロモーション）と混同 | 9観点表で違いを明示し棲み分け | プロモはCV最大化、ブランディングは長期信頼 |
| ブランドピラミッド第1階層（認知）未到達 | まずは認知度向上施策（広告・PR）を最優先 | 認知なしにロイヤルティは育たない |
| 第3階層（評価・感情）まで到達 | 第4階層（共感・絆）への施策（ロイヤルティプログラム・コミュニティ） | 共感はビジネスの最終ゴール |
| 認知度はあるが連想・イメージが弱い | プリズムでアイデンティティ再定義＋一貫メッセージ強化 | 認知だけでは選ばれない |
| 顧客視点と自社視点で乖離 | サーベイ実施＋プリズム再構築 | 一方通行のブランディングは形骸化 |
| 「ブルーオーシャン」「競合いない」と楽観評価 | `competitive-absence-audit` 併用 | CLAUDE.md「Competitive Absence Audit Rule」遵守 |
| 既存ブランドガイドラインが古い | リブランディング判断（数年単位の変更検討） | 時代変化に追従しないと陳腐化 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| プロモーションとブランディングを混同 | 9観点表で棲み分け、両者を併走させる | プロモは短期CV、ブランディングは長期信頼 |
| MVV不在のままブランディング着手 | `mvv-design` を先行 | 土台なしのブランドは一貫しない |
| アーカーモデルだけでケラーCBBE未確認 | 両モデル併用（アーカー＝資産分解、ケラー＝顧客心理） | 単一モデルでは多面性を逃す |
| プリズム6面のうち「自社視点」だけ書く | 必ず顧客視点（Reflection／Self-image）も書く | 一方通行ブランドは響かない |
| 認知度向上ばかりで第2階層以上に進まない | 段階的にピラミッドを登る（差別化→評価→共感） | 認知だけでは選ばれない |
| 「一貫メッセージ」と言いつつ媒体ごとに変える | ブランドガイドライン＋トーン&マナー文書化 | 一貫性は文書化で担保 |
| 測定なしで「ブランディングできた気」になる | 4軸（売上／ロイヤルティ／認知度／イメージ）で定期測定 | 測定なき施策は改善できない |
| 競合不在を根拠にした楽観ブランド戦略 | `competitive-absence-audit` を併用 | 楽観バイアスを封じる |

## 連携スキル

- `mvv-design`：ブランディングの土台となるミッション・ビジョン・バリュー設計
- `customer-journey`：顧客体験最適化（ステップ5の方法③）でCJM作成
- `persona-design`：ターゲット顧客（プリズム Reflection）の具体化
- `value-proposition`：差別化価値提案（アーカー連想・ケラー差別化ポイント）
- `stp-analysis`：セグメンテーション・ポジショニング設計
- `marketing-mix-4p4c`：4P/4Cでブランディングをマーケ全体に組み込む
- `promotion-strategy`：ブランディングメッセージのプロモ展開
- `pdca-cycle` / `ooda-loop`：改善サイクル運用
- `competitive-absence-audit`：「競合不在」系の楽観判断チェック
- コト（copywriter）：ブランドコピー・スローガン文言の作成
- カイ（lp-designer）／ユイ（web-designer）：ブランドビジュアルのLP・サイト設計

## 5状態出力契約

外部API・サブプロセス・外部設定依存を持たないため、CLAUDE.md「5状態出力契約」の対象判定基準には該当しない。ただし以下のマーカーを使うこと：

- `[OK:0件]`：ブランドエクイティが計測できず（データ皆無）、推奨施策のみ提示の場合
- `[PARTIAL:N成功/M失敗]`：アーカーモデル一部のみ評価できた場合（データある要素のみA/B/C評価）
- `[SKIP]`：MVV未確定のため `mvv-design` 先行を推奨し、本スキルの一部ステップを保留した場合
- `[FAIL]`：会社・ブランド情報が不足しすぎて診断不能な場合（再ヒアリング要請）
