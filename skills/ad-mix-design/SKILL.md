# ad-mix-design

広告の目的を起点に、4分類10種類の広告媒体から目的・ターゲット・予算に合致する組み合わせ（広告ミックス）を選定し、4つの注意点でリスク点検しKPIに落とすスキル。

## 適用条件

- このスキルを使う場面：
  - クライアントの「広告を打ちたい」「広告予算をどう配分するか」相談時
  - 商品/サービスのプロモーションで「広告」手段を採用すると決まった後、媒体を絞り込む段階
  - 既存広告の見直しで媒体ミックスを再設計したいとき
  - 新商品ローンチの広告計画立案時
- このスキルを使わない場面：
  - プロモーション5手段（広告／販売促進／人的販売／広報PR／ダイレクトマーケティング）全体から手段選定する段階 → `promotion-strategy` を使う
  - リスティング広告／ディスプレイ広告／SNS広告など個別広告タイプの実施詳細を設計したいとき → 個別スキル（リスティング広告／ディスプレイ広告等）を使う
  - 広告クリエイティブのコピー作成 → コト（copywriter）に委任
  - LPやバナーのデザイン → カイ（lp-designer）／ユイ（web-designer）に委任

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 商品/サービス名 | 必須 | 広告対象 | 「月額ITサポート」 |
| 広告の主目的 | 必須 | 4目的のうちどれか（複数可） | 「認知度向上＋リード獲得」 |
| ターゲット | 必須 | 性別/年齢/地域/業種/興味関心など | 「広島県の中小企業経営者・40-60代」 |
| 月間予算 | 必須 | 月額または期間総額 | 「月50万円」 |
| 広告期間 | 必須 | 実施期間 | 「3ヶ月」 |
| 既存実績 | 任意 | 過去の広告効果データ | 「過去にFacebook広告でCPA8,000円」 |
| 制約条件 | 任意 | 業種規制・媒体タブー等 | 「医療系で薬機法対応必須」 |

## 実行ステップ

### ステップ1：広告の目的を4目的から確定する

- 入力：商品/サービス名、ビジネス課題、商談背景
- 処理：以下の4目的のいずれか（または複数）に振り分ける。1-2個に絞る（3個以上は焦点が散る）
  | 目的 | 適する状況 | 典型的な指標 |
  |---|---|---|
  | 認知度向上 | 新商品/新ブランド／市場で名前を知られていない | リーチ数・インプレッション数・ブランド想起率 |
  | 購買促進 | 既知商品の販売拡大／在庫消化／キャンペーン期間 | CV数・売上・ROAS |
  | ブランド構築 | 中長期の信頼形成／競合との差別化 | ブランド指名検索・好意度・推奨意向（NPS） |
  | リードジェネレーション | BtoB／高単価商品で見込み顧客リスト構築が必要 | リード数・CPA・MQL/SQL転換率 |
- 出力：確定した広告目的（1-2個）＋目的別の主要KPI候補

### ステップ2：ターゲットと予算条件を整理する

- 入力：ターゲット情報、月間予算、広告期間
- 処理：
  - ターゲットを「リーチしたい属性」（年齢・性別・地域・職業・興味関心）に分解する
  - 予算を以下に区分する：
    - 月額予算（媒体費）
    - 制作費（クリエイティブ・LP制作）
    - 運用費（運用代行・ツール利用料）
    - 予備費（A/Bテスト・追加配信）
  - 期間総額 = 月額×期間＋制作費＋予備費 で総予算を確定する
- 出力：ターゲット属性リスト＋予算配分の初期案（媒体費／制作費／運用費／予備費）

### ステップ3：4分類10種類から候補媒体を絞り込む

- 入力：広告目的、ターゲット属性、月額媒体費
- 処理：以下の媒体マトリクスから、目的とターゲットと予算の3条件に合致する媒体を抽出する

  **マスメディア広告（信頼性・広範リーチ向き）**
  | 媒体 | 適する目的 | 費用相場 | 適する予算規模 |
  |---|---|---|---|
  | テレビ広告 | 認知度向上・ブランド構築 | 地域CM 100万〜300万円／全国CM 3,000万〜1億円超 | 月300万円〜 |
  | 新聞広告 | 認知度向上・ブランド構築 | 地方紙15段 200万円〜／全国紙15段 1,000万〜1,500万円 | 月200万円〜 |
  | 雑誌広告 | ブランド構築・ニッチ層認知 | 一般誌1P 50万〜150万円／巻頭・表紙裏 200万〜300万円 | 月50万円〜 |

  **デジタル広告（ターゲティング精度・コスパ向き）**
  | 媒体 | 適する目的 | 費用相場 | 適する予算規模 |
  |---|---|---|---|
  | リスティング広告 | 購買促進・リードジェネレーション | 月10万〜100万円以上 | 月10万円〜 |
  | SNS広告 | 認知度向上・購買促進・リードジェネレーション | 月5万〜100万円以上 | 月5万円〜 |
  | 動画広告 | 認知度向上・ブランド構築 | 月10万〜300万円以上 | 月10万円〜 |

  **アウトドア広告（地域密着・インパクト向き）**
  | 媒体 | 適する目的 | 費用相場 | 適する予算規模 |
  |---|---|---|---|
  | 看板広告 | 認知度向上・地域ブランド構築 | 月5万〜100万円以上 | 月5万円〜 |
  | 交通広告 | 認知度向上・地域認知 | 月10万〜300万円以上 | 月10万円〜 |

  **直接型広告（既存顧客・特定ターゲット向き）**
  | 媒体 | 適する目的 | 費用相場 | 適する予算規模 |
  |---|---|---|---|
  | ダイレクトメール | 購買促進・リテンション | 1通70〜150円×件数（1,000通で7-15万円） | 月10万円〜 |
  | クーポン広告 | 購買促進 | 月3万〜10万円以上 | 月3万円〜 |

- 絞り込みルール：
  - 媒体費が予算の70%を超える場合 → その媒体は単独運用（ミックスせず1点集中）
  - 媒体費が予算の30%以下の場合 → 2-3媒体組み合わせ可能
  - BtoB・高単価商品 → リスティング広告＋ホワイトペーパー連携が定番（リード獲得目的）
  - BtoC・地域密着 → SNS広告＋看板/交通の地域組み合わせが定番（認知＋来店促進）
  - 全国BtoC・大量認知 → テレビ＋デジタル動画の組み合わせ（予算1,000万円超向け）
- 出力：候補媒体リスト（2-4媒体）＋各媒体への予算配分案

### ステップ4：広告ミックスを構成する

- 入力：候補媒体リスト、予算配分案
- 処理：
  - 主軸媒体を1つ決める（予算配分の40-60%を割り当てる媒体）
  - 補助媒体を1-3つ決める（残り予算を配分）
  - 主軸と補助の役割を明示する：
    - 主軸＝メインの目的達成手段（例：リスティング広告でCV獲得）
    - 補助＝主軸の補完（例：SNS広告でリターゲティング、ディスプレイ広告で認知補強）
  - フェーズ分けが必要な場合は時系列で配置する：
    - 序盤（1ヶ月目）：認知系媒体に予算配分多め
    - 中盤（2-3ヶ月目）：CV系媒体にシフト
    - 終盤：効果の高い媒体に集約
- 出力：広告ミックス構成（主軸媒体／補助媒体／予算配分／フェーズ計画）

### ステップ5：4つの注意点でリスク点検する

- 入力：広告ミックス構成
- 処理：以下の4チェック項目で全媒体を点検する。NG該当があれば対策を併記して構成を見直す
  | # | 注意点 | チェック内容 | 対策例 |
  |---|---|---|---|
  | 1 | 過剰な費用 | 人気媒体・人気キーワードで単価高騰していないか／予算オーバーしないか | 入札上限設定／配信時間帯の絞り込み／除外キーワード設定 |
  | 2 | 信頼性の低下リスク | 誇大表現・薬機法/景表法違反・優良誤認の可能性はないか | 倫理的で正確な訴求／法令チェック（ケンに依頼可）／第三者確認 |
  | 3 | 広告疲れ（Ad Fatigue） | 同一クリエイティブを長期間配信していないか／フリークエンシーが高すぎないか | 月1回のクリエイティブ更新／A/Bテストで複数バージョン用意／フリークエンシーキャップ設定 |
  | 4 | 効果の見極めが難しい | 短期成果のみで判断していないか／KPI設定が単一指標に偏っていないか | 短期KPI（CTR/CPA）と中長期KPI（LTV/ブランド指名検索）の両方設定／月次レビュー |
- 出力：リスク点検結果（OK/NG/対策必要）＋修正後の広告ミックス構成

### ステップ6：KPIと効果測定方法を定義する

- 入力：広告ミックス構成、広告目的
- 処理：
  - 主要KPIを目的別に設定する：
    - 認知度向上 → リーチ数・インプレッション数・ブランド想起率
    - 購買促進 → CV数・売上・ROAS（広告費用対売上）
    - ブランド構築 → ブランド指名検索数・好意度調査・NPS
    - リードジェネレーション → リード数・CPA・MQL/SQL転換率
  - 補助KPIを設定する：CTR・CPC・CPM・滞在時間・直帰率など
  - レポーティング頻度を決める：日次（リスティング/SNS）／週次（ディスプレイ）／月次（マス系・効果総括）
  - A/Bテスト計画を盛り込む：クリエイティブ・配信時間帯・ターゲティング条件のうち何をテストするか
- 出力：KPIシート（主要KPI／補助KPI／目標値／測定頻度／A/Bテスト計画）

### ステップ7：広告ミックス計画書としてまとめる

- 入力：ステップ1〜6の全成果物
- 処理：以下の出力フォーマットに沿って計画書を整形する
- 出力：広告ミックス計画書

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 月予算10万円未満 | デジタル広告（SNS広告／クーポン広告）に集中 | マス系・アウトドア系は単独でも予算不足 |
| 月予算10-50万円 | デジタル中心＋直接型1媒体の補助構成 | 中規模で2-3媒体の組み合わせが効果的 |
| 月予算50-300万円 | デジタル＋アウトドア／雑誌などのミックス | 認知＋CVを両立できる予算帯 |
| 月予算300万円以上 | テレビ含むマス＋デジタルのフルミックス | 全国認知が狙える予算帯 |
| BtoB商材 | リスティング広告＋SNS広告（LinkedIn等）＋ホワイトペーパーDM | 検討期間が長くリード型が定番 |
| BtoC高関与商材（不動産・自動車等） | テレビ／雑誌＋デジタル動画＋リスティング | ブランド構築と最終CV両方が必要 |
| BtoC低関与商材（日用品） | SNS広告＋クーポン広告＋店頭販促連携 | 短期購買促進が中心 |
| 地域密着サービス | 看板/交通＋ローカルSNS広告＋折込チラシ | エリア限定リーチが必要 |
| 緊急セール/期間限定 | リスティング＋SNS＋クーポンで短期集中 | 即効性のある媒体に絞る |
| 業種規制あり（医療・金融等） | デジタル中心で薬機法/金商法対応の媒体に限定 | 法令違反リスク回避が最優先 |
| 過去広告で特定媒体の効果が高かった | その媒体を主軸に据え、補助で別媒体を追加 | 既存実績を活かす |
| 競合が特定媒体を独占している | 別媒体で差別化を図る | 同じ土俵で競うと予算負け |

### 記入例

```markdown
# 広告ミックス計画書 — 月額ITサポート（広島県中小企業向け）

## 1. 広告目的とKPI
- 主目的：認知度向上＋リードジェネレーション
- 主要KPI：月間リード獲得30件／CPA 12,000円
- 補助KPI：リスティングCTR 3%以上／SNS広告CPM 800円以下

## 2. ターゲットと予算
- ターゲット：広島県内・従業員10-50名・40-60代経営者/総務責任者
- 期間総予算：300万円（媒体費70%＝210万円／制作費20%＝60万円／運用費5%＝15万円／予備費5%＝15万円）
- 広告期間：3ヶ月

## 3. 広告ミックス構成
| 種別 | 媒体 | 役割 | 月額予算 | 期間総額 | 主要KPI |
|---|---|---|---|---|---|
| 主軸 | リスティング広告 | リード獲得（指名/比較検討層）| 40万円 | 120万円 | リード20件/月 |
| 補助1 | SNS広告（Facebook/LinkedIn）| 認知＋リターゲティング | 20万円 | 60万円 | リード10件/月 |
| 補助2 | 交通広告（広島電鉄ステッカー）| 地域認知補強 | 10万円 | 30万円 | 想起率5%向上 |

## 4. 時系列フェーズ計画
- 1ヶ月目：交通広告で地域認知形成＋リスティング立ち上げ
- 2ヶ月目：SNS広告でリターゲティング開始、リスティング最適化
- 3ヶ月目：効果上位媒体に予算寄せ、来期計画策定

## 5. リスク点検結果
- [x] 過剰な費用：リスティング入札上限を1クリック500円に設定
- [x] 信頼性低下リスク：「IT運用支援」など事実訴求のみ／薬機法対象外
- [x] 広告疲れ：月1回クリエイティブ更新（バナー3パターン用意）
- [x] 効果の見極め：CPA短期＋指名検索数中長期の両軸で評価、月次レビュー

## 6. 効果測定計画
- レポーティング頻度：リスティング日次／SNS週次／全体月次
- A/Bテスト計画：LP訴求文（「コスト削減」 vs 「業務効率化」）／配信時間帯（平日業務時間 vs 夜間）
- 中間見直しタイミング：開始30日後にミックス比率の再調整

## 7. ネクストアクション
1. リスティング広告のキーワードリスト作成（営業日3日以内）
2. SNS広告クリエイティブ3案制作（コトに依頼）
3. 交通広告の媒体社見積もり取得（5日以内）
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 4目的すべてを同時に追う計画にする | 1-2目的に絞る | 全部追うと予算が分散しどれも中途半端になる |
| 月10万円予算でテレビ広告を提案する | 予算規模に合った媒体だけを候補にする | 媒体費の最低ライン未満では効果が出ない |
| 主軸媒体を決めず全媒体に均等配分する | 主軸40-60%＋補助で構成する | フォーカスがないとどれも検証不能になる |
| 短期KPI（CTR/CPA）だけで効果判定する | 短期＋中長期（LTV/ブランド指名検索）を併用する | 短期最適化が中長期のブランド毀損を見逃す |
| 同一クリエイティブで3ヶ月間配信する | 月1回更新／3バージョンを並行配信する | 広告疲れでCTRが半減以下になる |
| 法令チェックを後回しにする | 薬機法・景表法・金商法該当業種は事前にケンに確認する | 配信停止・行政指導リスクが致命的 |
| 4分類のうち得意な1分類しか提案しない | 4分類×10種類のマトリクスから網羅的に検討する | 視野が狭まり最適解を見逃す |
| 競合と同じ媒体で同じ訴求をする | 媒体or訴求のどちらかで差別化する | 予算劣勢で勝てない構図になる |
| クリエイティブ制作費・運用費を予算に含めない | 媒体費＋制作費＋運用費＋予備費の総額で計画する | 媒体費だけで予算が尽きクリエイティブが質低下 |
