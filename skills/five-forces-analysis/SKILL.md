# five-forces-analysis

ポーター5フォース分析で業界の構造的収益性を評価し、競争優位を確保するための戦略を立案する。「現状把握止まり」を防ぐため、必ず「最大ネック要因への対応戦略」と「3〜5年後の動向予測」まで進める。

## 適用条件

- 使う場面：
  - 新規市場参入の前段階で参入魅力度を評価する
  - 既存事業で価格競争が激化したときの構造分析
  - 業界全体の収益性が下がる兆候を感じた際の構造把握
  - 大手が参入してきたときの自社ポジション再評価
- 使わない場面：
  - マクロ外部環境の整理 → PEST分析（`pest-analysis`）
  - 顧客・競合・自社の3視点 → 3C分析（`3c-analysis`）
  - 内部の経営資源評価 → VRIO分析（`vrio-analysis`）
  - 強み弱み機会脅威の整理 → SWOT分析（`swot-analysis`）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 分析対象業界 | 必須 | 業界・サービス分野 | 「マーケティングスクール業界」「コールセンター業界」 |
| 分析の目的 | 必須 | 参入判断／既存強化／リスク管理 | 「新規参入の可否を判断」 |
| 期間軸 | 任意 | 何年後を見るか | 「3年後／5年後」 |
| 既知の競合・代替品 | 任意 | 業界内主要プレイヤー | 「競合A・B、代替品X・Y」 |

## 実行ステップ

### ステップ1：① 新規参入の脅威を評価する

- 入力：分析対象業界、期間軸
- 処理：以下の観点で評価する
  - **参入障壁の高さ**: 必要資本、技術、ブランド力、規制、既存顧客の囲い込み
  - **想定される新規参入者**: スタートアップ、隣接業界からの侵入、海外勢の上陸
  - **参入インセンティブ**: 業界収益性が高い／成長率が高い → 参入引力大
- 評価ルール：
  - **強**: 参入障壁低 × 業界収益性高 → 新規参入者が増える可能性大
  - **中**: 参入障壁・インセンティブが拮抗
  - **弱**: 参入障壁高 × 業界収益性低 → 新規参入者は限定的
- 出力：新規参入の脅威 = 強／中／弱、根拠1〜3項目

### ステップ2：② 代替品の脅威を評価する

- 入力：分析対象業界、期間軸
- 処理：以下の観点で評価する
  - **代替品の存在**: 同じ顧客ニーズを別手段で満たす商品・サービス
  - **代替品の優位性**: 価格・性能・利便性で代替品が優れているか
  - **スイッチングコスト**: 顧客が代替品に乗り換える際のコスト
  - **技術的代替の可能性**: AI・自動化等で根本から代替される可能性
- 評価ルール：
  - **強**: 代替品が安価・高性能・スイッチングコスト低
  - **中**: 一部代替可能だが完全には代替できない
  - **弱**: 代替品なし／高コストで現実的でない
- 出力：代替品の脅威 = 強／中／弱、根拠1〜3項目

### ステップ3：③ 買い手の交渉力を評価する

- 入力：分析対象業界、既存の顧客構造
- 処理：以下の観点で評価する
  - **顧客の選択肢の多さ**: 競合が多い／少ない
  - **顧客の規模と集中度**: 大口顧客への依存度
  - **スイッチングコスト**: 顧客側の乗り換えコスト
  - **顧客の情報量**: 価格・仕様の比較が容易か
  - **顧客の価格感応度**: 価格で意思決定するか、品質で意思決定するか
- 評価ルール：
  - **強**: 競合多 × 大口顧客集中 × スイッチングコスト低 → 顧客が価格を決める
  - **中**: 一部の顧客に交渉力がある
  - **弱**: 競合少 × 顧客分散 × スイッチングコスト高 → 自社が価格を決められる
- 出力：買い手の交渉力 = 強／中／弱、根拠1〜3項目

### ステップ4：④ 売り手の交渉力を評価する

- 入力：分析対象業界、サプライチェーン構造
- 処理：以下の観点で評価する
  - **供給元の集中度**: 仕入先が少数か多数か
  - **代替供給元の有無**: 別の仕入先に切り替え可能か
  - **供給元の重要度**: その供給元なしでは事業が成立しないか
  - **垂直統合のリスク**: 供給元が下流に進出してくる可能性
- 評価ルール：
  - **強**: 供給元少数 × 代替なし → 仕入価格は供給元が決める
  - **中**: 一部の供給元に交渉力がある
  - **弱**: 供給元多数 × 代替容易 → 自社が仕入価格を決められる
- 出力：売り手の交渉力 = 強／中／弱、根拠1〜3項目

### ステップ5：⑤ 業界内の競争を評価する

- 入力：分析対象業界、競合情報
- 処理：以下の観点で評価する
  - **競合の数と規模**: 多数の中小か、少数の大手か
  - **市場成長率**: 拡大期（緩やかな競争）／成熟期（激しい奪い合い）
  - **差別化の難易度**: コモディティ化しているか
  - **固定費の高さ**: 高いほど稼働率維持のため値下げ競争になりやすい
  - **撤退障壁**: 高いほど赤字でも残り続け競争が続く
- 評価ルール：
  - **強**: 同質競合多 × 成熟市場 × 差別化困難 → 価格競争激化
  - **中**: 競争はあるが差別化余地あり
  - **弱**: 競合少／成長市場 → 個社が独占的価格設定可能
- 出力：業界内の競争 = 強／中／弱、根拠1〜3項目

### ステップ6：5フォースを統合して最大ネック要因を特定する

- 入力：ステップ1〜5の評価結果
- 処理：以下を実施する
  - 5フォースを評価マトリクスに整理（強／中／弱を視覚化）
  - **最大ネック要因（強と評価されたもの）を特定**: 業界収益性を最も圧迫している要因
  - **複数フォースが強の場合**: 業界全体が「魅力度低い」状態。参入は慎重に
  - **3年後／5年後の動向予測**: 各フォースが「強くなる／弱くなる／変わらない」を予測
- 4つの戦略視点で対応策を立案：
  - **真っ向勝負**: 最大ネック要因に正面から対抗（差別化・規模拡大・ブランド構築）
  - **勝負を避ける**: ニッチ市場へ移動／撤退判断
  - **分析対象の絞り込み変更**: 市場セグメントを切り直して再評価
  - **長期視点での先取り**: 3〜5年後の変化を先取りした戦略
- 出力：最大ネック要因＋戦略視点（2〜3個）

### ステップ7：5フォース分析シートを完成させる

- 入力：ステップ1〜6の全成果物
- 処理：以下のテンプレートに統合する
- 出力：5フォース分析完成シート

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 各フォースの評価が「中」ばかり | 評価軸を明確にして強／弱に振り分け直す | 「中」だらけは判断材料にならない |
| 評価根拠が抽象的（「多分」「感覚」） | 競合数・シェア・価格データ等で裏付ける | 主観評価の5フォースは形骸化 |
| 5フォースのうち4つ以上が「強」 | 業界全体が魅力薄い → 参入検討の場合は撤回も視野 | 構造的に儲からない市場に時間をかけない |
| 最大ネック要因が「業界内の競争」 | 差別化戦略 or ニッチ移動を最優先 | コモディティ化からの脱出が最重要 |
| 最大ネック要因が「新規参入の脅威」 | 参入障壁を高める施策（ブランド・特許・契約等）を最優先 | 既存ポジション防衛が最重要 |
| 最大ネック要因が「代替品の脅威」 | 顧客の真のジョブを再定義（Jobs to Be Done視点） | 代替品との真の戦いは「機能」ではなく「ジョブ」 |
| 最大ネック要因が「買い手の交渉力」 | 大口依存からの脱却・スイッチングコスト引き上げ | 価格決定権を取り戻す |
| 最大ネック要因が「売り手の交渉力」 | 供給元の多様化・内製化検討 | 単一供給元依存は経営リスク |
| 3年後の予測ができない | データ不足。業界レポート・専門家インタビューで補強 | 構造分析は将来予測も含めて完成 |
| 1度作ったら見直さない | 半年〜1年で再評価 | 業界構造は変わり続ける |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 全フォースを「中」評価で済ませる | 強／中／弱で必ず振り分ける | 中だらけは判断不能 |
| 競合の数や市場規模を調べずに評価 | 業界レポート・統計で裏付ける | 主観の5フォースは形骸化 |
| 5フォースだけで終わる | 最大ネック要因＋戦略視点まで進める | 構造分析は対応策まで含めて完成 |
| 最大ネックを正面突破しか考えない | 「真っ向勝負／回避／絞込／長期」の4視点で検討 | 撤退・回避も立派な戦略 |
| 3年後予測を省略 | 各フォースが強くなる／弱くなる予測を必ず入れる | 静的分析だけだと将来予測ができない |
| 1度作って放置 | 半年〜1年で再評価 | 業界構造は動く |
| 5フォースが4つ以上強でも参入する | 構造的に儲からない市場 → 撤退も検討 | 戦略は「やらないこと」を決めることでもある |
