# ma-strategy-basics スキル

M&A（Mergers and Acquisitions＝企業の合併・買収）戦略の基礎を体系的に理解し、意思決定を支援する。事業戦略スクール「知足」が定義する2分類／5目的／4リスク／5手法／6プロセス／4検討状態／5成功ポイントで構造化し、事業成長・市場拡大・競争優位性確立・事業承継の選択肢としてのM&Aを正しく検討できるようにする。

## 適用条件

- このスキルを使う場面：
  - M&Aの基礎概念（合併と買収の違い、種類、手法）を理解したい
  - 自社がM&Aを検討すべき状態にあるか診断したい
  - M&Aの目的・リスク・成功ポイントを整理したい
  - M&Aの6プロセス（戦略立案〜PMI）の全体像を把握したい
  - 事業承継の選択肢としてM&Aを検討したい
- このスキルを使わない場面：
  - デットファイナンス／エクイティファイナンスの選択 → `financing-strategy` を使う
  - 財務諸表（BS/PL/CF）の基礎理解 → `financial-statements-fundamentals` を使う
  - 個別プロジェクトの収支管理 → `project-finance-contract` を使う
  - 中長期事業計画 → `medium-term-business-plan` を使う
  - 詳細なデューデリジェンス実務（法務・税務）→ 専門家（弁護士・税理士・M&Aアドバイザー）に委任

## 実行ステップ

### ステップ1：M&Aの定義と2分類を確認する

**M&A（Mergers and Acquisitions）** = 企業の合併や買収を指し、事業成長・市場拡大・シナジー創出などを目的として実施される。

| 分類 | 内容 |
|---|---|
| 合併（Merger） | 2つ以上の企業が統合され、1つの新しい企業として運営される |
| 買収（Acquisition） | ある企業が他の企業の株式や資産を取得し、支配権を得る |

**情報源（参考）**：
- MARR Online（https://www.marr.jp/）
- 日本M&Aセンター
- 各社のM&A速報サイトで日々の事例を確認可能

### ステップ2：M&Aの5目的とメリットを整理する

| # | 目的 | 内容 |
|---|---|---|
| 01 | 事業拡大 | 新規市場への参入、シェア拡大 |
| 02 | 成長スピードの加速 | ゼロからの事業立ち上げに比べ、短期間で成長可能 |
| 03 | シナジー効果の創出 | 営業力・技術力・資源の統合による相乗効果 |
| 04 | 競争優位性の確立 | 競合他社との差別化、市場での優位性強化 |
| 05 | コスト削減 | スケールメリットによるコスト削減 |

**最終目的**：自社の競争力を高め、持続的に成長・生存していくこと。特に変化の激しい現代では、スピードと柔軟性を持って成長するための戦略的手段としてM&Aの重要性が高まっている。

### ステップ3：M&Aの4リスクとデメリットを認識する

| # | リスク | 内容 |
|---|---|---|
| 01 | 統合の失敗 | 組織文化の違いやコミュニケーション不足による統合不全 |
| 02 | 過大な買収コスト | 企業価値の過大評価による投資回収の困難化 |
| 03 | シナジー効果の未達 | 期待した効果が得られないケース |
| 04 | 財務・法務リスク | 未払い債務や訴訟リスクの見落とし |

### ステップ4：M&Aの5手法から適切なものを選定する

| 種類 | 英名 | 手法 |
|---|---|---|
| 水平統合 | Horizontal Integration | 同業種の企業を統合し、市場シェアを拡大する |
| 垂直統合 | Vertical Integration | サプライチェーン内の企業を買収し、供給の安定化やコスト削減を図る |
| コングロマリット型 | Conglomerate M&A | 異業種の企業を買収し、事業の多角化を図る |
| MBO | Management Buyout | 経営陣が自社の株式を取得し、オーナーシップを持つ |
| LBO | Leveraged Buyout | 借入金を活用して企業買収を行う |

### ステップ5：M&Aの6プロセスを実行する

| # | プロセス | 内容 |
|---|---|---|
| 1 | 戦略立案 | M&Aの目的を明確化し、対象企業の選定基準を策定 |
| 2 | ターゲット企業の選定・リサーチ | 財務状況、事業内容、シナジー効果を検討 |
| 3 | 初期交渉と意向表明 | 基本条件を提示し、買収の意向を示す（LOI／IM等） |
| 4 | デューデリジェンス（Due Diligence） | 財務・法務・事業リスクの詳細調査を実施 |
| 5 | 契約締結・クロージング | 最終合意後、契約を締結し、取引を完了 |
| 6 | PMI（Post Merger Integration） | 統合後の組織・業務・文化の融合を進める |

**重要**：6プロセスを順序どおり進めること。特にDD（4）を省略すると財務・法務リスクの見落としで致命傷になる。

### ステップ6：M&Aを検討すべき4状態を診断する

| # | 状態 | 内容 |
|---|---|---|
| 01 | 事業成長の限界 | 新規・既存事業の売上が伸び悩み、次の打ち手が見えない |
| 02 | 収益化や販路の課題 | 会員や技術はあるが、マネタイズや営業面でシナジーを活かせていない |
| 03 | 経営リソースの不足 | 人材・資金・体制に限界があり、自力での成長が困難 |
| 04 | 事業承継・戦略的な出口の必要性 | 後継者不在や将来の不安に備え、事業譲渡を視野に入れたい場合 |

### ステップ7：成功するM&Aの5ポイントを徹底する

| # | ポイント |
|---|---|
| 01 | 明確な戦略と目的を設定する |
| 02 | 適切なデューデリジェンスを行う |
| 03 | シナジー効果を最大限活用する |
| 04 | PMIを徹底し、スムーズな統合を実現する |
| 05 | 財務リスクと法務リスクを慎重に管理する |

### ステップ8：実例から学ぶ（参考事例）

**大手企業の事例**：
- 日本製鉄による米国USスチールの買収 → 財務健全性とシナジー効果を重視
- ローソンのKDDIと三菱商事による共同経営 → 異業種協業による組織文化の統合
- 駅探による広告テクノロジー企業の買収 → グループ全体ロードマップの可視化、自社メディア収益力向上、技術・人材獲得、リスク分散

**中小・ベンチャー企業の事例**：
- エアトリによるノックラーンの子会社化 → 旅行事業とのシナジー（採用支援サービス領域の拡大）
- ジオテクノロジーズによるGeoloniaの買収 → 位置情報サービス強化（地理空間データ×技術力の融合）
- LTSによるイオトイジャパンの買収 → DX支援とIoT技術の組み合わせ

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 事業成長の限界＋潤沢な資金 | 買い手側M&A検討（水平統合／コングロマリット型） | 自前成長より速い／既存リソース活用 |
| 後継者不在＋赤字ではない | 売り手側M&A検討（事業承継型） | 廃業より価値が残る／従業員雇用維持 |
| 自社事業のサプライチェーン依存度が高い | 垂直統合型M&A検討 | 供給安定化・コスト削減効果 |
| 経営陣が自社を引き継ぎたい | MBO検討 | オーナーシップ移転で経営自由度向上 |
| 自己資金不足だが買収機会あり | LBO検討（ただしリスクも高い） | 借入レバレッジで取引成立 |
| シナジーが定量化できない | M&A保留（自前成長検討） | シナジー仮説なきM&Aは失敗確率高い |
| ターゲットの組織文化が極端に異なる | 慎重判断（PMI失敗リスク高） | 統合失敗が最大リスク |
| DD省略を提案された | 即座に拒否、必ずDD実施 | 財務・法務リスク見落としの致命傷 |
| 「競合がいない／独自事業だから買収は安全」と楽観判断 | `competitive-absence-audit` で6仮説検証 | 競合不在は需要不在の可能性、事業価値過大評価リスク |
| 中小・ベンチャーで初めてのM&A | M&A仲介会社・FA（フィナンシャルアドバイザー）に委任 | 自社単独でのDD・交渉は失敗確率高い |
| PMI計画なしで契約締結直前 | 契約前にPMI計画を必ず策定 | 統合失敗の最大原因 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 「シナジーが出そうだから」だけでM&Aを進める | シナジー効果を売上・コスト・人材で定量化してから判断 | シナジー仮説なきM&Aは未達率が極めて高い |
| デューデリジェンスを省略or形式的に済ます | 財務・法務・事業の3DDを必ず実施 | 未払い債務・訴訟リスクの見落としで致命傷 |
| 契約締結時にPMI計画がない | 契約前にPMI 100日プランを策定 | 統合失敗の最大原因 |
| 売り手企業の組織文化を軽視する | 文化適合性を交渉初期から評価 | 文化不一致は人材流出・統合不全の主因 |
| 経営者の経験だけでM&Aを進める | M&A仲介会社・FA・弁護士・税理士の専門家チームを組成 | 中小・ベンチャー初M&Aの失敗パターン頻発 |
| LBO で過剰な借入レバレッジをかける | 返済可能性を保守シナリオで検証 | LBO失敗で買収企業ごと共倒れリスク |
| 売り手側で「うちは独自事業で競合がいないから高値で売れる」と楽観 | `competitive-absence-audit` で需要存在を検証 | 競合不在は需要不在の可能性、買い手評価で過小評価される |
| PMIを「契約後に考える」と先延ばし | 契約前から PMI 計画策定、CFO/PMI責任者を事前任命 | PMI不在で人材流出・業績悪化が即発生 |
| 一度のM&Aで多くの目的を同時達成しようとする | 第1優先目的を1つに絞る | 目的が分散するとシナジー判断・PMI設計が破綻 |
| 「うちは小規模だから簡単に売れる／買える」と楽観 | 中小M&A市場でも、財務透明性・組織体制が評価対象 | 売り手の準備不足は買い手から大幅減額対象 |

## 連携スキル

- `financial-statements-fundamentals`：BS/PL/CF基礎を理解し、ターゲット企業の財務分析に活用
- `financing-strategy`：M&A資金調達（デット／エクイティ）の選択
- `medium-term-business-plan`：M&A後の中長期計画策定
- `growth-phase-strategy`：自社の成長フェーズに応じたM&A戦略選択
- `mvv-design`：被買収企業との文化統合（PMIにおけるMVV再策定）
- `vrio-analysis`：買収ターゲットのリソース・能力の競争優位性評価
- `swot-analysis`：M&A後の自社SWOT再構築
- `competitive-absence-audit`：M&A戦略における「独自事業・競合不在」の楽観判断検証
- `kyt`：M&A実行前のリスク予知（特に財務・法務・統合リスク）

## 5状態出力契約

このスキルは外部API・サブプロセス実行・外部設定依存のいずれも持たないため、CLAUDE.md「スキル出力の5状態契約」の対象判定基準には該当しない。ただし以下のケースで明示マーカーを使うこと：

- `[OK:0件]`：自社の状況・M&A目的が定まっておらず、フレームワーク説明のみで終了した場合
- `[SKIP]`：競合不在の楽観判断が出た場合は `competitive-absence-audit` スキルへの誘導でスキップ／詳細DDは専門家委任が必要として手前で停止

外部API・サブプロセスを持たないため、`[FAIL]`（試行後の失敗）は通常発生しない。
