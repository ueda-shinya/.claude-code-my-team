# pest-analysis

PEST分析（4要因：政治／経済／社会／技術）で外部マクロ環境を体系的に整理し、戦略視点（実行可能な戦略方針）を2〜3個導出する。「環境を眺めて終わり」を防ぐため、必ず戦略視点まで進める。

## 適用条件

- 使う場面：
  - 新規市場参入の前段階で外部環境を評価する
  - 既存事業の戦略見直しで外部リスク・機会を再評価する
  - 中期計画の前提整理（マクロ環境の変化を捉える）
  - 差別化戦略立案の素材
- 使わない場面：
  - 競争構造の深掘り → 5フォース分析（`five-forces-analysis`）
  - 顧客・競合・自社の3視点 → 3C分析（`3c-analysis`）
  - 内部の強み・弱みの整理 → SWOT分析（`swot-analysis`）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 分析対象 | 必須 | 業界・事業・サービス | 「建設業界」「自社のIT支援事業」 |
| 分析の目的 | 必須 | 新規参入／戦略見直し／中期計画 | 「中期計画の前提整理」 |
| 期間軸 | 任意 | 何年後を見るか | 「3〜5年後」 |
| 既知の外部要因 | 任意 | 法改正・市場動向の予備知識 | 「2026年から〇〇法施行予定」 |

## 実行ステップ

### ステップ1：P（政治）要因を分析する

- 入力：分析対象、期間軸
- 処理：以下の観点で2〜4項目を整理する
  - **法規制の強化／緩和**: 直近・予定の法改正、業界規制の変化
  - **政府政策・補助金**: 業界に影響する公共投資、補助金、税制改定
  - **貿易規制・国際関係**: 関税変更、輸出入規制、地政学リスク
  - **労働政策**: 最低賃金、外国人労働者、働き方改革関連
- 各項目に「自社事業へのプラス／マイナス影響」を1行で添える
- データソース：官公庁発表、業界団体レポート、Newsピックアップ
- 出力：P 要因リスト

### ステップ2：E（経済）要因を分析する

- 入力：分析対象、期間軸
- 処理：以下の観点で2〜4項目を整理する
  - **景気動向**: GDP成長率、業界の市場規模推移
  - **為替・物価**: 円安／円高の影響、原材料価格、インフレ率
  - **金利・調達コスト**: 金利動向、資金調達難易度
  - **購買力・所得水準**: 顧客の可処分所得、価格感応度の変化
- 各項目に「自社事業へのプラス／マイナス影響」を1行で添える
- 出力：E 要因リスト

### ステップ3：S（社会）要因を分析する

- 入力：分析対象、期間軸
- 処理：以下の観点で2〜4項目を整理する
  - **人口動態**: 少子高齢化、地方人口減少、生産年齢人口
  - **文化・価値観の変化**: SDGs、ESG、サステナビリティ意識
  - **ライフスタイル変化**: リモートワーク、消費行動、健康志向
  - **教育・スキル**: 人材不足、技術継承、リスキリング動向
- 各項目に「自社事業へのプラス／マイナス影響」を1行で添える
- 出力：S 要因リスト

### ステップ4：T（技術）要因を分析する

- 入力：分析対象、期間軸
- 処理：以下の観点で2〜4項目を整理する
  - **技術革新**: 業界に影響するAI、IoT、自動化、新材料等
  - **デジタル化の進展**: DX、SaaS、クラウド、ノーコードツール
  - **業界特化技術**: その業界での技術トレンド（建設ならBIM/ICT等）
  - **技術導入率**: 業界平均の導入率、競合の技術投資状況
- 各項目に「自社事業へのプラス／マイナス影響」を1行で添える
- 出力：T 要因リスト

### ステップ5：4要因を統合し戦略視点を2〜3個導出する

- 入力：ステップ1〜4の P / E / S / T 全リスト
- 処理：
  - 4要因を一覧化し、影響度（大/中/小）と発生確率（高/中/低）でスコアリング
  - 影響度大×確率高の項目から優先的に着目
  - 4要因の組み合わせから戦略視点を抽出する：
    - **機会型戦略視点**: P/E/S/T のいずれかで「自社にプラス」要因を活かす方向性
    - **リスク回避型戦略視点**: 「自社にマイナス」要因を回避・緩和する方向性
    - **差別化戦略視点**: 競合がまだ対応していない P/E/S/T 変化を先取りする方向性
- 戦略視点は2〜3個に絞る（多すぎると優先順位が消える）
- 各戦略視点に「KPI候補」「アクション着手時期」を添える
- 出力：戦略視点 2〜3個

### ステップ6：PEST分析シートを完成させる

- 入力：ステップ1〜5の全成果物
- 処理：以下のテンプレートに統合する
- 出力：PEST分析完成シート

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 4要因のうち1つが項目ゼロ | 必ずその要因も2項目以上は埋める | 偏った分析は戦略視点が片寄る |
| 「影響不明」「分からない」が多い | データソースを必ず明記する（官公庁・業界レポート・ニュース） | データなき分析は形骸化 |
| 影響度・確率の評価が主観 | 過去の類似事例・統計データで裏付ける | 主観評価は将来予測を狂わせる |
| 戦略視点が「がんばる」「対応する」 | 動詞＋目的語＋数値で書き直す | 抽象戦略は実行不能 |
| 戦略視点が4個以上ある | 2〜3個に絞る | 多すぎると優先順位が消える |
| プラス影響が0個 | 視点を変えて「変化を機会と捉える」観点を追加 | プラス0は分析の偏り |
| マイナス影響が0個 | 楽観バイアスのチェックを行う | マイナス0は典型的な過大評価 |
| 1度作って放置 | 半年に1回は再作成 | マクロ環境は変わり続ける |

### 記入例（建設業界の例・要約）

```
## 1. P：①公共投資・国土強靭化計画→公共土木需要増（プラス・大・高）／②労働安全規制強化（マイナス・中・高）
## 2. E：①経済成長鈍化で民間土木需要減（マイナス・中・中）／②建設資材価格上昇（マイナス・大・高）
## 3. S：①少子高齢化で熟練労働者不足（マイナス・大・高）／②環境意識・SDGs対応需要増（プラス・大・高）
## 4. T：①ICT/BIM導入進展で生産性向上（プラス・大・高）／②エコ建材・新工法（プラス・中・中）

## 5. 戦略視点
1. 機会型：「デジタル化×自動化技術導入で生産性向上、業務効率化」（KPI：BIM導入率50%、人時生産性+20%、短期）
2. 差別化型：「持続可能建材・環境技術で差別化（SDGs認証）」（KPI：環境認証案件比率30%、中期）
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 4要因のうち得意な領域だけ書く | 4要因とも最低2項目ずつ埋める | 偏った分析は戦略視点が偏る |
| 「影響あり」止まり | プラス／マイナスを明示する | 方向性が見えないと戦略に繋がらない |
| データソースを書かない | 官公庁・業界レポート等の参照元を明記 | 主観の塊になる |
| 影響度・確率を評価しない | 大/中/小、高/中/低 で必ずスコア | 優先順位が出せない |
| 戦略視点が「対応する」「強化する」 | 動詞+目的語+数値で書く | 抽象戦略は実行不能 |
| プラス／マイナスが偏る | バイアスチェック（楽観／悲観の偏り）を行う | 偏った分析は誤判断に繋がる |
| 1度作って放置 | 半年〜1年で再作成 | マクロ環境は変わる |
| マクロ環境のみで完結 | 必ずSWOT・3Cと組み合わせて使う | PESTは外部環境の素材。単独では戦略にならない |
