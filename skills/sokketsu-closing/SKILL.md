# sokketsu-closing

即決営業の3コツを1本のクロージング台本に統合する。

**商談フェーズ位置**：クロージング（提案完了後〜成約）

## 適用条件

- 使う場面：短時間・即決型商談のクロージング、BtoC個人向け、決裁者同席・その場で決められる状況
- 使わない場面：ヒアリング段階、決裁権者不在、複数回検討型の高額BtoB
- 棲み分け：高関与・高単価で顧客にじっくり言語化させたい場合は `self-closing-4questions` を使う。本スキルは「その場で即決させる」用途に特化

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 商材/サービス名 | 必須 | クロージング対象 | 英会話スクール |
| 主要ベネフィット | 必須 | 相手の生活/業務がどう変わるか | 半年で英語面接が通る |
| 顧客タイプ | 任意 | BtoC/BtoB、個人/法人 | BtoC個人 |
| 価格/スタート条件 | 任意 | 初月価格・キャンペーン等 | 月額1万円 |

## 実行ステップ

### ステップ1：先回り（断る自由）のセリフを生成する
- 入力：商材
- 処理：
  - 型「〇〇様、これから商品のことをお伝えするんですが、全然ダメだ・いらないなと思ったら、バッサリ断ってもらって大丈夫ですから安心してくださいね」
  - 相手から「はい」を取るまで次に進まない指示を注記する
- 出力：先回り1文目

### ステップ2：先回り（始める同意）のセリフを生成する
- 入力：商材
- 処理：
  - 型「もし、これ自分に必要だなと思っていただけたら、ぜひスタートしてくださいね」
  - 「はい」を取る指示を注記する
  - 1文目と2文目を必ず分けて言う（1文にまとめない）
- 出力：先回り2文目

### ステップ3：if質問を3〜5個生成する
- 入力：商材、主要ベネフィット
- 処理：
  - 型A「もし〇〇様がこの商品をご購入いただいたら、生活はどう変わりそうですか？」
  - 型B「もしここに置くとしたら、どこが最適ですか？」
  - 必ず3個以上・最大5個生成する（型A・型Bのバリエーションで数を作る）
  - 「脳は想像と現実を区別しない」を注記として入れる
- 出力：if質問リスト

### ステップ4：刺さる理由を引き出す質問を生成する
- 入力：商材
- 処理：
  - 質問1「本日お話を聞いていただいて、どこのポイントが良かったと思っていただけましたか？」
  - 質問2「なぜそのポイントが良いと思われたんですか？」
  - 2段階必ずセットで配置する
- 出力：理由引き出し2質問

### ステップ5：顧客の言葉で締めるクロージング文を生成する
- 入力：ステップ4の想定回答
- 処理：
  - 型「〇〇様がおっしゃった〇〇の効果が得られるので、ぜひ今スタートしましょう」
  - プレースホルダー〇〇は顧客発言をそのまま差し込む指示を明記
- 出力：最終クロージング文

### ステップ6：3コツ統合台本として出力する
- 入力：ステップ1〜5
- 処理：先回り→if質問→刺さる理由→最終クロージングの順に並べ、各ブロックに「ここで相手の反応を待つ」指示を挿入する
- 出力：完成クロージング台本

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 相手が先回りで「はい」と言わない | ステップ2に進まず、不安要素をヒアリングに戻る | 2段階Yesが取れないとif活用が効かない |
| if質問で無言 | 質問を「今の生活で〇〇だと困りますよね？」に切替 | 想像が苦手な相手には過去の痛み側から |
| 刺さる理由で営業が先に言う | 絶対に言わない。相手が言語化するまで待つ | 自分で言った理由しか刺さらない |
| 価格の話が先に出る | ステップ4まで価格は出さない | 理由を先に固めるため |

## 出力フォーマット

```
【ステップ1：断る自由】
〇〇
（相手の「はい」を待つ）

【ステップ2：始める同意】
〇〇
（相手の「はい」を待つ）

【ステップ3：if質問（3〜5個）】
1. 〇〇
2. 〇〇
3. 〇〇

【ステップ4：刺さる理由を引き出す】
質問1: 〇〇
質問2: 〇〇
（相手の回答を正確にメモする）

【ステップ5：顧客の言葉でクロージング】
「〇〇様がおっしゃった【相手の言葉】の効果が得られるので、ぜひ今スタートしましょう」
```

### 記入例

```
【ステップ1：断る自由】
「田中様、これから商品のことをお伝えするんですが、全然ダメだ・いらないなと思ったら、バッサリ断ってもらって大丈夫ですから安心してくださいね。」

【ステップ2：始める同意】
「もし、これ自分に必要だなと思っていただけたら、ぜひスタートしてくださいね。」

【ステップ3：if質問】
1. もし田中様がこの英会話を始めたら、半年後の生活はどう変わりそうですか？
2. もし英語で面接が通ったら、一番やりたい仕事は何ですか？
3. もし毎朝30分学習時間を取るとしたら、どの時間帯が最適ですか？

【ステップ4：刺さる理由】
質問1: 本日お話を聞いていただいて、どこのポイントが良かったと思っていただけましたか？
質問2: なぜそのポイントが良いと思われたんですか？

【ステップ5：クロージング】
「田中様がおっしゃった【毎日30分で無理なく続けられる】という効果が得られるので、ぜひ今スタートしましょう。」
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 先回りを1文にまとめる | 必ず2文に分けて2段階Yesを取る | 一貫性の法則が2段階で働く |
| if質問が1〜2個で終わる | 最低3個、できれば5個 | 想像が浅いと感情が揺れない |
| 営業が「こういう点が良かったですよね」と代弁 | 顧客自身に言わせる | 自分の言葉でないと刺さらない |
| 価格を先に出す | 刺さる理由が固まってから | 価格が先だと理由が霞む |
