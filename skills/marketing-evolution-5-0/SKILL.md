# コトラーのマーケティング 5.0 診断スキル

マーケティング1.0〜5.0の進化論と AI時代の新要素を使って、自社のマーケティング成熟度を診断し、次フェーズへの移行課題を特定する。「いい製品を作れば売れる」（1.0）から「AI×人間中心」（5.0）まで、自社が今どこにいるかを言語化する。

## 適用条件

- このスキルを使う場面：
  - 自社のマーケティング成熟度を客観評価したい
  - AI・生成AI活用の方向性を検討したい
  - マーケ予算の配分判断（旧フェーズ施策 vs 新フェーズ施策）
  - 経営層への現状説明（マーケ進化のどこにいるか）
- このスキルを使わない場面：
  - 具体的なマーケ施策設計 → `marketing-planner` 関連スキル
  - PMF段階の診断 → `pmf-journey`
  - イノベーター層別プロモ → `innovator-theory`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 起動フレーズ | 必須 | 上記トリガー語 | 「マーケ5.0で診断」 |
| 商材 | 必須 | 製品・サービス | 「BtoB SaaS」「BtoCコスメ」 |
| 現在のマーケ施策 | 必須 | 何をやっているか | 「テレビCM中心」「Web広告＋SNS」 |
| 顧客接点 | 任意 | チャネル構成 | 「店舗のみ」「EC＋店舗」「オムニチャネル」 |
| データ活用度 | 任意 | データ・AI活用状況 | 「ほぼ未活用」「CRMあり」「AI予測実装済み」 |

## 実行ステップ

### ステップ1：マーケティング進化論5世代の特徴を提示

| 世代 | 別名 | 時期 | 特徴 |
|---|---|---|---|
| **マーケティング 1.0** | 製品志向（プロダクトアウト） | 19世紀後半〜20世紀初頭 | 高品質・効率生産が焦点。一方向コミュニケーション。競合少なく独占的 |
| **マーケティング 2.0** | 販売志向 | 1920年代〜1950年代 | 強力販促・営業活動。マスマーケ拡大。在庫消化型・価格訴求 |
| **マーケティング 3.0** | マーケットイン志向 | 1950年代〜1970年代 | 市場調査基準・STP戦略導入。ブランドと共感の関係づくり |
| **マーケティング 4.0** | 社会志向 | 1970年代〜1990年代 | CSR重視・SNS共創・「誰がどう作ったか」評価 |
| **マーケティング 5.0** | デジタル・人間中心 | 2000年代〜現在 | AI×データ分析×人間中心。オムニチャネル。ブロックチェーン／IoT |

### ステップ2：自社マーケの現フェーズ判定

- 入力：現在のマーケ施策、顧客接点、データ活用度
- 処理：以下の5問YES/NO診断で判定する：

| 問 | YES条件 |
|---|---|
| Q1: 「いい商品を作れば売れる」発想で動いているか？ | YESなら 1.0 |
| Q2: 営業・広告の押し売りが中心か？ | YESなら 2.0 |
| Q3: 市場調査・STPに基づき、ターゲット別に商品を作っているか？ | YESなら 3.0 |
| Q4: CSR・社会的価値・SNSコミュニティ運営をしているか？ | YESなら 4.0 |
| Q5: AI・データ分析で顧客行動を予測しオムニチャネル提供しているか？ | YESなら 5.0 |

複数該当する場合は**最も上位の世代**を採用する。AI時代の新要素（生成AI・メタバース）対応も判定する。

### ステップ3：マーケティング5.0の4要素チェック

5.0判定時は以下4要素のうちどれが充足／不足かをチェックする：

| 要素 | 充足条件 | 充足/不足 |
|---|---|---|
| **テクノロジーの活用** | AI・データ分析で消費者行動予測／パーソナライズ実施 | ◯/△/× |
| **人間中心マーケティング** | 感情的つながり構築／SNS双方向対話・ブランド忠誠心育成 | ◯/△/× |
| **多様なチャネルとプラットフォーム** | オフライン×オンラインのオムニチャネル戦略・チャネル間一貫性 | ◯/△/× |
| **イノベーションと変革** | ブロックチェーン・IoT等で透明性・信頼性提供 | ◯/△/× |

### ステップ4：AI時代の新要素チェック（5.0+α）

コトラー提唱から数年経過しているため、現在は以下の追加要素が必要：

| 新要素 | チェック内容 | 充足/不足 |
|---|---|---|
| **生成AI活用** | コンテンツ生成・カスタマーサポート・パーソナライズに生成AI使用 | ◯/△/× |
| **AIとリアルタイム最適化** | 個別体験の最適化・完全自動化（広告入札・レコメンド等） | ◯/△/× |
| **データプライバシー対応** | GDPR・個人情報保護法対応・倫理的データ活用 | ◯/△/× |
| **メタバース・新インターフェース活用** | VR/AR・音声インターフェース等の新接点開拓 | ◯/△/× |

### ステップ5：次フェーズへの移行課題を特定

- 入力：ステップ2の現フェーズ判定、ステップ3〜4の充足度
- 処理：
  1. 1.0〜3.0判定の場合 → 4.0／5.0の差分を移行課題として明示
  2. 4.0判定の場合 → 5.0の4要素＋AI新要素を移行課題として明示
  3. 5.0判定だが要素不足の場合 → 不足要素を順番に追加することを推奨
- 出力：優先実施すべき施策3〜5項目（影響度×実現可能性で優先順位付け）

### ステップ6：完了報告

ステップ1〜5を以下フォーマットで一括出力する。

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 「うちは5.0です」と即答する経営者 | 4要素・AI新要素チェックで実態評価 | 自己評価と実態がズレやすい |
| 1.0〜2.0と判定された企業 | 急に5.0を目指さず、3.0（市場調査・STP）から段階的に | 段階飛ばしは実装失敗 |
| AIを「使う」だけで5.0と称する | 「AI活用×人間中心」両立を求める | AI偏重は5.0の本意でない |
| BtoB企業 | 4.0（社会価値・コミュニティ）まで到達しているケースが多い | BtoBは5.0の感情的つながり要素より信頼性重視 |
| 中小企業で予算が限られる | 5.0全要素より、4要素のうち1つに集中投資を推奨 | 全方位は失敗 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 「うちは5.0です」を鵜呑み | 4要素・AI新要素で実態評価 | 自称と実態がズレる |
| 1.0から急に5.0を目指す | 3.0→4.0→5.0と段階的に | 飛ばすと組織が追いつかない |
| AI導入＝5.0と短絡 | AI×人間中心の両立を求める | 偏重は本意違い |
| 5.0全要素を一斉実装 | 4要素のうち1〜2つに集中投資 | 全方位は中小企業では失敗 |
| マーケティング進化論を歴史知識で終わらせる | 必ず自社診断と次フェーズ施策に落とす | 知識のための知識は無価値 |
| BtoBに5.0感情マーケを強要 | BtoBは4.0社会価値＋3.0信頼性重視で十分なケースも | 業態差を無視しない |

## 連携スキル

- **前段／併用**：`pmf-journey`（PMF段階）、`innovator-theory`（顧客層）
- **同時併用**：`stp-analysis`（ターゲット精緻化）、`value-proposition`
- **後段**：施策設計はマーケ実行系スキル（marketing-planner エージェント経由）

## 5状態出力契約

外部API・サブプロセス・外部設定依存なし。運用上のマーカー：
- `[FAIL]`：必須入力（商材・現在のマーケ施策）が欠落
- `[OK:0件]`：診断不能（情報不足）
