# 市場調査・競合調査スキル

市場（顧客ニーズ＋市場規模）と競合（自社の立ち位置＋勝てる領域）を体系的に調査し、戦略立案の基礎データを取得する。定性／定量の2軸 × オン／オフの2軸で18種の手法を組み合わせる。

## 適用条件

- このスキルを使う場面：
  - 新規事業立ち上げ前の市場・競合理解
  - 既存事業の差別化ポイント再考
  - LP・営業資料リニューアル前の競合FV調査
  - 市場規模の概算（億円単位）が必要
- このスキルを使わない場面：
  - 流入減少の原因究明 → `ga-gsc-diagnosis`
  - SEO監査 → `seo-audit`
  - LLMO監査 → `llmo-audit`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 起動フレーズ | 必須 | 上記トリガー語 | 「競合調査して」 |
| 調査対象 | 必須 | 商材・市場 | 「クラウド会計SaaS／中小企業向け」 |
| 調査目的 | 必須 | 何のための調査か | 「新規参入判断」「LPリニューアル」 |
| 既知の競合 | 任意 | 既に把握している競合社名 | 「freee, MF, 弥生」 |
| 予算/期間 | 任意 | 有料調査可否 | 「無料のみ／月10万まで／3週間」 |

## 実行ステップ

### ステップ1：調査計画の立案

- 入力：調査対象、調査目的、予算/期間
- 処理：
  1. 「市場調査」「競合調査」のどちらを優先するか判断する
     - 新規参入判断 → 市場調査優先
     - LP/差別化 → 競合調査優先
     - 中長期戦略 → 両方並行
  2. 使用する分析手法を選定する
     - 市場全体の理解 → `pest-analysis`
     - 業界競争 → `five-forces-analysis`
     - 自社×競合×顧客 → `3c-analysis`
     - 自社の立ち位置 → `stp-analysis` `swot-analysis`
  3. 調査範囲（地理／顧客層／時間軸）を確定する
- 出力：調査計画書（目的／優先軸／使用手法／範囲）

#### 2-B. 市場調査（定量／市場規模推定）

| 手法 | 内容 |
|---|---|
| 公的機関・統計データ活用 | 総務省／経産省／民間統計から市場規模把握 |
| デスクトップリサーチ | 各種資料・レポートから市場規模・成長性推定 |
| 民間調査会社レポート | 矢野経済研究所／富士経済／日経BP／Statista等を購入 |
| POSデータ／購買ログ | 小売・ECの販売データから市場シェア・消費動向 |
| モニター調査（インテージSCI等） | 消費者パネルでリピート率・トレンド把握 |
| 自社データ分析（A/Bテスト等） | トライアル提供の反応率から需要規模を逆算 |
| ファネル分析による市場推定 | 「認知→興味→比較→購買」段階の離脱率で全体推定 |

**市場規模推定の3アプローチ**：

1. **トップダウンアプローチ**（公的統計・調査データ活用）：
   - ① 総人口／世帯数／消費支出データ取得
   - ② 対象商品の利用者層を想定
   - ③ 想定利用率を掛け合わせる
   - ④ 想定利用額を掛け合わせる
   - 例：日本成人人口1億 × 健康食品購入率30% × 年間平均1.5万円 = **約4,500億円市場**

2. **ボトムアップアプローチ**（販売点・ユーザー数から積み上げ）：
   - ① 業界内の販売チャネルを洗い出す（例：ドラッグストア／EC／直販）
   - ② チャネルごとの店舗数・EC流通量を推定
   - ③ 1店舗あたり平均販売額 × 店舗数で積み上げ
   - 例：ドラッグストア20,000店舗 × 月平均10万円 + EC月販1億円 = **月30億円 → 年360億円**

3. **関連キーワード検索 × CV率推定**（デジタルマーケ視点）：
   - ① Googleキーワードプランナーで対象商材の月間検索数取得
   - ② ページ流入率・CVR・平均単価で試算
   - 例：月間検索数10万件 × クリック率10% × CVR3% × 単価6,000円 = **月販1,800万円 → 年2.1億円**（広告経由の一部）

### ステップ4：戦略策定

- 入力：ステップ2〜3の全データ
- 処理：
  1. **TAM→SAM→SOM** で市場規模を3階層に分解する：
     - **TAM**（Total Addressable Market）：実現可能な総市場
     - **SAM**（Serviceable Available Market）：自社が対応可能な市場
     - **SOM**（Serviceable Obtainable Market）：実際に獲れる市場
     - 関連スキル: 市場規模予測スキル
  2. ホワイトスペース（競合がカバーしていない領域）を特定する → `3c-analysis` のクロス3C
  3. 競合の強み・弱みを一覧化し、自社の勝てるポイント（ポジショニング）を導出 → `stp-analysis`
  4. 競合がすぐ動く市場では**定期モニタリング**の仕組み化（四半期が目安）
- 出力：実行可能な戦略仮説3案（自社の勝てる領域・差別化軸・市場規模根拠付き）

### ステップ5：競合不在の楽観バイアス排除

調査結果で「競合がいない／少ない」が出た場合、必ず `competitive-absence-audit` スキルを発動する（CLAUDE.md「Competitive Absence Audit Rule」遵守）。

### ステップ6：完了報告

ステップ1〜5の全成果物を以下フォーマットで出力する。

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 予算が無料のみ | デスクトップリサーチ＋競合サイト閲覧＋SNS分析＋口コミ分析を中心 | 有料レポート不要 |
| 月予算10万円以上 | SimilarWeb／矢野経済研究所等の有料活用 | 精度向上 |
| 既知の競合が3社未満 | SEO検索順位上位10位を競合と仮定して網羅 | 把握漏れ防止 |
| 「競合がいない／ブルーオーシャン」が出た | `competitive-absence-audit` 強制起動 | 楽観バイアス排除 |
| 市場規模が一意に決まらない | トップダウン＋ボトムアップ＋KW法の3アプローチで併用検証 | 単一手法は誤差大 |
| 1回調査して終わりにする | 四半期ごとの定期モニタリング推奨 | 競合は動き続ける |
| BtoB領域 | 展示会・商談会・IR情報を必ず含める | BtoCより一次情報の重み大 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 既知の競合のみ調査して網羅した気になる | SEO検索順位上位10位を必ずカバー | 知らない競合が一番危険 |
| 市場規模を単一手法で確定する | 3アプローチ併用で検証 | 単一は誤差大 |
| 「競合がいない」を結論にする | `competitive-absence-audit` 必須 | 楽観バイアス |
| 1回作って終わり | 四半期ごとの定期モニタリング | 競合は動き続ける |
| 5C観点（特にCustomer / Context）を省略する | 必ず5観点全て埋める | 戦略仮説の精度低下 |
| サイト分析で title/meta を見ない | ソースコードまで確認する | SEO観点欠落 |
| TAM だけ言って SAM/SOM を出さない | 必ず3階層で出す | 「実際に獲れる規模」が判断軸 |

## 連携スキル

- **前段／併用**：`pest-analysis` `swot-analysis` `five-forces-analysis` `3c-analysis` `stp-analysis` `vrio-analysis` `value-proposition`
- **強制起動**：`competitive-absence-audit`（競合不在主張が出た時）
- **後段**：`medium-term-business-plan`（戦略仮説の中長期計画化）、`pmf-journey`（PMF判定）、`innovator-theory`（フェーズ判定）
