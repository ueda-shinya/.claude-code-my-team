# sales-psychology-techniques

5つの営業心理テクニックを、商談の具体シーンに「いつ・どこで・何と言うか」まで落とし込む。

**商談フェーズ位置**：全フェーズ横断（冒頭〜クロージング直前まで配置設計）

## 適用条件

- 使う場面：商談設計段階、既存トークのブラッシュアップ、新人トレーニング台本作成
- 使わない場面：テクニックの座学説明（agent-studyを使う）、単発クロージングのみの設計

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 商談フェーズ | 必須 | 冒頭 / ヒアリング / 提案 / クロージング | 提案 |
| 商材 | 必須 | 対象商品・サービス | リフォーム |
| 現行トークの課題 | 任意 | 弱い部分 | 押しが強く断られる |
| 相手の利き手情報 | 任意 | 不明なら右利き前提 | 不明 |

## 実行ステップ

### ステップ1：ハロー効果の投入箇所を指定する
- 入力：商談フェーズ
- 処理：
  - 冒頭フェーズに限定配置する
  - チェック項目を出力：身だしなみ（ネクタイ・靴・爪）／笑顔／姿勢／明瞭な声
  - 「第一印象は6秒で決まる」を台本注記として入れる
  - 具体的な冒頭スクリプト生成は `first-impression-30sec` に委譲する
- 出力：ハロー効果チェックブロック

### ステップ2：聞き手の法則で立ち位置を指定する
- 入力：相手の利き手情報
- 処理：
  - 右利き前提 → 相手から見て左側（相手の利き手側）に立つ
  - パーソナルスペース120cmを確保する
  - 資料を広げる向きを指定する（相手側に正対）
- 出力：立ち位置・距離指示ブロック

### ステップ3：バックトラッキングのセリフパターンを生成する
- 入力：商談フェーズ
- 処理：
  - ヒアリングフェーズに配置する
  - 型A「〇〇されたんですね！」
  - 型B「〇〇が大変だったんですね」
  - 相手の発言のキーワードをそのまま反復する指示を明記する
  - 「全部繰り返さない。キーワードのみ」を注意書きとして追加
- 出力：バックトラッキング台本

### ステップ4：フット・イン・ザ・ドア（FITD）の2段階要求を設計する
- 入力：商材
- 処理：
  - 小要求：資料送付・デモ視聴・3分アンケート等、相手がYesと言いやすいものを1つ
  - 大要求：商談設定・見積・契約等の本命
  - 小→大の時間間隔（即日・翌日・1週間後）を指定する
- 出力：FITD 2段階セリフ

### ステップ5：ドア・イン・ザ・フェイス（DITF）の落とし込みを設計する
- 入力：商材
- 処理：
  - 最大要求：プレミアムプラン・フルセット等、断られる前提の上位提案
  - 本命要求：スタンダードプラン等、実際に売りたいもの
  - 2つの価格差を明示する（2〜5倍が目安）
- 出力：DITF 対比提案セリフ

### ステップ6：全テクを商談フェーズ別に配置する
- 入力：ステップ1〜5
- 処理：以下の配置表で出力する
  - 冒頭：ハロー効果＋聞き手の法則
  - ヒアリング：バックトラッキング
  - 提案：FITDまたはDITF（相手の反応で使い分け）
  - クロージング：別スキル（sokketsu-closing）に接続
- 出力：フェーズ別配置表

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 相手が慎重派 | FITDを優先 | 小さなYesから積み上げる方が安全 |
| 相手が即断派 | DITFを優先 | 対比で即決しやすい |
| 短時間商談（15分以内） | ハロー＋バックトラッキングのみ | FITD/DITFは時間切れになる |
| 電話営業 | 聞き手の法則は省略 | 立ち位置の概念がない |

## 出力フォーマット

```
【冒頭フェーズ】
- ハロー効果：〇〇
- 聞き手の法則：〇〇

【ヒアリングフェーズ】
- バックトラッキング：〇〇

【提案フェーズ】
- FITD：小要求〇〇 → 大要求〇〇
- DITF：最大要求〇〇 → 本命要求〇〇

【クロージング】
→ sokketsu-closing スキルへ接続
```

### 記入例

```
【冒頭フェーズ】
- ハロー効果：ネクタイ整え／靴磨き／90度お辞儀3秒／明るい声
- 聞き手の法則：相手から見て左側に立つ／120cm確保

【ヒアリングフェーズ】
- バックトラッキング：
  お客様「昨日家族で遊園地に行ったんですよ」
  営業「遊園地に行かれたんですね！」

【提案フェーズ】
- FITD：
  小「まず資料だけでもご覧いただけますか？」
  大（翌週）「来週お伺いして詳しくご説明させていただけますか？」
- DITF：
  最大「フルセットのプレミアムプラン月額50万円はいかがですか？」
  本命「では、スタンダードプラン月額10万円はいかがですか？」

【クロージング】
→ sokketsu-closing スキルへ
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| バックトラッキングで全文復唱 | キーワードだけ反復 | オウム返しが不自然になる |
| FITDとDITFを同時投入 | 相手タイプで使い分け | 矛盾メッセージで信頼喪失 |
| 聞き手の法則で真正面に立つ | 斜め横から話す | 圧迫感を与える |
| ハロー効果を中盤でチェック | 冒頭で固定 | 第一印象は6秒で決まる |
