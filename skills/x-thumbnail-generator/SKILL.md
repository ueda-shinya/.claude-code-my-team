# X Thumbnail Generator

Xサムネイル（2500×1000）向けの画像生成プロンプトを、用途に応じてテンプレートから選定・カスタマイズして提案するスキル。

**依拠リソース**: `~/.claude/knowledge/image-prompt-engineering/x-thumbnail-templates.md`（ほしの氏プロンプト100選）

## 適用条件

- 使う場面: Xの記事投稿サムネイル制作、X広告用ビジュアル、SNS用ヒーロー画像（2500×1000比率）
- 使わない場面: スワイプLP / LPヒーロー / Instagram投稿等、別比率のビジュアル（→ ルナ直接依頼へ）、手作りのゼロからプロンプト設計（→ ルナ直接依頼へ）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 記事タイトル | 必須 | 画像に入れる文言（文字入り系を使う場合） | 「AI時代の生存戦略」 |
| タイプ | 必須 | 文字入り / 文字なし（ビジュアルのみ） | 文字入り |
| 用途・テーマ | 必須 | 記事の内容・ジャンル | ビジネス / ニュース速報 / 分析 / 体験談 等 |
| トーン | 任意 | 高級感 / 緊急 / 知性 / 温かみ 等 | 高級感 |
| 文字色 | 任意 | 文字入り時。白 / チャコール / ゴールド 等 | ゴールド |

- 出力: テンプレート一覧の理解

### ステップ2: タイプを判定する

- 処理:

  - **文字なし**（ビジュアルのみで勝負）→ ステップ4へ

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| タイプ（文字入り/文字なし）が不明確 | ユーザーに確認 | 選定ロジックが根本的に違うため |
| 用途・テーマが曖昧 | 51番（深いネイビー×白）を第1候補に提示 | 「迷ったらこれ」の汎用デフォルト |
| トーンが複数の用途にまたがる | 第1・第2候補の2つを提示 | ユーザーが比較して選べるように |
| 記事タイトルが長すぎる（20文字超） | ユーザーに「サムネ用の短縮版」を提案 | Xサムネで長文は読まれない |
| カスタマイズ要望（色変更等）あり | プロンプト該当箇所を変更 | 例: "white" → "gold" |
| 100案のどれも合わない場合 | ルナに直接依頼（本スキルは使わない） | テンプレ外はスキル対象外 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| テンプレート1つだけ提示 | 第1候補＋第2候補を提示 | ユーザーが選べる・比較できる |
| `【タイトル】` を置き換え忘れ | 必ず実タイトルに置換 | そのままでは使えない |
| ジャンル無視で汎用51番ばかり推薦 | ジャンルに合わせて具体的に選ぶ | 51番は「迷ったら」用 |
| 100案にないデザイン要望を無理やり当てはめる | テンプレ対象外と判断し、ルナ直接依頼に切り替え | 本スキルはテンプレ選定に特化 |
| タイトル長すぎても指摘しない | 20字超は短縮提案する | サムネは短文前提 |

## 連携

- **画像生成実行**: 本スキルはプロンプト提案のみ。実際の生成は Gemini アプリ（ユーザー側）または `gemini-3.1-flash-image-preview` API（アスカ経由）
- **ルナとの使い分け**:
  - x-thumbnail-generator: テンプレベースでスピード重視のXサムネ生成
  - ルナ（nano-banana）: ゼロから設計、5コンポーネント式、複数画像の一貫性等
