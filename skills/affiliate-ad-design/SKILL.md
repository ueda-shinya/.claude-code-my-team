# affiliate-ad-design

成果報酬型のアフィリエイト広告を、3タイプから選定し、成果地点・条件・単価を設計するスキル。

## 適用条件

- このスキルを使う場面：
  - バックエンド商材（美容医療・転職・健康食品サブスク・保険・通信等）でアフィリエイトを始めたい
  - ASP（A8.net／バリューコマース／afb／TCS等）への登録と出稿設計
  - 既存アフィリエイトの成果地点や単価の見直し
  - インフルエンサーアフィリエイトの導入検討
- このスキルを使わない場面：
  - パブリッシャー（サイト運営者）側でアフィリエイト収益化したい → 別スキル（記事制作中心）
  - リスティング広告／ディスプレイ広告 → 個別スキル
  - 商品自体の魅力訴求 → `value-proposition`／`ad-copy-7-angles`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 商品/サービス名 | 必須 | 出稿対象 | 「セルライト特化エステ」 |
| 業種 | 必須 | カテゴリ | 「美容医療／転職／健康食品／保険等」 |
| 商品単価/LTV | 必須 | 1件あたり収益 | 「初回1,000円→継続平均LTV 5万円」 |
| 月間獲得目標 | 必須 | CV件数 | 「月100件」 |
| 月間予算 | 必須 | 想定アフィリエイト報酬上限 | 「月200万円」 |
| 既存LP/予約フロー | 任意 | 現状の遷移 | 「LP→電話/Webで予約→来店」 |

## 実行ステップ

### ステップ1：アフィリエイト適性を判定する

- 入力：商品単価、業種、商品特性
- 処理：以下の適性条件を満たすか確認
  - 1件あたりLTVが5,000円以上（成果報酬を払える経済性）
  - 競合性のある商材（比較対象がある／カテゴリ確立済み）
  - 業種がアフィリエイトと相性が良いか：
    - 高相性：美容医療／脱毛／転職／結婚相談所／保険／Wi-Fi／スマホキャリア／健康食品サブスク／発毛／不用品回収
    - 中相性：EC物販／教育／投資／フィットネス
    - 低相性：BtoB高単価／極端なニッチ／オーダーメイド
- 不適合の場合は `ad-mix-design` で他媒体を提案
- 出力：アフィリエイト適性判定（適合/中庸/不適合）

### ステップ2：3タイプから出稿手段を選定する

- 入力：商品特性、ターゲット行動、予算
- 処理：以下3タイプから選ぶ
  | タイプ | 媒体 | 適する商材 | 特徴 |
  |---|---|---|---|
  | アドアフィリエイト | SNS広告等のウェブ広告 | 美容医療／脱毛／健康食品／転職 | 代理店アフィリエイターが運用、広告費をアフィリエイター側が負担、成果時のみ広告主は報酬支払い |
  | SEOアフィリエイト | 「〇〇おすすめ30選」等の比較記事 | 保険／Wi-Fi／スマホキャリア／求人／結婚相談所／発毛／不用品回収 | じっくり比較したい商材向き、CV単価高い、リード長い |
  | インフルエンサーアフィリエイト | Instagram／X／TikTok等のインフルエンサー投稿 | 美容／ファッション／健康食品／教育 | 共感ベースの訴求、若年層リーチ、案件選定が肝 |
- 複数タイプの併用ルール：
  - 美容医療系 → アド＋インフル（認知＋獲得両軸）
  - 比較検討系（保険/転職）→ SEO中心＋アド補強
- 出力：選定タイプと併用方針

### ステップ3：成果発生地点を設計する（最重要）

- 入力：商品の購買フロー、CV閾値
- 処理：CVに至る階段を設計し、どの段階を「成果発生」と定義するか決める
  - **エステサロンの例（深い→浅い）**：
    1. 施術申込/入金（最深・最高難易度）
    2. 当日来店
    3. 電話で体験予約確認
    4. 体験予約フォーム送信（最浅・最低難易度）
  - **転職エージェントの例**：
    1. 内定→入社（最深）
    2. 面談実施
    3. 会員登録＋面談予約
    4. 会員登録のみ（最浅）
  - **保険の例**：
    1. 契約成立（最深）
    2. 資料請求＋電話通話
    3. 資料請求のみ（最浅）
- 設計ルール：
  - 浅い地点 → 件数取りやすい・単価低い・歩留まり管理が必須（無効CV多発リスク）
  - 深い地点 → 件数取りにくい・単価高い・アフィリエイターが集まりにくい
  - **推奨スタートライン**：中間地点（資料請求＋電話確認、体験予約＋電話確認等）
- 出力：成果発生地点の確定（1〜複数）

### ステップ4：成果発生条件を決める（無効CV防止）

- 入力：成果発生地点
- 処理：成果と認定する条件を明文化する
  - 浅い地点（フォーム送信等）の場合の必須条件例：
    - 電話番号・メールアドレスが実在する（ダミー番号NG）
    - 同一IPからの連続申込を除外
    - 重複申込（同一人物の複数申込）を除外
    - 自社/関係者からの申込を除外
  - 中間地点（来店・面談等）の場合：
    - 実際に来店/面談が完了
    - キャンセル・No Showは除外
  - 深い地点（契約・入金等）の場合：
    - 入金完了
    - クーリングオフ期間経過後
- 認定タイミング：申込発生から認定までの期間（一般的に30-90日）
- 出力：成果発生条件と認定タイミング

### ステップ5：成果単価を決める（LTVベース）

- 入力：商品LTV、目標CPA、成果発生地点
- 処理：以下の計算で単価を決定
  - **基本式**：成果単価 = 想定LTV × 利益率 × 配分率（30-50%）
    - 例）LTV 5万円・利益率40%・配分率40% → 単価 = 50,000 × 0.4 × 0.4 = 8,000円
  - **業界相場ベンチマーク**：
    | 業種 | 成果地点 | 単価相場 |
    |---|---|---|
    | 美容医療（脱毛/痩身）| 体験予約＋来店 | 3,000-15,000円 |
    | 転職エージェント | 会員登録＋面談 | 5,000-30,000円 |
    | 保険 | 資料請求＋電話 | 3,000-10,000円 |
    | 健康食品サブスク | 初回購入 | 2,000-8,000円 |
    | Wi-Fi/スマホ契約 | 契約成立 | 5,000-30,000円 |
    | 結婚相談所 | 入会面談 | 10,000-30,000円 |
  - 単価が低すぎ → アフィリエイターが集まらない
  - 単価が高すぎ → 採算割れ／不正申込誘発
- アフィリエイターとの交渉：成果地点と単価のバランス（深く・高単価 vs 浅く・低単価）
- 出力：確定成果単価（タイプ別・地点別）

### ステップ6：ASPと契約・出稿準備する

- 入力：選定タイプ、成果地点・条件・単価
- 処理：
  - 主要ASP選択：
    - A8.net（最大手・幅広い案件）
    - バリューコマース（EC強い）
    - afb（美容/健康食品強い）
    - TCS（インハウス向き）
    - レントラックス（クローズドASP・成果報酬高い）
  - 申込フォーム作成（パラメーター付きURL／成果計測タグ設置）
  - クリエイティブ素材準備（バナー・テキスト・LP）
  - アフィリエイター向けガイドライン作成（薬機法・景表法対応の訴求NGワード集）
- 出力：ASP契約計画＋必要素材一覧

### ステップ7：広告主側の2注意点でリスク点検する

- 入力：出稿計画
- 処理：以下2チェックを実施
  - **ブランドイメージ管理**：
    - アフィリエイターの不適切宣伝チェック（誇大表現／薬機法違反／景表法違反）
    - 宣伝ガイドラインを必ずASP登録時に提供
    - 月次でアフィリエイトサイトを巡回モニタリング
    - 違反発見時の即時通知・取り下げ運用
  - **報酬コスト管理**：
    - 月間予算上限を設定
    - キャンペーン期間を区切る（無期限はNG）
    - 不正CV検知（重複・ダミー・自演）の運用
    - ROAS（広告費用対売上）の月次レビュー
- 出力：リスク点検結果＋対策運用計画

### ステップ8：アフィリエイト広告計画書としてまとめる

- 入力：ステップ1-7
- 処理：出力フォーマットに沿って整形
- 出力：アフィリエイト広告計画書

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| LTV 5,000円未満 | アフィリエイト不向き → リスティング/SNS広告 | 成果報酬の経済性が成立しない |
| BtoB高単価商材 | アフィリエイト不適合 → リード獲得広告／コンテンツマーケ | アフィリエイトは消費者向けが主流 |
| 美容医療/脱毛 | アド＋インフルの併用 | 認知＋即決の両軸が必要 |
| 保険/比較検討系 | SEOアフィリエイト中心 | 比較記事経由の流入が定番 |
| 浅い成果地点（フォーム送信のみ） | 必ず認定条件を厳格化 | 不正申込多発リスク |
| 単価が業界相場の80%未満 | 引き上げ検討 | アフィリエイターが集まらない |
| 単価が業界相場の150%超 | 採算性見直し | 利益率を圧迫 |
| ガイドライン未整備で出稿 | 必ずガイドライン作成してから出稿 | ブランド毀損・法令違反リスク |
| 成果地点が1つだけ | 浅い地点＋深い地点の2段階単価を検討 | アフィリエイターの参加ハードルを下げる |
| クローズドASP（レントラックス等）と契約済み | アド＋インフル中心の戦略可能 | 大手アフィリエイターへ直アプローチ |
| 月間予算上限なしで出稿 | 必ず上限設定 | 想定外コスト発生 |

### 記入例

```markdown
# アフィリエイト広告計画書 — 痩身エステ「ラ・パルレ」

## 1. 商品情報
- 商品/サービス：セルライト特化型痩身エステ
- 業種：美容医療
- LTV：50,000円（初回1,000円→継続コース平均5万円）
- 利益率：40%

## 2. 出稿タイプ
- 主軸：アドアフィリエイト（TikTok中心）
- 補助：インフルエンサーアフィリエイト
- 配分：主軸70% / 補助30%

## 3. 成果地点設計
| 成果地点 | 認定条件 | 想定単価 | 想定月CV |
|---|---|---|---|
| 体験予約フォーム送信 | 電話番号・メアド実在 | 3,000円 | 200件 |
| 当日来店 | 受付確認 | 8,000円 | 80件 |

## 4. ASP/契約
- 利用ASP：afb（美容系強い）
- 契約案件数：1案件（メイン）
- 月間予算上限：100万円

## 5. 必要素材
- LP：example.com/lp-cellulite
- バナー：300x250 / 728x90 / 320x50（各3パターン）
- ガイドライン：薬機法対応NGワード集（「痩せる」「効く」「治る」NG／「サポート」「整える」OK）

## 6. リスク点検
- [x] ブランドイメージ管理：ガイドラインafb経由配布／週次でTikTok・Instagram巡回
- [x] 報酬コスト管理：月100万円上限／キャンペーン期間3ヶ月／重複IP除外設定

## 7. KPI
- 月間CV（来店）：80件
- ROAS：300%以上（売上240万円/予算80万円）
- 不正CV率：5%以下
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 浅い成果地点だけで運用（フォーム送信のみ）| 浅＋深の2段階単価設計 | 不正CV多発／アフィリエイターのモラル低下 |
| アフィリエイター向けガイドライン未整備 | 必ずガイドライン提供 | 薬機法/景表法違反でブランド毀損 |
| 単価を業界相場無視で安く設定 | 業界ベンチマーク＋10-20%で出す | アフィリエイターが集まらない |
| 月間予算上限なし | 必ず上限設定 | 想定外コスト爆発 |
| 不正CV検知の運用なし | IP重複・電話番号実在・自演除外を仕組み化 | 採算性を破壊 |
| 出稿後のサイト巡回モニタリングなし | 月次で巡回 | ブランド毀損案件の発見が遅れる |
| アフィリエイター紹介の記事を読まずに任せきり | 主要記事は事前確認 | 誇大表現・誤情報リスク |
| LTV低い商材（5,000円未満）で無理に出す | 別媒体を選定 | 成果報酬の経済性なし |
| BtoB商材でアフィリエイト | リスティング+コンテンツマーケ | アフィリエイト市場はBtoC中心 |
| ASP登録だけで放置 | 定期的に案件露出をASPにプッシュ | アフィリエイターに見つけてもらえない |
