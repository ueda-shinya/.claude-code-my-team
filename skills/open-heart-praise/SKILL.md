# open-heart-praise

初対面の相手に使う「オーラ褒め→二度褒め」の3ステップ褒めトークを生成する。

**商談フェーズ位置**：アイスブレイク（冒頭5分以内）

## 適用条件

- 使う場面：初回商談の冒頭5分、初対面アイスブレイク、緊張した場の解きほぐし
- 使わない場面：2回目以降の商談（既に使用済みのため）、クレーム対応、高圧的な相手

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 相手の属性 | 必須 | 性別・推定年代・役職 | 50代男性 社長 |
| 第一印象キーワード | 必須 | 会った瞬間に感じた雰囲気 | 誠実そう / 知的 |
| 見える具体物 | 任意 | ネクタイ・時計・オフィスの花等 | ネクタイ |
| 営業担当者の年齢 | 任意 | 二度褒め（自分との比較）用 | 38歳 |

## 実行ステップ

### ステップ1：羞恥心を外すマインドセット注記を固定出力する
- 入力：なし（固定テンプレート）
- 処理：以下を台本冒頭に必ず配置する
  - 「恥ずかしさを捨てて堂々と言い切る」
  - 「目線は外さない」
  - 「小声・語尾濁し・目そらしはNG」
- 出力：固定マインドセット注記ブロック

### ステップ2：オーラを褒める一文目を生成する
- 入力：第一印象キーワード
- 処理：
  - 型「初めましてお会いした瞬間に感じたことなんですけど、〇〇な方ですよね。オーラめっちゃ出てますね」
  - キーワードは以下から優先選択：誠実そう／知的／優しそう／仕事ができそう／信頼できそう
  - オーラ褒めステップでは容姿直接表現（「かわいい」「かっこいい」等）は避ける（具体物褒めステップ=ステップ5では許容）
- 出力：一度目の褒め文

### ステップ3：二度褒めのパターンを選ぶ
- 入力：営業担当者の年齢、相手の属性
- 処理：以下の2パターンから1つ選択
  - (a) 自分との比較：「私〇〇年生きてますけど、〇〇とか1回も言われたことないんですよ」
  - (b) 今までとの比較：「今までお会いした方と比べても、〇〇さんは別格で〇〇ですね」
- 出力：二度褒めパターン指定

### ステップ4：二度褒め文を生成する
- 入力：二度褒めパターン、第一印象キーワード
- 処理：
  - 「これほんまの話でね」等の前置きを1つ入れる
  - 1文目の褒めワードを繰り返して強化する
  - 最後に「羨ましいです」「尊敬します」で締める
- 出力：二度目の褒め文

### ステップ5：具体物を褒める追加フレーズを生成する
- 入力：見える具体物
- 処理：
  - 型「〇〇おしゃれですね。めっちゃかっこいいじゃないですか」
  - 具体物が無ければステップ5は省略可
- 出力：具体物褒めフレーズ

### ステップ6：3ステップを台本として結合する
- 入力：ステップ1〜5の全ブロック
- 処理：マインドセット指示→一度目→（相手の反応待ち）→二度目→（任意）具体物褒めの順に並べる
- 出力：完成台本

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 相手が謙遜で返してきた | そこで二度褒めを即投入 | 返しのタイミングが最も刺さる |
| 相手が無反応 | 二度褒めは省略し具体物褒めへ | しつこいと逆効果 |
| BtoB硬い業界 | キーワードを「知的」「信頼できそう」に限定 | 軽薄感を避ける |
| 相手が年下 | 「尊敬します」は避け「羨ましい」に寄せる | 年上から年下への褒めは温度調整 |

## 出力フォーマット

```
【マインドセット】
- 〇〇

【一度目：オーラ褒め】
〇〇

（ここで相手の反応を待つ）

【二度目：比較で強化】
〇〇

【具体物褒め（任意）】
〇〇
```

### 記入例

```
【マインドセット】
- 恥ずかしさを捨てて堂々と言い切る
- 目線は外さない

【一度目：オーラ褒め】
「初めましてお会いした瞬間に感じたことなんですけど、めちゃくちゃ誠実そうな方ですよね。オーラ、めっちゃ出てますね。」

（ここで相手の反応を待つ）

【二度目：比較で強化】
「いやこれほんまの話でね、私38年生きてますけど、知的とか誠実そうとか1回も言われたことないんですよ。ですから羨ましいです。」

【具体物褒め】
「あ、そのネクタイおしゃれですね。めっちゃかっこいいじゃないですか。」
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 容姿を直接褒める | オーラ・雰囲気を褒める | 下心に聞こえる |
| 一度褒めて終わる | 必ず二度褒める | 快楽成分は2回目で開く |
| 小声・目をそらす | 堂々と目を見て言う | 嘘っぽく聞こえる |
| お世辞っぽい万能褒め | 第一印象で感じた具体キーワード | 刺さらない |
