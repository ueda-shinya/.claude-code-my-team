# first-impression-30sec

飛び込み・テレアポ・訪問営業の冒頭30秒を、即決営業メソッドに沿った4ステップアプローチに構造化する。

**商談フェーズ位置**：冒頭（最初の30秒）

## 適用条件

- 使う場面：新規飛び込み、テレアポ、初回訪問、初対面の商談冒頭
- 使わない場面：既存顧客との2回目以降の商談、紹介経由で信頼がある面談

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| チャネル | 必須 | 対面 / 電話 / 飛び込み | 飛び込み |
| 自社情報 | 必須 | 会社名・担当者名・所在地（ビル名・階） | 株式会社〇〇 田中 / 大阪駅第一ビル8階 |
| ターゲット | 必須 | 相手の業種・役職 | 製造業 社長 |
| 商材の切り口 | 任意 | クエスチョンマークを作るための話題タネ | 助成金 / 保険会社数 等 |

## 実行ステップ

### ステップ1：圧倒的な第一印象の型を選ぶ
- 入力：チャネル
- 処理：
  - 対面/飛び込み → 「90度お辞儀・3秒キープ・機敏に入る・ゆっくり顔を上げて笑顔で目線」を指示書化する
  - 電話 → 「明瞭・元気・トーン高め・1文目で笑顔の声」を指示書化する
  - NG行動（浅いお辞儀・ペコペコ・お辞儀中に話す）を禁止事項として明記する
- 出力：所作指示ブロック

### ステップ2：気遣い＋感謝のクッション文を生成する
- 入力：チャネル
- 処理：以下から相手時間への配慮を1〜2文選び、冒頭に配置する
  - 「お忙しいところすいません」
  - 「お手数をおかけして申し訳ございません」
  - 「ありがとうございます」
- 出力：クッション文ブロック

### ステップ3：特別感を演出する一言を挿入する
- 入力：クッション文
- 処理：「お世話になっております」を初対面であっても挿入し、"特別に選ばれた感"を演出する
- 出力：特別感ブロック

### ステップ4：親近感のある自己紹介文を生成する
- 入力：自社情報
- 処理：
  - NG型「株式会社〇〇の田中です」は使わない
  - OK型「（地名＋建物名＋階）で営業を行っております、株式会社〇〇の〇〇と申します」に変換する
  - 地名・建物名が不明な場合は相手に一番近いランドマークを指定する
- 出力：自己紹介ブロック

### ステップ5：興味喚起の「?」を投げかける
- 入力：商材の切り口
- 処理：商材の特徴説明ではなく、相手の頭に疑問符を作る質問を1つ作る
  - 型A「突然ですが、これ受け取られましたか？」
  - 型B「〇〇はおいくらもらわれましたか？」
  - 型C「〇〇業界に△△は何社あると思いますか？」
- 出力：クエスチョン文

### ステップ6：堂々と要求する導線をつける
- 入力：クッション〜クエスチョンまで
- 処理：
  - テレアポ → 「恐れ入ります。すぐに担当者様に代わっていただけますか？」
  - 飛び込み → 「突然失礼します。大切なお話がございますので、玄関先まで失礼させていただけますか？」
  - 「お時間よろしいですか？」型の弱腰表現は禁止
- 出力：要求文

### ステップ7：全ブロックを結合し30秒スクリプトとして出力する
- 入力：ステップ1〜6の全ブロック
- 処理：所作指示→クッション→特別感→自己紹介→クエスチョン→要求の順に結合し、読み上げ時間が25〜35秒に収まるか文字数で確認する（日本語300〜400字目安）
- 出力：完成スクリプト＋所作メモ

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 電話チャネル | お辞儀ステップを声トーン指示に差し替える | 所作は見えないため |
| 担当者不在 | ステップ6を「担当者様への取次依頼」バージョンに切替 | 要求先が変わる |
| 商材が複雑 | ステップ5は「助成金/保険会社数」型の一般話題に置換 | 冒頭で商品説明しないため |
| 相手が明らかに多忙 | クッション文を2文にし、要求文は1文で短縮 | 時間圧に配慮 |

## 出力フォーマット

```
【所作指示】
- 〇〇
- 〇〇

【30秒スクリプト】
（クッション）〇〇
（特別感）〇〇
（自己紹介）〇〇
（クエスチョン）〇〇
（要求）〇〇

【NG行動リマインド】
- 〇〇
```

### 記入例

```
【所作指示】
- 90度のお辞儀を3秒キープ
- ゆっくり顔を上げ、笑顔で目線を合わせる

【30秒スクリプト】
（クッション）お忙しいところすいません、お手数をおかけして申し訳ございません。
（特別感）お世話になっております。
（自己紹介）大阪駅第一ビルの8階で営業を行っております、株式会社〇〇の田中と申します。
（クエスチョン）突然ですが、製造業向けの助成金、今年度いくらもらわれましたか？
（要求）大切なお話がございますので、玄関先まで失礼させていただけますか？

【NG行動リマインド】
- 「お時間よろしいですか？」と弱腰に聞かない
- 45度以下の浅いお辞儀をしない
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 「株式会社〇〇の田中です」 | 「大阪駅第一ビル8階で営業を行う、株式会社〇〇の田中です」 | 地名・建物名で親近感が生まれる |
| 「お時間よろしいですか？」 | 「玄関先まで失礼させていただけますか？」 | 弱腰要求は断られやすい |
| 冒頭から商品説明 | 冒頭はクエスチョンで「?」を作る | 聞く姿勢が作れない |
| お辞儀45度・すぐ顔上げ | 90度・3秒キープ | 他営業との差別化が消える |
