# webinar-design

BtoBマーケティングのウェビナー（オンラインセミナー）を、目的×4パターン×KPI逆算で設計するスキル。1回で対面セミナーの5〜10倍の集客力を引き出すために、目的に合ったパターン選択（カンファレンス／共催／事例／自社）から、企画構成・タイトル設計・事前集客・フォローアップまでを一気通貫で計画する。

## 適用条件

- 使う場面：
  - BtoBで新規リード獲得・商談獲得のためのウェビナーを企画
  - 月間リード目標から逆算してウェビナー開催計画を立てる
  - 共催ウェビナーの相手企業選定と運営設計
  - 既存ハウスリストの育成（リードナーチャリング）にウェビナー活用
  - カンファレンス（大規模ウェビナー）の戦略設計
- 使わない場面：
  - リードナーチャリング全般（メール／SNS／コンテンツ等の総合設計） → `lead-nurturing`
  - リード定義・MQL/SQL設計 → `lead-definition-mql-sql`
  - ホワイトペーパー単体の企画 → `whitepaper-content-design`
  - SNS戦略全般 → `sns-strategy-overview`
  - プレゼン能力スキル → `presentation-skill`
  - 営業提案書 → `proposal-builder`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 主目的 | 必須 | ブランド認知／リード獲得／リード育成・商談獲得 | 「新規リード月間1000件獲得」 |
| ターゲット | 必須 | 業種・役職・課題 | 「中堅企業マーケ部長／リード不足」 |
| 月間リード目標 | 推奨 | 数値目標 | 「1000リード／月」 |
| 既存ハウスリスト | 推奨 | 自社保有のメールリスト数 | 「2万件のハウスリスト」 |
| 検討フェーズ | 推奨 | 初回接点／検討中／商談直前 | 「初回接点中心」 |
| 開催リソース | 推奨 | 自社で運営できる頻度・人員 | 「月3-5回開催可」 |

## 実行ステップ

### ステップ1：用語と目的を整理する

- 入力：主目的
- 処理：基本概念を共有
  - **ウェビナー（Webinar）**：「ウェブ（Web）」と「セミナー（Seminar）」の造語。オンライン上で行われるセミナーや講義
  - **リードジェネレーション**：製品・サービスに興味を持つ可能性のある顧客（リード）を見つけ出し、接触のきっかけを作るマーケティング活動
  - **リードナーチャリング**：獲得したリードと継続的に関係を築き、購買意欲を高めて成約へ導くプロセス
- **BtoBマーケでのウェビナー目的（3階層）**：
  - **ブランド認知向上**：自社の専門性や信頼性をアピール
  - **リード育成・案件化**：既存見込み顧客の関心を高め、商談化
  - **リード獲得**：新規見込み顧客の情報収集
- **ウェビナーのメリット**：これまで対面形式で参加が難しかった人にリーチできる。新しいお客様を増やし、対面セミナーより5〜10倍の集客力が期待できることもある
- 出力：目的の優先順位（3階層×自社現状）

### ステップ2：テーマ設定3ポイントで方向性を決める

- 入力：ターゲット、主目的
- 処理：以下3ポイントでテーマを設定
  1. **ターゲットの明確化**：参加者の業種・役職・課題を特定
  2. **価値提供**：参加者が直面する課題解決や最新情報の提供
  3. **独自性**：他社との差別化を図るユニークな視点や内容
- 出力：テーマドラフト

### ステップ3：企画構成6POINTで詳細を設計する

- 入力：ステップ2のテーマ
- 処理：6POINTを順番に詳細化
  - **POINT 01：目的の明確化**：ウェビナーのゴールを設定
  - **POINT 02：ターゲット設定**：参加者のペルソナを具体化
  - **POINT 03：コンテンツ設計**：参加者の興味・関心に合わせた内容作成
  - **POINT 04：配信形式の選択**：ライブ配信／録画配信／ハイブリッドなど
  - **POINT 05：集客計画**：SNSやマーケティングを活用した告知
  - **POINT 06：フォローアップ**：アンケートや録画提供による参加者フォロー
- **ウェビナー形式の選択（3区分）**：自社／共催／外部
  - 目的が「リード獲得」か「リード育成」かでも開催形態を検討
- 出力：6POINT別計画

### ステップ4：タイトル集客効果6観点で命名する

- 入力：ステップ3のコンテンツ設計
- 処理：以下6観点でタイトル候補を磨き込む
  1. **興味を引く力**：直感的に内容を伝え、参加メリットを示す
  2. **緊急性を演出する**：「先着50名」「残り3日！」など限定性・緊急性
  3. **ターゲットの明確化**：「マーケティング初心者向け」「経営者必見」など具体表現で自分事化
  4. **信頼性の構築**：「業界経験20年の専門家が指導する」など講師の肩書きや実績
  5. **参加メリットを訴求**：「受注数を2倍にする方法」「業界トレンドを知る」など成果を示す
  6. **SEO効果**：適切なキーワードを含み、検索結果に表示されやすくする
- 出力：タイトル候補3案＋採用案

### ステップ5：4パターンから開催形式を選定する

- 入力：月間リード目標、ステップ1の目的優先順位
- 処理：以下4パターンから選択（複数並行可）

#### パターン①：カンファレンス（KPI貢献：大量リード獲得）

- **タイプ**：ターゲットが同じ複数社で開催／パネルディスカッションメインで有識者対談／共催ウェビナーで相性が良かった企業を中心に声がけ
- **目的**：ブランディング／新規ハウスリスト獲得／初回接点として幅広いテーマを取り扱う
- **獲得期待**（規模別）：
  - **他社登壇カンファレンス（小〜中規模・複数社で分担）**：1回で 200〜500リード
  - **自社主催カンファレンス（大規模・自社主導）**：1回で 800〜1,000リード以上
- **狙い**：具体的な施策やノウハウをテーマにしたウェビナーには反応しない、潜在層または役職者のリードを獲得し、ターゲットリードの保有数を上げる

#### パターン②：共催ウェビナー（KPI貢献：ターゲットリード獲得）

- **タイプ**：ターゲットが同じ複数社で開催／パネルディスカッションメインで対談実施／1回開催あたり30-50名/社集客／基本的にリードは全社で共有
- **目的**：新規ハウスリスト獲得／Tierリードに対しては商談獲得も目的に／初回タッチポイントとして幅広いテーマ
- **背景**：自社単独開催のみで新規リードを集客しつづけることは難しい。共催にすることでハウスリストを共有でき、リードを増やせる。月間リード目標に合わせて共催先社数・開催回数を決める

#### パターン③：事例ウェビナー（KPI貢献：商談獲得・態度変容）

- **タイプ**：自社の代表的なクライアントに登壇頂く／パネルディスカッションメインで対談実施／1回開催あたり50名集客
- **目的**：ハウスリスト育成と商談獲得／第三者視点で商材価値を伝えてもらうことで態度変容を促す
- **狙い**：「ここでしか聞けない」「自社での活用をイメージできる」内容を伝える
- **運用**：自社単体での集客になるため、インサイドセールスと連携しながらハウスリストの温度感を上げる

#### パターン④：自社ウェビナー（KPI貢献：商談獲得）

- **タイプ**：自社単体で実施／興味を持ってくれている見込み顧客向けに育成コンテンツ／1回開催あたり30名集客
- **目的**：商談獲得／見込み顧客の検討フェーズをあげる／スコープを絞ったテーマで深い話を伝える
- **狙い**：すでにタッチポイントのあるハウスリストに対して商談手前/商談後に参加してもらうことで検討フェーズを上げる
- **運用**：商品紹介から具体的な課題解決のためにスコープを絞ったノウハウ展開。参加企業はSQLリードとして扱い、営業組織が追いかける

- **「リード獲得」と「商談獲得」を明確に区別する**：ニーズ顕在度×リード獲得数のマトリクスでチャネル位置付け
- 出力：選定パターン＋ねらい

### ステップ6：KPIから逆算して施策回数を決める

- 入力：月間リード目標、選定パターン
- 処理：以下表で計画

| パターン | 目的 | 新規リード | ハウスリスト |
|---|---|---|---|
| カンファレンス（自社主催・大規模） | 新規リード獲得・ハウス育成・ブランディング | 800〜1,000+ | 200 |
| カンファレンス（他社登壇・小〜中規模） | 新規リード獲得・ブランディング | 200〜500 | 100〜200 |
| 共催6社 | 新規リード獲得・ハウス育成・ブランディング | 200〜400 | 150 |
| 共催4社 | 新規リード獲得・ハウス育成 | 100〜300 | 100 |
| 共催2社 | 新規リード獲得・ハウス育成・商談獲得 | 30〜100 | 50 |
| 事例ウェビナー | ハウス育成・商談獲得 | 20 | 50 |
| 自社ノウハウ | ハウス育成・商談獲得 | 10 | 50 |
| 自社商材紹介 | 商談獲得 | 5 | 25 |

- **1,000リード獲得の場合の逆算例**：
  - 2社共催 最低でも月10回
  - 4社共催 最低でも月4回
  - 6社共催 最低でも月3回
- 出力：月間開催計画（パターン×回数）

### ステップ7：効果的な事前集客6手法を組み合わせる

- 入力：ステップ6の開催計画
- 処理：以下6手法を組み合わせて事前集客を設計
  1. **魅力的なランディングページ**：タイトル、日付、時間、スピーカー情報、参加メリットを明確に。登録を促すCTAを設置することが必要
  2. **メールマーケティング**：既存顧客リストや過去のウェビナー参加者に定期的に告知メール。参加意欲を高める効果がある
  3. **SNSプロモーション**：SNSで情報を広める。定期的に投稿し、関連グループやフォロワーにシェアを促す。より多くの人にリーチできる
  4. **外部メディアの利用**：PeatixやTechplayなどイベント集客プラットフォームで告知。広範囲なオーディエンスにアプローチ
  5. **参加特典の提供**：資料配布、特別なコンテンツへのアクセスなど。参加の動機付けが強化される
  6. **リマインダーの送信**：開催日が近づいたら参加者にリマインダー送信。開催日の前日や当日が効果的
- 出力：事前集客プラン

### ステップ8：成功の鍵を確認しPDCAを設計する

- 入力：ステップ1〜7
- 処理：以下5要素で実行可否を確認
- ウェビナーは効果的なリード獲得・育成・ブランド認知向上に効果的な手段
- **5要素**：①目的の明確化 ＋ ②ターゲット理解 ＋ ③価値提供 ＋ ④集客 ＋ ⑤継続的な振り返り・分析・改善
- これらを実践することで、ウェビナーを効果的なマーケティング手段として活用できる
- **コンテンツ策定 7ステップワーク**：
  1. 目的を明確にする（ゴール：リード獲得／既存育成／ブランド認知）
  2. ターゲットを設定する（ペルソナ具体化）
  3. テーマを決める（ターゲットの関心テーマ）
  4. 内容を具体的に設計する（導入5分→本編20分→Q&A 10分の構成案）
  5. タイトルを考える（キャッチー＋具体的）
  6. 資料やスライドを作成する（簡潔なテキスト＋視覚的グラフ）
  7. フィードバックを得る（同僚・チームの確認）
- 出力：PDCA運用計画

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 月間1000リード必要 | 共催ウェビナー（2社共催 月10回 or 4社共催 月4回 or 6社共催 月3回） | 自社単独だと到達困難 |
| ブランディング重視・大量リード | パターン①カンファレンス（自社主催） | 自社主催なら800〜1000+リード、他社登壇なら200〜500リードで認知獲得 |
| Tier1企業のターゲットリード強化 | パターン②共催（Tierが共催相手として近い企業） | リード品質を担保 |
| ハウスリスト商談化が課題 | パターン③事例ウェビナーでクライアント登壇 | 第三者視点で態度変容 |
| 商談直前の決め手を打ちたい | パターン④自社ウェビナー（深いノウハウ） | 検討フェーズ最後段の押し出し |
| 既存リードと新規リードの両方が必要 | カンファレンス＋自社ウェビナー併用 | 認知獲得→育成→商談化 |
| 集客が苦戦 | ステップ7の6手法を全部適用 | 多面的な集客が必要 |
| 開催リソース限定 | 共催で工数分散＋月間回数調整 | 1社で抱え込まない |
| 業界KOLを巻き込みたい | カンファレンスで有識者対談 | パネルディスカッションが効果的 |
| 営業との連携が弱い | パターン③④で営業と握る | SQLリードを営業に渡す前提 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 自社ウェビナーだけで月間1000リード狙い | 共催ウェビナーで分担＋カンファレンス併用 | 単独では数が足りない |
| ターゲットを絞らない汎用テーマ | ステップ2-1で業種・役職・課題まで具体化 | 関心薄く参加率低下 |
| タイトルに緊急性なし | 「先着50名」「残り3日」を必ず入れる | 行動喚起が弱い |
| 集客LPだけで6手法のうち1つだけ | ステップ7の6手法全部活用 | 多面的集客が必須 |
| リマインダー送信なし | 前日・当日リマインダー必須 | 申込→参加の歩留まり改善 |
| パネルディスカッションなしで講演形式のみ | ステップ5パターン①②でディスカッション主体 | 参加者の引き込みが弱い |
| 事例ウェビナーで匿名事例 | ステップ5パターン③で実名クライアント登壇 | 第三者性なしでは態度変容しない |
| Q&A時間なし | 構成案で必ずQ&A 10分を確保 | 双方向性で記憶定着 |
| フォローアップなし | アンケート・録画提供・営業連携必須 | 参加者の意欲を商談化に繋げる |
| KPI逆算せず開催回数を決める | ステップ6で月間目標から逆算 | 目標未達の常態化 |
