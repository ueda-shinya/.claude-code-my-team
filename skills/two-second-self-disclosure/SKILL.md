# two-second-self-disclosure

長々とした自己紹介ではなく、会話の流れに各2秒で差し込む3段階の自己開示フレーズを生成する。

**商談フェーズ位置**：アイスブレイク〜ヒアリング（冒頭30秒の後、本題前）

## 適用条件

- 使う場面：初回〜2回目商談のアイスブレイク中、雑談→本題の橋渡し
- 使わない場面：既に親密な関係、クレーム対応、冒頭30秒アプローチ（そこでは使わない）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 営業担当者情報 | 必須 | 家族構成・趣味・強み・カバーしたい弱み | 娘あり / 傾聴が得意 / 料理が苦手 |
| 相手情報 | 任意 | 想定される雑談フック | 休日にゴルフ |
| 業界/商材 | 任意 | 強みの開示を業務文脈に寄せたい場合 | 保険営業 |

## 実行ステップ

### ステップ1：レベル1（趣味・嗜好）フレーズを生成する
- 入力：営業担当者情報（プライベート側）
- 処理：
  - 2秒で言い切れる長さ（30字前後）に圧縮する
  - 相手への質問をセットで付け、会話を返す形にする
  - 型：「私〇〇なんですけど、〇〇さんは〇〇ですか？」
- 出力：レベル1フレーズ

### ステップ2：レベル2（強み・長所）フレーズを生成する
- 入力：営業担当者情報（強み）、業界/商材
- 処理：
  - 自慢にならないよう「心がけている」「意識している」の語尾で柔らかく出す
  - 業務に接続する強みを優先選択する
  - 2秒・30字前後で圧縮する
- 出力：レベル2フレーズ

### ステップ3：レベル3（弱み）フレーズを生成する
- 入力：営業担当者情報（弱み）
- 処理：
  - 業務の致命傷にならない弱みを選ぶ（プライベート・不器用系が安全）
  - 笑える温度感の語尾にする（「〜下手で」「〜できないんですよ」）
  - 2秒・30字前後で圧縮する
- 出力：レベル3フレーズ

### ステップ4：投入タイミングを指定する
- 入力：レベル1〜3フレーズ
- 処理：以下の順序と条件を明記する
  - レベル1 → アイスブレイク冒頭
  - レベル2 → 本題に入る前（信頼の土台作り）
  - レベル3 → 本題の途中、相手が一度笑った後
  - 「一度に全部話さない」「最初から弱みを出さない」を禁止事項として併記
- 出力：投入タイミング表

### ステップ5：3フレーズセットを出力する
- 入力：各フレーズと投入タイミング
- 処理：台本として商談フェーズごとに並べる
- 出力：自己開示3段セット

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 相手が寡黙 | レベル1で質問を強めにする | 会話を返してもらう必要があるため |
| 相手が年上・役職上 | レベル2の強みは「努力中」トーンに下げる | 生意気に聞こえないため |
| BtoB高単価商材 | レベル3の弱みはプライベート領域に限定 | 業務弱みは信頼を損なう |
| 30分未満の短時間商談 | レベル3は省略可 | 全部入れると濃すぎる |

## 出力フォーマット

```
【レベル1：趣味・嗜好】（アイスブレイク冒頭）
〇〇

【レベル2：強み・長所】（本題前）
〇〇

【レベル3：弱み】（本題途中・相手が一度笑った後）
〇〇

【禁止事項】
- 一度に全部話さない
- 最初から弱みを出さない
- 自分語りを30秒以上続けない
```

### 記入例

```
【レベル1：趣味・嗜好】
「私、休みの日は娘と公園に行くのが大好きなんですけど、〇〇さんは休みの日は何をされてるんですか？」

【レベル2：強み・長所】
「私は人の話を聞き出すのが得意で、毎回ニーズを的確に引き出すよう心がけています。」

【レベル3：弱み】
「実は僕、料理がめちゃくちゃ下手で、家事もほとんどできないんですよ（笑）」
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 冒頭で経歴を5分語る | 2秒×3回に分けて差し込む | 熟知の法則は「少しずつ」が効く |
| 最初から弱みを開示 | 弱みは最後 | 信頼の土台がないと不安にさせる |
| 業務の致命的弱みを開示 | プライベート系に限定 | 商談価値を壊す |
| 強みを断定形で言う | 「心がけている」で柔らかく | 自慢に聞こえる |
