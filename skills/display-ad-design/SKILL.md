# display-ad-design

GDN（Google Display Network）等のディスプレイ広告を、種類選定・ターゲティング・クリエイティブまで設計するスキル。

## 適用条件

- このスキルを使う場面：
  - GDN（Webサイト・YouTube・Gmail・アプリ）への画像/動画バナー出稿
  - リターゲティング広告の設計（自社サイト訪問者への再アプローチ）
  - ブランド認知拡大が目的の広告キャンペーン
- このスキルを使わない場面：
  - リスティング広告（検索連動）→ リスティング広告スキル
  - Meta広告（Facebook/Instagram）→ `meta-ad-campaign-design`
  - 4分類10種類から媒体を選定する段階 → `ad-mix-design`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 商品/サービス | 必須 | 広告対象 | 「ITサポート月額サービス」 |
| 広告目的 | 必須 | 認知拡大／リターゲティング／CV | 「リターゲティング」 |
| ターゲット | 必須 | 属性＋興味関心 | 「広島県・経営者・IT興味」 |
| 月予算 | 必須 | 媒体費 | 「月20万円」 |
| 既存サイト訪問者リスト | 任意 | リターゲティング用 | 「過去30日4,000UU」 |
| LP URL | 必須 | 遷移先 | 「example.com/lp」 |

## 実行ステップ

### ステップ1：広告目的を3つから決める

- 入力：商品特性、ビジネス課題
- 処理：以下から目的を選ぶ
  - **ブランド認知拡大**：新商品/新サービスの大量リーチ → CPM課金が効率的
  - **リターゲティング**：訪問済ユーザーの再来訪・購入促進 → CV重視
  - **直接CV**：購買意欲層への訴求 → 検索広告併用が前提
- 判断基準：
  - 認知系 → ディスプレイ単独可
  - CV系 → リスティング（検索）と併用必須（ディスプレイ単独だとCTR低い）
- 出力：広告目的＋主要KPI（リーチ数／再訪率／CV数）

### ステップ2：ディスプレイ広告5種類から選定する

- 入力：商品特性、保有素材、配信先
- 処理：以下5タイプから選ぶ
  | 種類 | 特徴 | 用途 |
  |---|---|---|
  | バナー広告 | 静止画/GIFアニメーション。サイドバー・記事内表示 | 商品画像でのプロモーション |
  | 動画広告 | YouTube等で配信される動画形式 | 感情訴求／商品紹介 |
  | レスポンシブ広告 | 配信先枠に合わせ自動調整 | デバイス問わず効率配信（基本これを選ぶ）|
  | ネイティブ広告 | コンテンツに自然調和する広告 | UX重視で違和感なく配信 |
  | リターゲティング広告 | 訪問済ユーザーへの再アプローチ | 離脱防止／CV回収 |
- 推奨組み合わせ：
  - 新規認知 → レスポンシブ＋動画広告（YouTube）
  - リターゲティング → リターゲティング広告＋バナー
  - 高UX重視 → ネイティブ広告
- 出力：選定タイプ＋配置先（Webサイト／YouTube／Gmail／アプリ）

### ステップ3：ターゲティングを設定する

- 入力：ターゲット属性、既存訪問者リスト
- 処理：以下の軸から設定
  - **デモグラ軸**：地域／年齢／性別／世帯収入
  - **興味関心軸**：興味カテゴリ（ペット好き／引っ越し検討中等）／購買意向
  - **行動軸**：過去訪問サイト／検索履歴
  - **リターゲティング軸**：自社サイト訪問者リスト（過去N日）／類似ユーザー
- リターゲティング設定例：
  - 過去1ヶ月以内の訪問者
  - カート離脱者
  - 特定ページ閲覧者
  - 既存顧客の類似ユーザー
- 出力：ターゲティング設定（属性＋リスト）

### ステップ4：予算と入札方式を決める

- 入力：月予算、広告目的
- 処理：
  - 日予算：月予算 ÷ 30日（最低1,000円/日〜）
  - 入札方式：
    - 認知系 → CPM入札（インプレッション課金）
    - CV系 → CPA最適化／クリック課金（CPC）
  - 配信期間を指定（短期キャンペーン or 継続配信）
- 出力：日予算／入札方式／配信期間

### ステップ5：クリエイティブを準備する（4要素）

- 入力：商品情報、ブランドガイドライン
- 処理：以下4要素を準備
  - **画像/動画**：ブランドカラー・ロゴで統一感ある素材
  - **タイトル**：30文字以内のキャッチコピー（`ad-copy-7-angles`参照）
  - **説明文**：90文字以内
  - **遷移先URL**：パラメーター付き（utm_source=gdn等）
- レスポンシブ広告の場合：
  - 画像 複数サイズ（横長 1200x628／正方形 1200x1200／縦長 1080x1350）
  - タイトル 5パターン
  - 説明文 5パターン
- 出力：クリエイティブ素材一覧

### ステップ6：出稿手順6ステップを実行する

- 入力：上記成果物
- 処理：
  1. **Google広告アカウント作成**：Google広告にアクセス→メアド登録→支払い情報登録
  2. **キャンペーン作成**：「ディスプレイ広告」選択→目標選択
  3. **ターゲット設定**：ステップ3の内容を入力
  4. **予算設定**：日予算・配信期間を入力
  5. **広告作成**：画像・タイトル・説明文・URLを入力
  6. **審査→配信開始**：自動審査後、即配信／管理画面で表示数・クリック数確認
- 出力：稼働中キャンペーン

### ステップ7：4注意点で運用品質を担保する

- 入力：稼働中キャンペーン
- 処理：以下4チェックを定期実施
  | # | 注意点 | チェック内容 |
  |---|---|---|
  | 1 | 過剰な広告表示を避ける | 誇大広告NG／LPと整合する訴求／フリークエンシー上限設定 |
  | 2 | ターゲティング精度を上げる | 興味関心の細分化／過去N日訪問者に限定／除外オーディエンス設定 |
  | 3 | クリエイティブの質を重視 | ブランドカラー統一／月1回更新／A/Bテスト |
  | 4 | 効果測定と最適化 | CTR/CPM/CPA週次レビュー／低CTR広告の差し替え |
- 出力：運用品質チェック結果

### ステップ8：ディスプレイ広告計画書としてまとめる

- 入力：ステップ1-7
- 処理：出力フォーマットに沿って整形
- 出力：ディスプレイ広告計画書

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 新規認知が目的 | レスポンシブ＋動画／CPM入札 | 大量リーチが効率的 |
| リターゲティングが目的 | リターゲティング広告＋バナー／CPC入札 | CVに近い層に再アプローチ |
| 月予算3万円未満 | リターゲティング1本に絞る | 認知用の予算が足りない |
| 既存訪問者リストなし | リターゲティング不可 → 興味関心ターゲティング中心 | リスト構築から開始 |
| BtoB商材 | リスティング広告と併用必須 | ディスプレイ単独だとCV取れない |
| BtoC高関与商材 | 動画広告＋ネイティブ広告 | 視覚訴求＋UX重視 |
| 広告ブロッカー対策 | ネイティブ広告比率UP | 通常のバナーは表示されない |
| 広告疲れ（フリークエンシー5超） | クリエイティブ刷新＋表示頻度抑制 | UX劣化を防ぐ |
| CTR 0.5%未満 | クリエイティブ全面刷新／ターゲット見直し | ディスプレイのCTR目安は0.5-1% |
| 検索広告で既に成果あり | 同じターゲット層にリターゲティングディスプレイ追加 | CV回収率UP |

### 記入例

```markdown
# ディスプレイ広告計画書 — ITサポート月額サービス

## 1. 広告目的
- 目的：リターゲティング（過去訪問者の再来訪・問い合わせ促進）
- 主要KPI：問い合わせ数 月10件／再訪率 5%

## 2. 広告タイプ
- 主軸：リターゲティング広告
- 補助：レスポンシブ広告
- 配信先：Webサイト＋YouTube

## 3. ターゲティング
- デモグラ：広島県＋近隣／35-65歳／全性別
- 興味関心：中小企業経営／ITサービス
- リターゲティング：過去30日訪問者（4,000UU）＋類似オーディエンス2%

## 4. 予算
- 月予算：20万円（日予算 約6,700円）
- 入札方式：CV最適化（問い合わせ）
- 配信期間：2026-05-01〜継続

## 5. クリエイティブ
- 画像：1200x628／1200x1200／1080x1350 各3パターン
- タイトル：「IT全部おまかせ、月3万円」「広島中小企業100社が選んだ」「ITトラブル放置していませんか？」
- 説明文：90文字以内×3パターン
- 遷移先URL：example.com/lp?utm_source=gdn&utm_campaign=retargeting-202605

## 6. 出稿手順チェック
- [x] Google広告アカウント／支払い情報
- [x] キャンペーン作成（ディスプレイ）
- [x] ターゲット設定
- [x] 予算設定
- [x] 広告作成
- [x] 審査通過・配信開始

## 7. KPI
- CTR：1%以上
- CPC：50円以下
- 問い合わせ数：月10件
- フリークエンシー：3-5回
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| ディスプレイ単独でCV狙う | リスティングと併用 | ディスプレイ単独はCV弱い |
| クリエイティブ1パターンで配信 | 3-5パターン用意してA/Bテスト | 広告疲れ早い |
| ターゲティング設定なしで配信 | 必ずデモグラ＋興味関心を絞る | 無駄打ちで予算消化 |
| LPと広告の訴求がズレる | 遷移先LPと統一感ある訴求 | 離脱率が跳ね上がる |
| 月1回もクリエイティブ更新しない | 月1回更新／フリークエンシー5超で刷新 | 広告疲れでCTR半減 |
| パラメーター付与なしで遷移 | 必ずutm付与 | 効果計測できない |
| リターゲティングリストなしでリターゲティング設定 | 先に訪問者リスト構築 | リターゲティング機能しない |
| 誇大広告／薬機法違反表現 | 薬機法・景表法準拠 | 配信停止・行政指導 |
| 広告ブロッカー想定なし | ネイティブ広告比率を上げる | 一定割合のユーザーに届かない |
| 認知系なのにCPC課金 | 認知系はCPM課金 | コスト効率が悪い |
