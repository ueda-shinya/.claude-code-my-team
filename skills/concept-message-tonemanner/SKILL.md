# concept-message-tonemanner

ブランド/事業のコンセプト→メッセージ→ビジュアルトンマナを段階的に固めるスキル。LP/サイト/広告/プロダクト制作前の土台作り。

## 適用条件

- このスキルを使う場面：
  - 新規ブランド/プロダクト立ち上げ時のコンセプト設計
  - 既存ブランドのリブランディング
  - LP/サイト/広告制作前のブランド土台確認
  - 制作チーム内でデザインの方向性合意
  - ブランドガイドライン作成
- このスキルを使わない場面：
  - MVV（Mission/Vision/Value）の組織内設計 → `mvv-design`
  - ブランドエクイティ全体設計 → `branding`
  - 価値提案そのもの → `value-proposition`
  - LP実装 → `lp-create`／`lp-design-system`
  - 広告コピー → `ad-copy-7-angles`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| ブランド/事業名 | 必須 | 対象 | 「LORANS（ロランス）」 |
| 業種 | 必須 | カテゴリ | 「コーヒー／スイーツ」 |
| 提供価値 | 必須 | 顧客への価値 | 「廃棄豆を活用したアートなスイーツ」 |
| 主要ターゲット | 必須 | ペルソナ | 「30代女性／自分へのご褒美層」 |
| 競合 | 任意 | 参考にする競合 | 「KIEHL'S／BAUM」 |
| MVV/ブランド理念 | 任意 | 既存があれば | 「世界中に幸せな団らんを広める」 |
| 既存ブランド資産 | 任意 | ロゴ・色・フォント等 | 「現状なし」 |
| 利用シーン | 任意 | LPかSNSか商品パッケージか | 「LP＋Instagram投稿」 |

## 実行ステップ

### ステップ1：コンセプト設計4要素を明確化する

- 入力：ブランド情報
- 処理：以下4要素を埋める
  | # | 要素 | 内容 | 質問 |
  |---|---|---|---|
  | 1 | ターゲットオーディエンス | ペルソナの明確化（年齢・居住地・ジェンダー・職業・所得）| 誰のために提供するか？ |
  | 2 | 競合分析と差別化要素 | 同業他社比較／自社のユニークな強み（製品・価格・評価・ニーズ）| 何が違うか？ |
  | 3 | ブランドビジョンとミッション | 長期目標（ビジョン）＋実現に向けた行動計画（ミッション） | どこを目指すか？／そのために何をするか？ |
  | 4 | ブランドストーリー | 共感を呼ぶ背景・理念・成り立ち | なぜ存在するか？ |
- 例（タイガー魔法瓶）：ビジョン「世界中に幸せな団らんを広める」→ミッション「ほかほかのご飯をいつでも食べられるように」
- 例（ナイキ）：有名選手の物語をストーリーに取り入れ感情的つながり構築
- 出力：コンセプト4要素シート

### ステップ2：コアメッセージを設計する

- 入力：コンセプト4要素シート
- 処理：以下3観点でコアメッセージを作る
  | # | 観点 | 内容 |
  |---|---|---|
  | 1 | メッセージの一貫性と適応 | 全タッチポイントで一貫して伝える／シーンに合わせて微調整 |
  | 2 | 利用シーンごとのメッセージング | カスタマーサポートは解決志向／広告は訴求力重視 |
  | 3 | 顧客の声を反映させたメッセージング | フィードバックを取り入れ共感を得る |
- コアメッセージは「提供する価値」「解決する問題」を端的に表現
- 1-2文で表現できること
- 出力：コアメッセージ＋シーン別バリエーション

### ステップ3：メッセージング3手法でアイデアを広げる

- 入力：コアメッセージ
- 処理：以下3手法のうち1-3つを実施
  - **手法1：マインドマップ連想ゲーム**
    - 中心に「自社製品/ブランド名」
    - 関連キーワードを3視点で広げる：
      - 消費者の視点（例：上品／洗練／贈り物／自分のご褒美／リラックス）
      - 企業・作り手の視点（例：パティシエ／クラフトマンシップ／新素材）
      - 社会・環境の視点（例：脱プラ／SDGs／代替食材／途上国農園）
    - 3視点の交差点に注目してコンセプトを抽出
  - **手法2：ブランディングサーベイ**
    - 社員/関係者にブランド関連質問を実施
    - 回答からコアになる言葉を抽出
    - `branding` スキル参照
  - **手法3：ブランド・アイデンティティ・プリズム**
    - プリズム6面（物理的特性／パーソナリティ／文化／関係性／反映／自己イメージ）に現れる言葉から展開
    - `branding` スキル参照
- 例（緑空間ビジネス）：自社→地域・事業・雇用・循環・ダイバーシティの5方向に広げ、各方向の言葉（活発さ／温かさ／賑やかさ／ギフト／生活／癒し／花咲く／植物／優しさ／開花／幸福感／生きる／HUMAN／はたらきがい／繋がり／助け合う／差別化／日本で一番／信頼感／成長／居場所／共生社会／サスティナブル／環境保全／心）を抽出
- 出力：マインドマップ＋抽出したコアキーワード10-20個

### ステップ4：トーン＆マナー（トンマナ）3要素を定義する

- 入力：抽出キーワード、コアメッセージ
- 処理：以下3要素で定義
  | # | 要素 | 内容 | 例 |
  |---|---|---|---|
  | 1 | ブランドのトーン（言葉遣いとスタイル） | ブランドの個性を反映する言葉の使い方 | 温かみ／プロフェッショナル／カジュアル |
  | 2 | ブランドのマナー（感情の伝え方） | 誠実さや共感を込めた表現 | 丁寧／フランク／熱量 |
  | 3 | メディア別トンマナ調整 | SNS／広告／Webサイト／カスタマーサポート別 | InstagramはカジュアルでWebはプロフェッショナル等 |
- 具体的なメッセージ文例を3-5個挙げて、各メディアでの使い分けを示す
- 出力：トンマナ定義シート（3要素＋メディア別文例）

### ステップ5：トンマナ4ヒントでビジュアル定義する

- 入力：トンマナ定義シート
- 処理：以下4ヒントで具体化
  - **HINT1：カラーコード＋フォント定義**
    - ベースカラー（背景）
    - メインカラー（主要色）
    - アクセントカラー（差し色）
    - ロゴカラー
    - 補助色（特定の表現用：例「柔らかさの表現＝グリーン」）
    - フォント（明朝/ゴシック／太/細／硬/柔の方向性）
  - **HINT2：ブランドパーソナリティ5タイプ**から選定
    | タイプ | 形容詞群 |
    |---|---|
    | 誠実 | 現実的／家庭的／素朴／素直／正直／誠実／親切／健全／健康そう／感情的／友好的／明るい |
    | 刺激 | 斬新／流行／ワクワク／活発／若々しい／勇ましい／積極的／創造的／ユニーク／自由／最先端／独立／現代的 |
    | 能力・適正 | 頼りになる／信頼／安心／責任感／勤勉／かしこい／理解力／組織立った／専門的／成功／上出来／指導者／自信 |
    | 洗練 | 上流階級／顔立ち良い／高級／グラマラス／魅力的／すてき／女性的／おしゃれ／知的／なめらか |
    | 頑丈 | アウトドア／力強い／男性的／頑丈／無骨／荒々しい |
    - 1-2タイプに絞る（複数混ざるとブレる）
    - 出典：株式会社日本カラーデザイン研究所
  - **HINT3：4象限／2局ゲージで視覚的合意**
    - **4象限分析**：
      - 縦軸「彩色 ↔ モノクロ」×横軸「くすみのある色 ↔ ビビット」
      - 縦軸「光・色鮮やか ↔ 影・モノトーン」×横軸「人工的 ↔ 自然的」
      - 縦軸「明/彩度高 ↔ 明/彩度低」×横軸「カジュアル ↔ 洗練」
    - **2局ゲージ**（8軸）：
      - 男性的 ↔ 女性的
      - 人工的 ↔ 自然的
      - サイエンス ↔ アート
      - エレガント ↔ エネルギッシュ
      - ラグジュアリー ↔ カジュアル
      - モノトーン ↔ 色鮮やか
      - 冷たい ↔ 温かい
      - 暗い ↔ 明るい
    - メンバー全員でゲージ位置を決め、合意取る
  - **HINT4：ムードボード作成**
    - 1枚のボード（PowerPoint 1ページでもOK）に集約
    - 含めるもの：カラーコード／テクスチャー／フォント／イメージに近い写真3-6枚
    - メンバー全員が同じイメージを共有できる成果物
- 出力：カラーコード／フォント／パーソナリティタイプ／象限ゲージ位置／ムードボード

### ステップ6：コンセプト・メッセージ・トンマナ統合シートとしてまとめる

- 入力：ステップ1-5
- 処理：出力フォーマットに沿って整形
- 出力：ブランドコンセプト統合シート

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| ターゲットが「全員」 | 必ず1ペルソナに絞る | 全員向けは誰にも刺さらない |
| 競合との差別化が「品質が高い」のみ | 製品/価格/評価/ニーズで複数軸の差別化を出す | 抽象的な差別化は機能しない |
| ブランドストーリーがない | 創業者の原体験／顧客の声／開発エピソードから抽出 | 共感を生む素材が必須 |
| コアメッセージが3行以上 | 1-2文に圧縮 | 覚えてもらえない |
| パーソナリティ5タイプから3つ以上選択 | 1-2タイプに絞る | 軸がブレる |
| トンマナを言葉だけで定義 | 必ず視覚情報（4象限／ムードボード）を併用 | 言葉だけでは伝わらない |
| メディア別調整なし | SNS／広告／Web／CSで個別調整 | 同じトーンが全媒体で機能することはない |
| ムードボードに10枚以上の画像 | 3-6枚に絞る | 多すぎると軸がブレる |
| カラーコード未決定で進む | 必ず最低3色（ベース／メイン／アクセント）決定 | 制作で迷う |
| BtoB商材で「カジュアル」のみ採用 | 「能力・適正」「誠実」要素を加える | BtoBは信頼軸が必須 |
| 高級ブランドで「カジュアル」採用 | 「洗練」中心に | ポジショニングがブレる |

### 記入例

```markdown
# ブランドコンセプト統合シート — LORANS（ロランス）

## 1. コンセプト設計（4要素）
- ターゲット：30-40代女性／都市部在住／世帯年収700万円以上／自分へのご褒美層
- 差別化要素：廃棄コーヒー豆を活用したアートなスイーツ（業界初）
- ビジョン：作り手と消費者と社会の三方良しの新しいスイーツ文化を作る
- ミッション：コーヒー2050年問題に向き合いながら、毎日に小さな幸福をお届けする
- ブランドストーリー：創業者がコーヒー農園視察で目撃した廃棄豆問題と、独自の粉砕技術開発の3年間

## 2. コアメッセージ
- 主メッセージ：「捨てられる豆から、捨てられない瞬間を。」
- シーン別バリエーション：
  - LP：「コーヒーの新しいレシピ。社会貢献消費という、上質な選択。」
  - Instagram：「ちょっといい一日に、ちょっといいスイーツ。 #LORANS」
  - カスタマーサポート：「お問合せありがとうございます。お気軽にご相談ください。」

## 3. キーワード（マインドマップ抽出）
- 消費者視点：上品／洗練／贈り物／自分のご褒美／リラックス／華やかさ／香り
- 作り手視点：パティシエ／バリスタ／クラフトマンシップ／粉砕技術／新素材／代替食材
- 社会・環境視点：コーヒー2050年問題／廃棄豆／環境問題／SDGs／途上国農園
- コアキーワード10選：上品／クラフトマンシップ／社会貢献消費／自分のご褒美／代替食材／アートなスイーツ／華やかさ／持続可能／創造的／知的

## 4. トンマナ3要素
- ブランドのトーン：洗練と温かみのバランス／知的だが親しみやすい
- ブランドのマナー：誠実で控えめ／押し付けがましくない／顧客の選択を尊重
- メディア別調整：
  - LP：洗練×情報量豊富（じっくり読ませる）
  - Instagram：温かみ×ビジュアル中心（短文＋画像）
  - 広告：訴求力強め（数字・社会性をフロント）

## 5. ビジュアル定義
- カラーコード：
  - ベース：オフホワイト #F5F2EC
  - メイン：コーヒーブラウン #6B4226
  - アクセント：ゴールド #C9A646
  - ロゴ：ダークブラウン #3D2817
  - 補助：オリーブグリーン #7A8B4E（自然・社会性表現）
- フォント：欧文 Playfair Display／和文 游明朝（洗練×知的）
- ブランドパーソナリティ：洗練（メイン）＋誠実（サブ）
- 4象限位置：「光・色鮮やか × 自然的」象限
- 2局ゲージ位置：女性的6/人工2/アート8/エレガント7/ラグジュアリー6/色鮮やか4/温かい7/明るい6

## 6. ムードボード
- 添付：mood-board-lorans.pdf（PPTスライド1ページ）
- カラーパレット：オフホワイト＋ブラウン＋ゴールド＋オリーブの4色実物パレット
- フォントサンプル：「LORANS」表記をPlayfair Displayで配置
- イメージ写真：①コーヒー豆クローズアップ ②上品なカフェ空間 ③女性が手にスイーツを持つ ④自然光でのスイーツ撮影 ⑤クラフトマンシップ感じる職人の手元（計5枚）
```

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| ターゲットを「20-60代女性」 | 1ペルソナ（30代独身女性／世帯年収700万）に絞る | ペルソナ広いと刺さらない |
| 競合差別化が「品質」「サービス」のみ | 製品/価格/評価/ニーズの複数軸で具体化 | 抽象表現は実行に落ちない |
| コアメッセージが3行以上 | 1-2文に圧縮 | 覚えてもらえない |
| パーソナリティ5タイプ全部該当 | 1-2タイプに絞る | 軸がブレる |
| トンマナを言葉のみで定義 | 4象限／2局ゲージ／ムードボード併用 | 視覚情報なしでは伝わらない |
| ムードボードに15枚以上画像 | 3-6枚に絞る | 軸がブレて合意できない |
| カラーコード未決定で制作開始 | 最低3色決定してから | 制作で迷子になる |
| メディア別調整なしで全媒体同じトーン | SNS／広告／LP／CSで個別調整 | メディア特性無視は機能しない |
| 競合と全く同じトンマナを採用 | 必ず1軸以上で差別化 | ポジショニング消失 |
| 制作チームに口頭でトンマナ説明 | ムードボード＋定義シートを文書化 | 解釈ブレが起きる |
