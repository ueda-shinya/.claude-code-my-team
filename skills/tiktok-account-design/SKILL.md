# tiktok-account-design

SNS戦略3段階フローの**第3段（媒体特化）**として、TikTokを「未来の顧客に出会う発見メディア」と捉え、運用目的→企画→台本→施策→分析改善のサイクルを実行可能な形で設計する。フォロワー数に依存しないアルゴリズム特性を活かし、非フォロワーへの初回接点を最大化する。

## SNS戦略3段階フロー上の位置づけ

| 段階 | スキル | 役割 |
|---|---|---|
| ① 第1段 | `sns-strategy-overview` | 媒体選定／KGI・KPI設計 |
| ② 第2段 | `sns-content-design` | **アカウントテーマ／ターゲット態度変容5区分／ポジショニング4象限／コンセプト一言／コンテンツ分類ツリー**（本スキルではここを再実施しない） |
| ③ 第3段（本スキル） | `tiktok-account-design` | TikTok固有のアルゴリズム理解・機能配置・台本・施策・分析改善 |

**前段からの引き継ぎ前提**：本スキルは `sns-content-design` の出力（具体ペルソナ／ポジショニング4象限／コンセプト一言／コンテンツ分類ツリー）が**既に確定している前提**で起動する。媒体特化スキル間（IG/TikTok/YT）でこれらを重複実施しない。

### 実行停止プロトコル（必須遵守）

本スキル起動時、最初に以下を確認する。**いずれかの前段成果物が引き継がれていない場合は本スキルの実行を直ちに停止する**:

- [ ] **第1段成果物**: `sns-strategy-overview` のアウトプット（媒体選定／KGI/KPI／アカウントタイプ）が確定済みか
- [ ] **第2段成果物**: `sns-content-design` のアウトプット（具体ペルソナ／ポジショニング4象限／コンセプト一言／コンテンツ分類ツリー）が確定済みか

**未確定の場合の対応**:
1. 本スキルの実行を**停止**する（ステップ1以降に進まない）
2. ユーザー（または呼び出し元）に「前段（`sns-strategy-overview` / `sns-content-design`）が未実施です。先に起動してください」と通知する
3. 確定済みアウトプットが提供されたことを確認してから、本スキル ステップ1から実行を開始する

**例外**: シンヤさんが「前段スキップで直接TikTok設計だけ進める」と明示した場合のみ、警告ログを残した上で本スキルを実行可（Notion 案件管理に「前段スキップ例外」と記録必須）。

## 適用条件

- このスキルを使う場面：
  - 第2段の `sns-content-design` 完了後、TikTokを採用したケースの媒体特化設計
  - 既存TikTok運用が伸び悩んでおり、アルゴリズム・台本・分析からテコ入れしたいとき
  - 認知拡大／販売促進／採用強化のいずれかでTikTokを使うとき
  - 台本作成・分析改善・勝ちパターン展開のいずれかを実行したいとき
  - TikTok Shop／ライブコマース導入を検討するとき
- このスキルを使わない場面：
  - SNS全体戦略（媒体選定）の設計 → `sns-strategy-overview`
  - アカウントテーマ・ポジショニング・コンセプト・ターゲット具体化 → `sns-content-design`
  - Instagram／YouTube単体の設計 → 各媒体特化スキル
  - 投稿1本のクリエイティブ制作 → コピー／デザイン担当へ
  - 純粋な動画編集テクニック → 別スキル／外注へ

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 運用目的 | 必須 | 認知拡大／販売促進／採用強化のいずれか | 販売促進（D2C） |
| 主要ターゲット | 必須 | デモグラ＋サイコグラ | 仕事終わりにスマホを見る20代後半女性 |
| 商材／サービス | 必須 | 売る／伝える対象 | レザーバッグ |
| USP（独自強み） | 必須 | 競合にない切り口 | お手入れ不要の革 |
| 競合アカウント | 任意 | 同業界の参考2〜3アカウント | 未指定の場合はステップ3で調査依頼 |
| TikTok Shop活用 | 任意 | あり／なし | あり |
| 既存運用状況 | 任意 | フォロワー数・投稿頻度・現状指標 | フォロワー800・週1投稿 |

## 実行ステップ

### ステップ1：運用目的を3パターンから選定する

- 入力：事業ゴール、商材タイプ
- 処理：
  1. TikTokを「発見メディア（まだ知らない潜在層に届く場所）」として捉え直す
  2. 目的を以下3パターンから1つ選ぶ（複数選択不可、副目的までで2つまで）：

  | 目的 | 主要シーン | KPI候補 |
  |---|---|---|
  | 認知拡大 | 新規顧客との初回接点／ブランド認知 | 再生数／リーチ／インプレッション |
  | 販売促進 | TikTok Shop経由の購入／ライブコマース | 購入数／売上／リンククリック |
  | 採用強化 | 社内文化の可視化／応募促進 | 応募数／プロフィールクリック数 |

  3. 「TikTokは即効的に売上を作る場所ではない」と認識し、ジャーニー5区分（認知→興味→比較検討→行動→リピート）の導線を他SNS（Instagram／YouTube／LINE等）と組み合わせる前提を確認する
- 出力：運用目的選定書（主目的1＋副目的0〜1）

### ステップ2：3フェーズアルゴリズムを理解して投稿戦略に反映する

- 入力：運用目的選定書
- 処理：
  1. 以下の3フェーズを把握し、各フェーズで重要視される指標を投稿設計に反映する：

  | フェーズ | 仕組み | 重要指標 |
  |---|---|---|
  | ①テストフェーズ | 投稿直後、少数ユーザーにテスト配信 | 視聴完了率／いいね・コメント・シェア |
  | ②おすすめ表示フェーズ | 反応良ければ多数ユーザーへ拡散 | 視聴維持率／エンゲージメント率／メタデータ（キャプション・ハッシュタグ・音声） |
  | ③バズフェーズ | 広い層・他地域へ拡散 | 再生回数／視聴維持率／リピート再生／シェア |

  2. テストフェーズで止まる動画が多い場合 → 冒頭3秒・視聴完了率の改善に集中する
  3. おすすめ表示フェーズで止まる場合 → メタデータ（キャプション・ハッシュタグ・音声）と維持率の改善に集中する
- 出力：フェーズ判定＋改善優先項目

### ステップ3：競合リサーチで「いま伸びる切り口」を特定する

- 入力：商材／サービス、主要ターゲット
- 処理：
  1. 似た商品・サービスのTikTokアカウントを検索して2〜3アカウント観察する
  2. 各アカウントについて以下3項目を記録する：
     - **冒頭3秒で何を言っているか**：スクロールを止めるフレーズ・映像
     - **話し方・テンポ**：間の取り方／カット割り／話速
     - **コメント欄の反応**：視聴者のリアルなニーズ・疑問・共感ポイント
  3. 伸びている共通要素を2〜3個抽出する
- 出力：競合リサーチ表＋伸びる切り口リスト

### ステップ4：TikTok向け企画方針を確定する（前段からの引き継ぎ＋TikTok特化の補強）

- 入力：運用目的選定書、競合リサーチ表、USP、`sns-content-design` で確定した具体ペルソナ／コンセプト一言
- 処理：
  1. **前提確認**：`sns-content-design` の出力（具体ペルソナ／コンセプト一言）が引き渡されていることを確認する。未実施なら本スキルを止めて `sns-content-design` の起動を案内する
  2. **ペルソナはそのまま流用**（再生成しない）。「仕事終わりにスマホを見る20代後半女性」のような具体ペルソナを `sns-content-design` から引き継ぐ
  3. **TikTok特化の補強として「動画1本のメッセージ絞り込み」を行う**：1動画1メッセージ。「自分たちが言いたいこと」ではなく「相手が知りたいこと」をベースに考える
  4. **TikTokトレンド・競合**（ステップ3結果）を反映して、ペルソナに刺さる切り口を3〜5本ピックアップする
  5. **コンセプト一言を「TikTokトーン」にリメイクする**（再生成ではなく、`sns-content-design` のコンセプト一言を**短尺動画文脈で言い換える**）：
     - `sns-content-design` の例：「30代からの本格スキンケアメディア」
     - TikTokトーン例：「忙しい毎日にスッと馴染む、30代スキンケアの正解集」
     - 同じコンセプトを保ちつつ、TikTok特有の短尺・テンポ・親しみのある言い回しに調整する

> **重要**：コンセプト一言・ターゲット具体化は `sns-content-design` の成果を**そのまま流用**する。媒体特化スキル側で再生成すると、Instagram/YouTubeとの整合性が崩れて出力統合不能になる（過去の重大指摘B）。
- 出力：TikTokトーン化したコンセプト一言＋TikTok向け企画切り口リスト

### ステップ5：4機能を目的に応じて配置する

- 入力：コンセプト一言、運用目的選定書
- 処理：
  1. 4機能の特性を以下表で確認する：

  | 機能 | 役割 | 重要指標 | 成功ポイント |
  |---|---|---|---|
  | ショート動画 | 認知獲得・理解促進／非フォロワーとの新接点 | 視聴維持率／アクション率 | 冒頭3秒のフック／テンポと構成（1動画1ネタ）／テロップ・音源・構成／CTA設置 |
  | ライブ配信 | リアルタイム接点／CV（販売・応募）／拡張性高 | コメント数／同時接続数／購入数 | 固定コメント・特典／コメント拾い／事前告知／回遊設計 |
  | プロフィール設計 | 動線設計（TikTok内で唯一の外部リンク設置場所） | プロフィールクリック率／外部遷移数 | ターゲット明確化／世界観統一／誘導先リンク整備／固定動画活用 |
  | TikTok Shop | ダイレクト購入／ライブ内販売／クーポン | 購入数／購入率 | Shop連携アカウント／ライブ限定特典／構成台本＋MC力／コメント対応 |

  2. 目的別に推奨される機能優先度を以下から選ぶ：

  | 目的 | 優先機能 | 戦術 | 実践例 |
  |---|---|---|---|
  | 認知拡大 | ショート動画／ハッシュタグ／クリエイター起用 | 冒頭3秒フック徹底／トレンド音源活用／一貫性 | 商品紹介／使い方／ハッシュタグチャレンジ |
  | 販売促進 | TikTok LIVE／TikTok Shop | 定期ライブ／特典付ライブ／動画→Shop誘導 | ライブコマース／限定クーポン／商品タグ付投稿 |
  | 採用強化 | ショート動画／TikTok LIVE／コメント対応 | 社内雰囲気発信／社員起用動画／採用情報定期発信 | 社内ルーティン公開／採用説明会ライブ／社員インタビュー |

  3. 初期段階は「認知獲得」を主軸に1機能から始め、徐々に他機能を追加する
  4. **ジャーニー5区分（認知／興味／比較検討／行動／リピート）との対応**：TikTokは特性上「認知」フェーズが圧倒的に強く、次に「興味」、TikTok Shop/ライブ活用時は「行動」もカバーする。「比較検討」「リピート」は他媒体（Instagram／YouTube／LINE等）に渡す前提で設計する（本3段階フロー全体で語彙統一）
- 出力：機能配置マトリクス（機能×頻度×担当×担当ジャーニー区分）

### ステップ6：台本3ステップで動画ごとの構成を設計する

- 入力：コンセプト一言、機能配置マトリクス
- 処理：
  1. 各動画は以下3ステップで構成する：

  | STEP | 役割 | 内容 |
  |---|---|---|
  | ①つかみ（最初の3秒） | スクロールを止める | 驚き・質問・共感ワードで視聴者の注意を引く（「え？なにこれ？」と思わせる） |
  | ②メイン（1つだけ伝える） | 内容を理解させる | 伝えたい内容をシンプルに、例を交えて話す（情報詰め込み禁止） |
  | ③最後にひとこと | 行動につなげる | 「フォローしてね」「◯◯で検索」など。CTAを強く入れすぎると完全視聴率が下がるため、自然な誘導に留める |

  2. 動画1本につきメッセージは1つに絞る（複数禁止）
  3. テロップは内容を強調・補完。話す内容と映像とテロップを一致させる
  4. トレンド音源を活用しつつ、世界観と合わない場合は採用しない
- 出力：台本フォーマット（動画1本分のひな型）

### ステップ7：3施策とスケジュール（曜日別テーマ）を設計する

- 入力：機能配置マトリクス、運用目的選定書
- 処理：
  1. 以下3施策から組み合わせる：

  | 施策 | 目的 | 内容 |
  |---|---|---|
  | フィード投稿 | 認知拡大 | 商品の使い方紹介／ストーリー性投稿／トレンド活用 |
  | TikTok Shop | 購入 | 投稿・ライブから直接販売／Shopタブ／クーポン・セール／クリエイター販売連携 |
  | TikTok LIVE | 興味を深める | Q&A配信／実演販売／限定キャンペーン |

  2. 「思いついたら投稿」は禁止。曜日ごとに種別とテーマを固定する：
     - 例：月曜＝短尺動画（商品魅力テンポ良く）／木曜＝ライブ配信（人気商品TOP3実演）／金曜＝ストーリーズ（限定クーポン告知）
  3. ライブ配信を含む場合の3コツ：
     - **実物を見せる**：サイズ感・質感などリアル情報
     - **比較する**：他商品・以前モデルとの違いを視覚的に
     - **限定企画にする**：ライブ限定特典・割引で参加価値UP
  4. TikTok Shop活用時の3ポイント：
     - **値段・在庫管理を徹底**（リアルタイム更新／更新担当を明確化）
     - **限定特典を用意**（「ライブ中だけ◯%OFF」「限定ノベルティ付き」）
     - **送料・返品ルールを事前明示**（信頼トラブル防止）
- 出力：施策プラン＋曜日別スケジュール表

### ステップ8：5基本指標で分析する

- 入力：投稿実績データ
- 処理：
  1. 以下5指標を確認する：

  | # | 指標 | 意味 |
  |---|---|---|
  | 1 | 再生数 | 投稿が何人の目に触れたか |
  | 2 | 最初の3秒での離脱率 | 冒頭で離脱していないか |
  | 3 | 視聴完了率 | 最後まで見られているか |
  | 4 | エンゲージ率 | 反応（いいね・コメント・保存・シェア）の割合 |
  | 5 | プロフィールやショップへの遷移数 | 動画きっかけの行動数 |

  2. 「ジャンル×訴求」のマトリクスで成功・失敗を整理する：
     - 縦軸：投稿ジャンル（Vlog系／ハウツー系／商品紹介系）
     - 横軸：訴求軸（高品質／永年補償／今だけ◯%オフ等）
     - 各セルに ◎○△× を記入し、「効く組み合わせ」を特定する
- 出力：5指標サマリー＋ジャンル×訴求マトリクス

### ステップ9：分析改善5ステップで「1つずつ」変える

- 入力：5指標サマリー、ジャンル×訴求マトリクス
- 処理：
  1. 伸びない理由を以下表で特定する：

  | 伸びない理由 | 主な原因 | 解決策 |
  |---|---|---|
  | 再生数が少ない | ネタ・テーマが弱い／タイトル・テロップが刺さらない／投稿時間ズレ／サムネ・冒頭の引き弱い／一貫性なし | 「誰が見ても気になる切り口」設計／タイトルを自分ごと化／ターゲットアクティブ時間に投稿／冒頭に結果・結論／シリーズ化 |
  | 冒頭ですぐ離脱 | 最初3秒に目的・面白さが伝わらない／話し出しゆっくり／映像単調／抽象的すぎ／期待ズレ | 最初3秒で「何の話か」「なぜ見るべきか」明確化／結論→理由→詳細の順／変化（動き・効果音・表情）／テロップで先出し／問いかけ・共感ワード |
  | 最後で視聴落ちる | 話が長くテンポ悪い／山場が中盤／結論・オチ弱い／CTA唐突／エンディング長い | 1テーマ1メッセージで短く／序盤引き→中盤展開→終盤納得／意外性・まとめ一言／自然なCTA／エンディング短く |

  2. 改善は以下5ステップで「1つずつ」変える：
     - **STEP1**：課題を特定する（例：再生数が少ない／離脱が多い）
     - **STEP2**：原因を仮説立てする（例：ネタが弱い／テンポが遅い）
     - **STEP3**：最も影響が大きい部分から1つずつ変える（複数同時変更は禁止）
     - **STEP4**：次の投稿で検証し、効果を比較する
     - **STEP5**：うまくいった改善を「再現して仕組み化」する
  3. 同時に複数要素を変えると、何が効果を出したか判断できなくなるため厳守する
- 出力：改善プラン（課題1＋原因仮説＋次に試す改善1）

### ステップ10：勝ちパターンを5工夫で広げる

- 入力：改善プラン、過去のヒット動画
- 処理：
  1. 当たった動画の要素を以下5パターンで再利用する：

  | 工夫 | 再利用方法 | 期待効果 |
  |---|---|---|
  | 話す人を変える | 同じテーマを別の人が話す | 視点違いで新鮮さ・リーチ層拡大 |
  | 季節を変える | 季節・時期に合わせて撮り直す | タイムリー感UP・保存伸び |
  | 商品の色・モデルを変える | 見た目を変えて再投稿 | 視覚変化で飽きさせない |
  | 切り口を変える | 「Aの方法」→「Bの方法」など比較形式 | 教育・エンタメ要素強化 |
  | フォーマットを変える | ロング→ショート→インタビュー等 | 新層に届きやすくなる |

  2. 「再現して仕組み化する」をルール化する：勝ちパターンは1度きりにせず、構成テンプレートとして言語化し、チームで共有する
- 出力：勝ちパターン展開リスト

### ステップ11：法律・規約遵守を確認する

- 入力：制作した動画・台本
- 処理：以下3点を必ずチェックする：

| 注意事項 | 内容 |
|---|---|
| 禁止ワード | 暴力的・差別的表現／過度な誹謗中傷／性的・センシティブな内容は即NG |
| 薬機法（医薬品・化粧品） | 「効果がある」「治る」「絶対」など効能効果を断定する表現は法律で制限 |
| 著作権・音源使用ルール | 市販音源・他者の映像・画像の無断使用は著作権侵害の可能性 |

  - ペナルティを受けるとアルゴリズム上の露出が極端に下がるため、投稿前にダブルチェックを行う
- 出力：法令・規約遵守チェック済みフラグ

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 目的が「短期で売上を作る」 | TikTok単独では非推奨／他SNS・広告と組合せ前提 | TikTokは認知導線であって即購入導線ではない |
| ターゲットが10代〜20代前半 | TikTok優先（ボリュームゾーン適合） | 平均年齢36歳でも若年層がコア |
| ターゲットが40代以上 | TikTok単独は非推奨／IG・YouTube優先 | ボリュームゾーンが合わない |
| フォロワー0でも始められるか | 始められる（フォロワー数非依存アルゴリズム） | おすすめフィードが主流通経路 |
| 投稿フェーズが「テストフェーズ」で止まる | 冒頭3秒・視聴完了率を改善に集中 | テスト通過の判定基準 |
| 「おすすめ表示フェーズ」で止まる | メタデータ（キャプション・ハッシュタグ・音声）と維持率に集中 | 拡大判定の基準 |
| 1動画に複数メッセージ詰め込み | 1メッセージに絞る | 何が大事か視聴者が理解できない |
| トレンド音源と世界観が合わない | 世界観優先・音源は採用しない | 世界観の一貫性が信頼の源泉 |
| 改善で複数要素を同時に変える | 1つずつ変える | 効果要因が判別不能になる |
| ライブ視聴者ゼロ | 事前告知3回以上＋限定特典＋台本準備 | 集客動線がないと誰も来ない |
| TikTok Shop売れない | 値段・在庫管理／限定特典／送料返品ルール明示の3点を確認 | 信頼を失うとShop全体が売れなくなる |
| 薬機法に触れる表現 | 即修正（「効果がある」「治る」等の断定禁止） | アカウントBANリスク |
| ヒット動画が出た | 5パターン（話す人／季節／色モデル／切り口／フォーマット）で再利用 | 1度きりで終わらせず勝ちパターン化 |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 「とりあえず投稿」で始める | 目的・ターゲット・コンセプトを決めてから始める | 戦略なき投稿は成果に繋がらない |
| ターゲットを「20〜30代女性」など広く設定 | 「仕事終わりにスマホを見る20代後半女性」など具体的に | メッセージが研ぎ澄まされない |
| 1動画に複数メッセージ詰め込む | 1動画1メッセージに絞る | どれが大事か視聴者が理解できない |
| 「自分たちが言いたいこと」を発信 | 「相手が知りたいこと」をベースに発信 | 一方通行の発信は共感されない |
| 冒頭3秒に挨拶や前置き | 冒頭3秒で結果・結論・問いかけ | スクロールで離脱される |
| トレンド音源と世界観が合わなくても使う | 世界観優先・合わない音源は不採用 | ブランド一貫性が崩れる |
| CTAを強く入れすぎる | 自然な誘導に留める | 完全視聴率が下がりアルゴリズム評価が落ちる |
| 改善で複数要素を同時に変える | 1要素ずつ変えて検証 | 効果要因が判別不能 |
| ヒット動画を1度きりで終える | 5パターン（人／季節／色モデル／切り口／フォーマット）で再利用 | 勝ちパターンの再現性を失う |
| ライブを告知なしで実施 | 事前告知3回＋限定特典＋台本準備 | 視聴者ゼロになる典型 |
| TikTok Shopで在庫切れ放置 | 在庫・価格をリアルタイム更新／更新担当を明確化 | 信頼を失いアカウント全体に影響 |
| 薬機法・著作権を軽視 | 投稿前にダブルチェック | アカウントBANで全コンテンツ消失 |
