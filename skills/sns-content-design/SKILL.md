# sns-content-design

媒体選定後の「次の一手」として、コンテンツの中身を設計する。マーケティングファネル全体でのSNS位置づけ→4手法の組み合わせ→BtoB/BtoCの目的差分→コンテンツ設計3ステップで、ターゲットの態度変容（COLD→COLDα→MID）に応じた発信内容を体系化する。

## 適用条件

- このスキルを使う場面：
  - SNSの媒体選定後、コンテンツの中身を体系的に設計したいとき
  - BtoB／BtoCで運用目的・コンテンツ方針を分けて設計したいとき
  - オーガニック投稿／広告／インフルエンサー／UGCの組み合わせを設計したいとき
  - アカウントのテーマ・トーン・画像イメージを4象限で整理したいとき
  - 投稿コンテンツが思いつかない／散漫になっているとき
- このスキルを使わない場面：
  - 媒体選定段階の戦略設計 → `sns-strategy-overview`
  - Instagram／TikTok／YouTube単体の機能・企画設計 → 各媒体特化スキル
  - 投稿1本のクリエイティブ制作 → コピー／デザイン担当へ

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 業態タイプ | 必須 | BtoC／BtoB／採用 | BtoB（コンサル） |
| 運用目的 | 必須 | 認知／興味検討／購入維持／採用 のどれを主に強化するか | 興味検討（リード獲得） |
| 主要ターゲット | 必須 | デモグラ＋サイコグラ | 30代経営者・SNSが苦手 |
| 自社の強み（USP） | 必須 | 競合にない切り口 | 売上1億円までの再現性ある営業設計 |
| 競合アカウント | 任意 | 同業界の参考3〜5アカウント | 未指定の場合はステップ4で調査依頼 |
| 媒体 | 必須 | Instagram／X／TikTok／YouTube／LinkedIn 等（複数可） | Instagram + X |

## 実行ステップ

### ステップ1：マーケティングファネル全体での役割を確定する

- 入力：運用目的、媒体
- 処理：
  1. SNSがマーケティングファネル全体（認知→興味関心→検索→比較検討→購入→リピート→ロイヤル化）の**全フェーズで活躍可能**であることを確認する
  2. 自社が強化したい段階を以下3区分から特定する：

  | 段階 | 主な役割 | 該当例 |
  |---|---|---|
  | 認知段階 | ブランドや商品を多くのユーザーに認知させる | Instagram広告／TikTokバズ企画 |
  | 興味・検討段階 | 商品サービスへの興味を喚起し検討を促進 | YouTubeで解説／レビュー動画配信 |
  | 購入・維持段階 | 購入促進やリピートを促す | Facebookで限定クーポン／フォロワー限定セール |

  3. SNSマーケティングの3特徴を強化施策に必ず織り込む：
     - **双方向性**：ユーザーとリアルタイムでコミュニケーション
     - **拡散性**：投稿シェアで大勢に情報が届く
     - **ターゲティング精度**：年齢・性別・興味関心に基づく精密な広告配信
- 出力：強化フェーズ＋3特徴の活用方針

### ステップ2：BtoB／BtoC で運用目的を切り分ける

- 入力：業態タイプ、運用目的
- 処理：
  1. SNSの主要運用目的7パターンから自社が狙うものを特定する：
     - 認知獲得／問い合わせ獲得／（EC）購入者獲得／オウンドメディアとして／ファンとの交流／リード獲得（ウェビナー/WP/ハウスリスト）／採用
  2. **業態別の発信コア**を以下から選ぶ：

  | 業態 | コア訴求 | 補足 |
  |---|---|---|
  | BtoC | 「共感」と「親しみやすさ」 | 商品魅力の視覚訴求／世界観／日常感 |
  | BtoB | 「ターゲットとなるユーザーを集め、商品・サービス価値を感じてもらうこと」 | 企業アカウントの多くは採用目的／リード獲得は個人名で有益コンテンツ継続発信のリファラ問合せが主流 |
  | 採用 | 社内文化・価値観の発信 | 理念・社風・社員の声 |

  3. BtoBの場合、企業アカウント（採用）と個人アカウント（リード獲得）のどちらで運用するかを決める
- 出力：業態別運用目的書

### ステップ3：4手法を組み合わせる

- 入力：強化フェーズ、業態別運用目的書
- 処理：
  1. 以下4手法から自社が使う手法を選ぶ（複数選択可）：

  | 手法 | 目的 | 例 |
  |---|---|---|
  | オーガニック投稿 | フォロワーとの関係性構築・ブランド認知向上 | 日常投稿／キャンペーン情報／ストーリーズ更新 |
  | 広告 | 特定ターゲット層へのリーチ・コンバージョン | Facebookクリック誘導型広告／Instagramショッピング広告 |
  | インフルエンサーマーケティング | インフルエンサーの影響力で商品・サービスをプロモーション | TikTokコラボ動画／Instagramレビュー投稿 |
  | UGC活用 | ユーザー作成コンテンツ（レビュー写真・動画）をプロモーション活用 | ハッシュタグキャンペーンで投稿収集・拡散 |

  2. 4手法の優先順位を決める（リソース・予算・現状フォロワー数で判定）
  3. 初期段階ではオーガニック投稿を主軸に、効果が出てから広告・インフルエンサー・UGCを追加する
- 出力：4手法の組み合わせ＋優先順位

### ステップ4：媒体ごとのアカウントテーマを決める（コンテンツ設計STEP1）

- 入力：業態別運用目的書、媒体、自社の強み
- 処理：
  1. SNS運用の目的・自社の強み・競合との差別化を整理してから、アカウントのテーマを決める
  2. 媒体ごとに以下5項目を埋める（媒体併用時は媒体別に作る）：

  | 項目 | 内容例（保育園のIG） | 内容例（保育園のX） |
  |---|---|---|
  | 特徴 | 画像中心のビジュアル訴求／ハッシュタグでターゲットリーチ | ユーザー間距離が近く共感を得やすい／情報収集として使う |
  | コンテンツのテーマ | 職員の仲睦まじい様子／理念（誉める保育/夢中で遊べる環境）／採用情報 | 園の日常／新卒生・保護者の声／社風・理念／採用情報 |
  | 訴求点 | 求職者が気にする職員同士の仲の良さ／自社特有の保育方法と考え方 | 日々の園の様子や雰囲気／OOOで働くというイメージ |
  | 運用周期 | 週2〜3回 | 週2〜3回 |
  | ハッシュタグ | #xxx（4〜5個） | #xxx（1〜2個） |

  3. **画像イメージを4象限で整理する**（業態別に縦軸・横軸を選ぶ）：
     - 化粧品例：「光・色鮮やか⇄影・モノトーン」×「人工的⇄自然的」／「カジュアル⇄洗練」×「明・彩度高⇄明・彩度低」
     - 競合アカウントを4象限にプロットし、自社の配置と差別化ポイントを言語化する
  4. イメージに近い他社アカウント・イメージと異なるアカウントを各3つピックアップして方向性をすり合わせる
- 出力：媒体別アカウントテーマ表＋画像イメージ4象限マップ

### ステップ5：ターゲットがファネル別に知りたい情報を精査する（コンテンツ設計STEP2）

- 入力：媒体別アカウントテーマ表、主要ターゲット
- 処理：
  1. ターゲットの態度変容を3段階に分類する（金融・不動産の例）：

  | 段階 | 興味関心 | 実施経験 | 知りたい情報 | 提供すべき価値 |
  |---|---|---|---|---|
  | COLD | 無 | 無 | お金の基礎知識／身近な例／将来への不安 | 認知 |
  | COLD α | 有 | 無 | 自分の現在地／資産運用の考え方／投資のリスク | 機会＋知識 |
  | MID | 有 | 有 | 20代の今だから得られるメリット／今やる理由／今できる選択肢 | 優先順位の提供 |

  2. 段階間の **GAP（情報不足ゾーン）** を特定し、そこを埋めるコンテンツを優先する
  3. 自社のターゲットを上記表と同じ構造（興味関心×実施経験のマトリクス）で配置し、各段階で必要な情報を箇条書きで列挙する
- 出力：ターゲット態度変容マトリクス＋段階別必要情報リスト

### ステップ6：具体コンテンツ分類を決める（コンテンツ設計STEP3）

- 入力：段階別必要情報リスト、自社の強み
- 処理：
  1. 投稿コンテンツのアイデアを以下のような大カテゴリ→中カテゴリに分解する（不動産投資の例）：

  | 大カテゴリ | 中カテゴリ |
  |---|---|
  | 専門性 | 不動産投資の仕組み／よくある誤解／向いている人向いていない人／投資の注意点／会社の見極め方／担当者の見極め方 |
  | 企業の信頼性 | 会社について（規模/業績/実績／企業理念パーパス／クリーンさ／アフターフォロー）／社員について（清潔感／専門性／人柄／顧客との関係性） |
  | 業界の嘘/ほんと | 儲からない？／こんな営業トークには気をつけろ／不動産は見て買うべき？／今買うべき？ |
  | お金の話 | 節税について／金利について／資産形成の考え方 |

  2. ユーザーが知りたい情報と意思決定要因の仮説を照らし合わせて、投稿するコンテンツを決定する
  3. 各コンテンツの優先度を「ターゲット段階のGAPを埋めるか」「自社の独自性が出せるか」で決める
- 出力：コンテンツ分類ツリー（大カテゴリ×中カテゴリ）＋優先度

### ステップ7：戦略書を1枚にまとめる

- 入力：ステップ1〜6の全出力
- 処理：以下のテンプレートで戦略書を生成する
- 出力：SNSコンテンツ設計書

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 業態がBtoC | 「共感」「親しみやすさ」を発信コアに | BtoC顧客の購買決定要因 |
| 業態がBtoB（リード獲得目的） | 個人名アカウント＋有益コンテンツ継続発信 | 企業アカウントは採用向き、リードはリファラから来る |
| 業態がBtoB（採用目的） | 企業アカウントで社内文化・価値観発信 | 応募者の安心感に繋がる |
| 認知強化したい | TikTok＋Instagram（リール）優先 | 拡散性の高い媒体 |
| 興味検討強化したい | YouTube＋Instagram（ハイライト）優先 | 詳細情報提供に強い |
| 購入維持強化したい | Facebook＋LINE優先 | 既存顧客向け施策と相性◎ |
| 4手法のうち何から始めるか | オーガニック投稿から | 最小コストで開始可能・効果見て他手法追加 |
| アカウントテーマがぼやける | 競合4象限マップで空白ゾーンを狙う | 差別化が明確になる |
| 投稿ネタが枯渇 | ターゲット態度変容3段階×大カテゴリ×中カテゴリで分解 | 体系的にネタが出る |
| 段階間のGAP（情報不足）がある | そこを埋めるコンテンツを優先 | ターゲットの態度変容を促進 |
| 媒体併用したい | 媒体ごとにアカウントテーマ表を別々に作る | 媒体特性が違うため使い回しNG |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 媒体ごとに同じコンテンツを使い回す | 媒体ごとにアカウントテーマ表を別々に作る | 媒体特性（ユーザー層・アルゴリズム）が違うため |
| BtoBで企業アカウントから「共感」発信 | BtoBは「ターゲット価値」発信、共感はBtoC | 業態によって購買決定要因が違う |
| 4手法を全部同時に開始 | オーガニック投稿から始める | 最小コストで効果検証可能 |
| アカウントテーマを言語化せずに投稿開始 | 5項目（特徴／テーマ／訴求点／周期／ハッシュタグ）を埋めてから開始 | 散漫運用になる典型 |
| 競合分析なしで4象限を決める | 競合をプロットしてから空白ゾーンを狙う | 差別化できない |
| ターゲット段階を1区分でしか考えない | COLD／COLD α／MIDの3段階で必要情報を分ける | 態度変容のGAPが見えない |
| 投稿ネタを思いつきで決める | 大カテゴリ→中カテゴリで体系分解してから優先度付け | 一貫性が出ない |
| 「フォロワー数」だけを成果指標にする | ターゲットとの関係性構築・コンバージョンも見る | フォロワー多くても売上に繋がらない |

## 連携スキル

- `sns-strategy-overview`：媒体選定・KGI/KPI設計（このスキルの前段）
- `instagram-account-design`：Instagramを採用した場合の機能・企画設計
- `tiktok-account-design` / `youtube-account-design`：他媒体の機能・企画設計
- `customer-journey`：ステップ5（ターゲット態度変容）の精緻化
- `persona-design`：主要ターゲット設定の詳細化
- `funnel-design`：ファネル設計の精緻化（COLD/COLD α/MIDの3段階を5段階以上に拡張）

## 5状態出力契約

このスキルは外部API・サブプロセス実行・外部設定依存のいずれも持たないため、CLAUDE.md「スキル出力の5状態契約」の対象判定基準には該当しない。
