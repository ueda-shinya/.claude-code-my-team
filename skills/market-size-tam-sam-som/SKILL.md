# 市場規模予測スキル（TAM/SAM/SOM）

ビジネスや投資判断で必須の3階層市場規模指標を、計算式・前提条件・出典付きで定量化する。市場の全体像（TAM）から実際にアプローチ可能な範囲（SAM）、最終的に獲得できる市場（SOM）まで、一貫したロジックで導出する。

## 適用条件

- このスキルを使う場面：
  - 投資家・銀行向けピッチ資料の市場規模章
  - 新規事業の収益機会評価
  - 売上目標の根拠ロジック設計
  - 中長期計画の市場規模根拠
- このスキルを使わない場面：
  - 包括的な市場・競合調査 → `market-competitor-research`
  - フィット段階の診断 → `pmf-journey`
  - 市場シェアKPI設定（イノベーター理論視点） → `innovator-theory`

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 起動フレーズ | 必須 | 上記トリガー語 | 「TAM SAM SOM出して」 |
| 商材 | 必須 | 製品・サービス名 | 「ビジネススクール」 |
| 地理範囲 | 必須 | どこの市場か | 「日本国内」「東京23区」「全世界」 |
| ターゲット顧客 | 必須 | 対象顧客層 | 「中小企業経営者」「20-40代ビジネスパーソン」 |
| 既知の市場データ | 任意 | 公開統計・有料レポート等 | 「ビジネススクール市場 5,000億円（矢野経済）」 |
| 仮想シェア | 任意 | SAMからSOM算出時のシェア仮定 | 「10%」（不明なら本スキルで仮置き） |

## 実行ステップ

### ステップ1：市場の定義（地理／顧客／時間軸）

- 入力：商材、地理範囲、ターゲット顧客
- 処理：
  1. 地理範囲を確定する（国／都道府県／世界）
  2. 顧客層を絞り込む（年齢／業種／規模／所得層）
  3. 時間軸を設定する（年間市場 or 累計市場）
  4. 通貨単位を統一する（円／ドル）
- 出力：市場定義文（1〜2文）

### ステップ2：TAM の計算（獲得できる可能性のある総市場）

- 入力：市場定義
- 処理：
  1. 公的統計・調査レポートから関連市場規模を取得する
  2. 計算式：**TAM = 潜在顧客数 × 顧客一人当たり平均収益**
  3. 出典を明記する（電通／IDC／矢野経済研究所／総務省等）
- 出力：TAM 金額 + 計算根拠 + 出典

### ステップ3：SAM の計算（実際にサービス提供できる市場）

- 入力：TAM、地理的・規制的・製品適合性の制約
- 処理：
  1. TAM から自社が**実際にアプローチ可能な範囲**に絞り込む
     - 地理制約（海外展開しないなら国内のみ）
     - 規制制約（資格・許認可の壁）
     - 製品適合性（B2B限定／法人サイズ限定）
  2. ターゲット顧客の**人口・社数 × 一人/社あたり投資額**で算出
  3. 計算式：**SAM = ターゲット顧客数 × 想定投資額**
- 出力：SAM 金額 + 計算根拠 + 絞り込みの前提条件

### ステップ4：SOM の計算（実際に獲得できる市場シェア）

- 入力：SAM、競合状況、自社のマーケ・営業力
- 処理：
  1. SAM のうち、競合と自社の力関係から**実際に取れる%**を見積もる
  2. 計算式：**SOM = SAM × 獲得可能な市場シェア**
  3. 仮定の市場シェアは以下を参考に決める：
     - 大手競合がいる成熟市場：1〜5%
     - 中堅競合がいる成長市場：5〜15%
     - 新興市場で先行者あり：10〜25%
     - 新興市場で先行者なし（要 `competitive-absence-audit` 検証）：20〜30%
  4. 競合不在主張が出たら → `competitive-absence-audit` 強制起動（CLAUDE.md「Competitive Absence Audit Rule」遵守）
- 出力：SOM 金額 + シェア仮定 + 競合状況考慮の前提

### ステップ5：3階層を可視化する

3つの円が入れ子になる図（TAM ⊃ SAM ⊃ SOM）と、3階層テーブルを並べる。

| 階層 | 定義 | 一言で言うと | 金額 | 計算方法 |
|---|---|---|---|---|
| **TAM** | Total Addressable Market | 獲得できる可能性のある市場 | 約〇〇億円 | 潜在顧客数 × 平均収益 |
| **SAM** | Serviceable Available Market | 実際にサービス提供できる市場 | 約〇〇億円 | ターゲット顧客数 × 想定投資額 |
| **SOM** | Serviceable Obtainable Market | 実際に獲得できる市場規模 | 約〇〇億円 | SAM × 獲得可能シェア（〇%） |

### ステップ6：よくある問題のチェック

以下4点を必ず確認する：

| カテゴリ | チェック内容 |
|---|---|
| **データ信頼性と偏り** | 古い・不正確データ／一次情報と二次情報の不整合がないか？ |
| **外部環境と市場変化** | 景気・規制・技術革新による構造変化を織り込んでいるか？ |
| **予測手法の限界・過大過小評価** | 楽観/悲観バイアスがかかっていないか？過去延長で構造変化を見落としていないか？ |
| **ブラックボックス化** | 計算式と前提が明確で、第三者が再現できるか？ |

### ステップ7：戦略への接続

- 入力：算出した TAM/SAM/SOM
- 処理：
  1. **投資家説明用**：3階層と成長戦略を1ページにまとめる
  2. **売上目標**：SOM を年次に分解して KPI 設計（→ `KGI-KPI-KAI` 関連スキル誘導）
  3. **マーケ戦略**：SAM を獲得対象として STP 設計（→ `stp-analysis` 誘導）
  4. **PMF 段階**：SOM 規模を `pmf-journey` のScale Fit指標と接続
- 出力：戦略へのつなぎ込み案内

### ステップ8：完了報告

ステップ1〜7を以下フォーマットで一括出力する。

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 公的統計が見つからない | 民間調査会社の有料レポート購入を提案／ボトムアップで再計算 | TAM根拠は必須 |
| シェア仮定が無条件に20%以上 | `competitive-absence-audit` 強制起動で楽観バイアス排除 | 過大評価リスク |
| 地理範囲が「全世界」 | 第1版は本拠国のみで算出し、海外展開を別途SAM追加 | スコープが大きすぎると精度低下 |
| TAM = SAM = SOM になる | 絞り込みロジックの再考（地理／規制／顧客層のいずれかで必ず差を作る） | 3階層の意味がなくなる |
| 1社あたり投資額の根拠が不明 | アンケート・公開事例・既存LTV から算出。それでも不明なら2-3パターンの感度分析を出力 | 前提が崩れると全体が崩れる |
| 出典が不明な数字 | リク（fact-checker）に検証依頼を案内 | CLAUDE.md「Deliverable Quality Gate」遵守 |

## 3階層市場規模

| 階層 | 一言で言うと | 金額 | 計算方法 |
|---|---|---|---|
| **TAM** | 獲得できる可能性のある市場 | 約 8,000億円 | 日本のビジネススクール市場5,000億円 + オンライン教育市場3,000億円（出典：矢野経済研究所） |
| **SAM** | 実際にサービス提供できる市場 | 約 4,000億円 | 中小企業経営者+20-40代ビジネスパーソン300万社 × 年間5万円投資 |
| **SOM** | 実際に獲得できる市場規模 | 約 400億円 | SAM × 10%（競争環境・顧客ファネル考慮） |

## 計算根拠

### TAM 約 8,000億円
- 日本国内の企業向け研修・ビジネススクール市場：約 5,000億円
- 日本国内のオンライン教育市場（ビジネス関連）：約 3,000億円
- **出典**：矢野経済研究所「2024年人材開発市場レポート」等

### SAM 約 4,000億円
- ターゲット：中小企業経営者／20-40代ビジネスパーソン
- 日本の中小企業（年商1億円以下）約 300万社
- 1社あたり年間5万円投資想定
- **計算**：300万社 × 5万円 = 1,500億円 + 個人層分 → 約4,000億円規模

### SOM 約 400億円
- SAM の 10%（競争環境・顧客ファネル考慮）
- **シェア仮定理由**：
  - 大手競合（〇〇／△△）が存在
  - オンライン教育の拡大トレンド追い風
  - 有料サービス抵抗感のある層が一定数

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| TAM だけ巨大に出して終わる | 必ず SAM/SOM まで3階層出す | TAM だけは投資家評価できない |
| 「日本人口1.2億人 × 1万円」など雑なTAM | 公的統計か民間レポートで根拠固める | 出典なしは無効 |
| シェア仮定を根拠なく30%設定 | 競合状況×市場成熟度から1〜25%で根拠付け | 過大評価リスク |
| 「競合がいないからシェア50%取れる」 | `competitive-absence-audit` 強制起動 | 楽観バイアス |
| 通貨単位がバラバラ（円とドル混在） | 統一する | 比較不能 |
| 円グラフ・棒グラフで数字だけ出して計算過程を隠す | 計算式を必ず明記 | 再現性なし |
| 「1社あたり投資額」を当てずっぽうで決める | アンケート・事例・LTVから根拠付け | 前提崩壊 |

## 連携スキル

- **前段／併用**：`market-competitor-research`（市場・競合調査全般）、`pest-analysis` `five-forces-analysis`（市場環境）
- **同時起動**：`competitive-absence-audit`（競合不在主張時）
- **後段**：`pmf-journey`（PMF段階診断）、`innovator-theory`（市場シェアKPI、キャズム超え）、`stp-analysis`（ターゲット精緻化）、`medium-term-business-plan`（中長期計画）
