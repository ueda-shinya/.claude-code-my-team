# adsense-monetization

Google AdSenseでサイト・ブログを収益化するための設計スキル。出稿側広告スキル（`ad-mix-design`等）と立場が異なり、サイト運営者（パブリッシャー）の視点で構築する。

## 適用条件

- このスキルを使う場面：
  - ブログ／メディアサイト／情報サイトの収益化を AdSense で行う場面
  - 既存サイトの AdSense 収益が伸び悩んでいるとき
  - サイトリニューアル時の広告配置設計
- このスキルを使わない場面：
  - 広告を「出稿」する側 → `ad-mix-design`／`meta-ad-campaign-design`／リスティング広告スキル等
  - アフィリエイト広告 → `affiliate-ad-design`（別スキル）
  - SEO本体 → `seo-audit`／`/blog-post`
  - LPやEC サイトのCV最大化（広告枠と相反する目的）

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| サイトURL | 必須 | 対象サイト | 「example.com/blog」 |
| サイトジャンル | 必須 | コンテンツテーマ | 「IT・ビジネス情報」 |
| 月間PV/UU | 必須 | 現状トラフィック | 「PV 5万／UU 3万」 |
| AdSense承認状況 | 必須 | 未申請／審査中／承認済み | 「承認済み」 |
| 現状のAdSense収益 | 任意 | 月額収益 | 「月3万円」 |
| 現状の広告配置 | 任意 | サイドバー・記事内など | 「記事下部のみ」 |
| 主要流入KW | 任意 | アクセス上位KW | 「中小企業 IT 導入」 |

## 実行ステップ

### ステップ1：AdSense導入前提を確認する

- 入力：サイトURL、AdSense承認状況、サイトジャンル
- 処理：
  - AdSense承認の前提条件を満たすか確認：
    - オリジナルコンテンツが20-30記事以上あるか
    - プライバシーポリシー・お問い合わせフォームが設置済みか
    - Googleポリシー違反コンテンツ（成人向け／違法／著作権侵害／誤クリック誘導等）がないか
    - 独自ドメインで運用しているか
  - サイトジャンルがAdSenseで収益化に向くか診断：
    - 向く：金融／保険／法律／不動産／転職／教育／IT（CPC高）
    - 中庸：旅行／グルメ／生活情報
    - 向きにくい：個人日記／写真集／極端なニッチ
- 判定：未承認かつ前提未達 → AdSense以外の収益化手段（アフィリエイト等）を提案
- 出力：AdSense適用可否＋必要な事前対応リスト

### ステップ2：訪問者数を増やす施策を設計する（コツ①）

- 入力：現状PV/UU、主要流入KW、サイトジャンル
- 処理：以下2軸で施策を立てる
  - **SEO対策**：
    - 検索ボリュームのあるキーワードを `/blog-post` または `seo-audit` でリサーチ
    - 記事タイトル・見出し・メタディスクリプションにKW最適化
    - 内部リンク強化／構造化データ実装
    - 月N本の新規記事ペースを決める（最低週1本＝月4本）
  - **SNS活用**：
    - 記事公開時にX/Threads/Instagram/Facebookで告知
    - 過去記事の定期的な再シェア
    - SNSとブログの動線設計
- 目標値の設定：
  - PV月20%増を最低ライン
  - UU月15%増を目標
- 出力：流入増施策リスト＋月次目標値

### ステップ3：高単価キーワードを狙う（コツ②）

- 入力：サイトジャンル、現状記事リスト
- 処理：
  - **Googleキーワードプランナー**で関連キーワードのCPC相場を調査
  - 高単価キーワードカテゴリ例：
    - 金融系：「カードローン」「住宅ローン」「FX」（CPC 500-2,000円）
    - 保険系：「自動車保険 比較」「生命保険 おすすめ」（CPC 300-1,500円）
    - 法律系：「弁護士 相談」「労働問題」（CPC 1,000-3,000円）
    - 転職系：「転職エージェント」「年収アップ」（CPC 200-1,000円）
    - 不動産系：「マンション 査定」「賃貸 探し方」（CPC 200-800円）
  - 既存サイトジャンルと親和性のある高単価KWを5-10個ピックアップ
  - 該当KWで上位表示が現実的か競合難易度（KD）を確認
- 記事戦略：
  - 高単価KWに紐づく記事を月2-4本制作
  - 既存記事の改善（タイトル変更・内容追記）も検討
- 出力：高単価KW候補リスト＋記事制作計画

### ステップ4：広告配置を最適化する（コツ③）

- 入力：サイトテンプレート、現状の広告配置
- 処理：
  - **効果的な広告配置3箇所**を確認：
    - 記事上部（First view直下）：CTR最高、ファーストビュー外に配置
    - 記事内（H2見出し直前）：本文との関連度高くCTR次点
    - サイドバー（PCのみ）：常時表示でリーチ最大化
  - **レスポンシブ広告**を必ず使用（モバイル/タブレット/PC自動最適化）
  - 1ページあたりの広告数：3-5枠（過剰掲載回避）
  - 自動広告（Auto ads）vs 手動配置の判断：
    - 初期はAuto adsで開始 → データ取得後に手動配置で最適化
- 配置パターン例：
  - パターンA（記事系メディア）：記事冒頭1＋記事中2＋記事末1＋サイドバー1＝5枠
  - パターンB（情報サイト）：トップ1＋カテゴリ一覧2＋記事中2＝5枠
  - パターンC（ニュース系）：記事上1＋記事中3＝4枠
- 出力：広告配置設計図＋実装手順

### ステップ5：3つの注意点でリスク点検する

- 入力：広告配置設計、サイトコンテンツ
- 処理：以下3チェック項目で点検する
  | # | 注意点 | チェック内容 | リスク回避策 |
  |---|---|---|---|
  | 1 | Googleポリシー遵守 | 誤クリック誘導／禁止コンテンツ／著作権侵害／自己クリックがないか | ポリシー全文確認・社内ガイドライン化 |
  | 2 | 過剰掲載回避 | 広告がコンテンツ面積を圧迫していないか／UX損なっていないか | 1ページ広告3-5枠まで／画面占有率30%以下 |
  | 3 | 収益データ分析 | AdSense管理画面で収益・CTR・CPC・eCPMを追跡しているか | 週次レビュー設定・改善PDCAサイクル化 |
- ポリシー違反の代表例（NG）：
  - 「広告をクリック！」と誘導するテキスト
  - 自分でクリックする（自己クリック）
  - 成人向け／違法薬物／武器販売コンテンツ
  - 著作権侵害（無断転載・無断利用画像）
- アカウント停止リスク回避が最優先（停止すると以降利用不可になる場合あり）
- 出力：リスク点検結果＋対策必要項目

### ステップ6：収益最大化の運用サイクルを設計する

- 入力：ステップ2-5の成果物
- 処理：
  - **週次レビュー項目**：
    - PV/UU推移
    - 広告クリック数・CTR・eCPM・収益額
    - 上位記事の特定
    - 広告配置別パフォーマンス
  - **月次レビュー項目**：
    - 高単価KW記事のランキング推移
    - 広告配置のA/Bテスト結果
    - ジャンル別収益比率
  - **改善アクション**：
    - 低CTR広告枠 → 配置位置変更
    - 低CPC流入 → 高単価KW記事の比率UP
    - PV停滞 → SEO/SNS強化
- 出力：運用カレンダー（週次・月次タスク）

### ステップ7：AdSense運用計画書としてまとめる

- 入力：ステップ1-6の全成果物
- 処理：出力フォーマットに沿って整形する
- 出力：AdSense運用計画書

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 月PV 1万未満 | AdSenseより先にコンテンツ充実とSEO強化を優先 | 訪問者数不足で収益化困難 |
| 月PV 1-5万 | AdSense導入＋手動広告配置で月数千〜数万円 | 中規模での標準ライン |
| 月PV 5万以上 | 高単価KW記事比率UP＋広告枠最適化 | 収益スケールが見込める |
| サイトジャンルが個人日記 | アフィリエイトや有料note等を検討 | AdSense向きでない |
| 高単価KWで競合過多 | ロングテール（「中小企業向け 〇〇 比較」等）に絞る | 上位表示不能なKWは収益化できない |
| Googleポリシー違反疑い | 即座にコンテンツ削除／配置変更 | アカウント停止リスク |
| 1ページ6枠以上の広告 | 4-5枠に削減 | UX劣化／ポリシー違反リスク |
| 広告がファーストビュー圧迫 | スクロール後に配置変更 | UX低下＋Google Page Experience評価低下 |
| 自動広告（Auto ads）導入後3ヶ月 | 手動配置に切替して最適化 | データに基づく最適化が可能になる |
| eCPMが業界平均（300-500円）大幅下回る | 高単価KWへの記事シフト | 流入KWの単価が低すぎる |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 広告を1ページ8枠以上配置 | 3-5枠に絞る | UX劣化＋ポリシー違反リスク |
| 「広告をクリックしてください」と誘導 | 誘導表現は禁止 | アカウント停止確定 |
| 自分でクリックして収益確認 | 絶対にクリックしない（家族/同僚も含む）| 自己クリック検知でアカウント停止 |
| 個人日記ブログでAdSenseに固執 | アフィリエイト等別手段に切替 | ジャンルでCPC上限が決まる |
| 月PV 5,000程度でAdSense本格運用 | 先にPV 1万到達を優先 | 訪問者不足で収益化困難 |
| 自動広告Auto adsを入れっぱなし | 3ヶ月後に手動最適化に切替 | データ取得後の最適化が必要 |
| ファーストビューに広告ベタ貼り | スクロール後にずらす | Page Experience評価低下＋UX劣化 |
| ポリシー違反の灰色領域コンテンツ放置 | 即座に削除/書き換え | 全体停止リスク |
| 高単価KWだけ追いかけてサイトジャンルとズレる | 親和性のあるKWに絞る | 上位表示できないKWは無価値 |
| 収益データを月1回しか見ない | 週次で追跡 | 異常検知が遅れる |

## 5状態出力契約

本スキルは純粋なテンプレート展開のみで外部API・サブプロセス・外部設定依存を持たないため、CLAUDE.md「スキル出力の5状態契約」の対象外。
