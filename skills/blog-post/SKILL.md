# ブログ記事作成ガイド

## 構成ルール
1. タイトル（30文字以内・検索されやすいキーワードを含む）
2. リード文（3〜5行・記事全体の要約）
3. 本文（H2見出し3〜5つ、各800〜1200文字）
4. まとめ（3〜5つの箇条書き）
5. CTA（読者に次のアクションを促す一文）

## 文体ルール
- 一文は80文字以内
- 専門用語には必ず説明を添える
- 具体例・数字・事例を必ず1つ以上含める

## SEOルール
- キーワードはタイトル・リード文・H2に必ず含める
- 内部リンクを最低2つ提案する
