# sns-strategy-overview

SNS運用を「成長投資」と捉え、媒体選定→アカウントタイプ決定→KGI/KPI設計→運用準備→外注/内製化判定を、現状課題と事業ゴールから逆算して設計する。

## 適用条件

- このスキルを使う場面：
  - SNS運用をこれから始める／立て直したいときの戦略設計
  - 複数媒体を運用している企業で媒体ポートフォリオを再設計したいとき
  - 認知拡大／採用／集客／ファン化のどの目的でSNSを使うか整理したいとき
  - SNSのKGI/KPIが曖昧で、事業目標と紐づいていないとき
- このスキルを使わない場面：
  - 単一媒体の運用詳細（Instagram単体の企画／TikTokの台本作成）→ 媒体特化スキルへ
  - 投稿1本のクリエイティブ制作 → コピーライター／デザイナーへ
  - 広告運用の具体設計 → ad-mix-design / meta-ad-campaign-design 等へ

## 入力

| 項目 | 必須/任意 | 説明 | 例 |
|---|---|---|---|
| 事業ゴール | 必須 | SNSで実現したい事業上の成果 | 売上向上／採用強化／認知拡大／ファン化 |
| 主要ターゲット | 必須 | 届けたい顧客像（属性＋心理） | 30代女性・育児中・東京 |
| 商材タイプ | 必須 | BtoC／BtoB／採用 のいずれか | BtoC（D2Cコスメ） |
| 既存運用状況 | 任意 | 現在運用している媒体・成果 | Instagramのみ・フォロワー1,500人 |
| リソース制約 | 任意 | 人員・予算・スキル | 専任なし・月予算20万 |
| 運用開始時期 | 任意 | いつから始めるか | 来月から／3ヶ月以内 |

## 実行ステップ

### ステップ1：事業ゴールとSNS適性をマッチングする

- 入力：事業ゴール、商材タイプ
- 処理：
  1. SNS運用で「出来ること」「出来ないこと」を確認し、事業ゴールが達成可能かを判定する
     - 出来ること：認知拡大／ブランド構築／行動喚起／ファン化
     - 出来ないこと：①短期間での直接的な売上向上 ②全世代へのリーチ ③商品・サービス品質の補完
  2. 事業ゴールが「出来ないこと」に該当する場合 → SNS以外の施策（広告／PR／商品改善）を優先するよう警告する
  3. 該当しない場合 → ステップ2へ進む
- 出力：SNS適性判定結果（適性あり／部分適性／適性なし）

### ステップ2：5媒体比較表で候補を絞り込む

- 入力：主要ターゲット、商材タイプ、事業ゴール
- 処理：
  1. 以下の5媒体特性表でターゲット適合度を評価する：

  | 媒体 | 国内月間アクティブ | ボリューム年代 | 主特性 | 強い領域 |
  |---|---|---|---|---|
  | Instagram | 6,600万人 | 20〜40代 | ビジュアル特化／ハッシュタグ文化 | 世界観・ファンマーケ・購買導線 |
  | TikTok | 3,300万人 | 10〜20代 | 縦型ショート動画／フォロワー数非依存 | 新規認知・拡散 |
  | YouTube | 7,120万人 | 20〜60代 | 長尺動画／検索可 | 詳細説明・比較検討・リピート |
  | X | 6,700万人 | 10〜40代 | 短文・拡散・リアルタイム | トレンド・話題性・限定商品 |
  | Facebook | 2,600万人 | 30〜50代 | 実名制・ビジネス層 | BtoB・高価格帯商材 |

  2. カスタマージャーニー適性表で、自社が強化したいフェーズに最適な媒体を選ぶ：

  | 媒体 | 認知 | 興味 | 比較 | 行動 | リピート |
  |---|---|---|---|---|---|
  | Instagram | ◯ | ◎ | ◎ | ◯ | ◯ |
  | TikTok | ◎ | ◎ | △ | △ | × |
  | YouTube | △ | ◯ | ◎ | ◎ | ◎ |
  | X | ◎ | ◎ | × | × | × |
  | Facebook | △ | ◯ | ◎ | ◎ | ◎ |

  3. ターゲット年代×ジャーニー強化フェーズの両軸で1〜3媒体に絞り込む
  4. リソース制約が「専任なし／月予算少」の場合 → 1媒体に絞る
- 出力：選定媒体リスト（最大3媒体・主従を明示）

### ステップ3：アカウントタイプを決定する

- 入力：選定媒体、事業ゴール
- 処理：
  1. 以下の3アカウントタイプから事業ゴールに最適なものを選ぶ：

  | アカウントタイプ | KGI候補 | 主要KPI | 推奨媒体 |
  |---|---|---|---|
  | オウンドメディア型 | ブランド認知度／自社サイト流入／CVR向上 | フォロワー増加率／投稿エンゲージメント率／サイト流入数 | Instagram、YouTube |
  | 認知獲得型 | 新規顧客獲得／商品認知拡大／口コミ・バズ | リーチ数／インプレッション／動画再生完了率／シェア数 | TikTok、X、Instagram(リール) |
  | リピーター向け型 | 顧客満足度／リピート率／顧客単価 | 既存顧客エンゲージメント率／クーポン利用率／LTV伸び率 | Instagram、LINE、YouTube |

  2. 複数媒体運用の場合 → 媒体ごとに役割（メイン／サブ）を割り当てる
- 出力：アカウントタイプ決定書（媒体×アカウントタイプ×役割）

### ステップ4：KGI／KPIを段階的に設計する

- 入力：アカウントタイプ決定書
- 処理：
  1. KGI（最終目標）を事業数値に紐付けて1つだけ定める（例：「年間売上10%向上」「採用応募数30%増加」）
  2. KGI達成のための段階的KPIを「フォロワー→エンゲージメント→サイト流入→成約」の順で設計する
  3. 各KPIに以下4要素を必ず付与する：
     - **数値目標**（例：3ヶ月で1,000人増加）
     - **達成期限**（月次／四半期）
     - **測定方法**（インサイト／GA4／自社CRM等）
     - **検証頻度**（月次or四半期）
  4. KPIが定量測定不能な場合 → 代替の定量指標に置き換える（「ブランド好感度」→「コメント称賛文の月間件数」等）
- 出力：KGI/KPIツリー（KGI1個＋段階的KPI3〜5個）

### ステップ5：運用開始準備の4ステップを設計する

- 入力：選定媒体、アカウントタイプ、KGI/KPIツリー
- 処理：
  1. **競合・市場リサーチ**：競合3〜5アカウントを選定し、フォロワー数・投稿頻度・エンゲージメント率・伸びている投稿フォーマットを記録する
  2. **ペルソナ・ジャーニー設計**：理想顧客像を1人定義し、認知→興味→行動→ファン化の各段階で必要なコンテンツを設計する
  3. **投稿企画立案**：商品紹介／お役立ち情報／ユーザー事例の3カテゴリでバランスをとり、月単位の企画リストを作成する
  4. **コンテンツカレンダー作成**：日時／内容／媒体／担当者を明記した月次カレンダーをスプレッドシートで作成する
- 出力：運用開始準備書（4セクション）

### ステップ6：外注／内製化の役割分担を決定する

- 入力：運用開始準備書、リソース制約
- 処理：
  1. 以下の判定表で、業務ごとに外注／内製を振り分ける：

  | 業務 | 外注推奨 | 内製推奨 | 判定基準 |
  |---|---|---|---|
  | 戦略策定 |  | ◎ | ブランドの一貫性／長期的ノウハウ蓄積 |
  | コンテンツ企画 |  | ◎ | 自社の世界観・顧客理解 |
  | KPI設定・評価 |  | ◎ | 事業数値との紐付けが必要 |
  | 投稿管理 |  | ◎ | リアルタイム対応 |
  | ユーザー対応（DM/コメント） |  | ◎ | ブランド口調統一 |
  | デザイン制作 | ◎ |  | 専門スキル必要 |
  | 動画編集 | ◎ |  | 時間とスキル要 |
  | 写真撮影 | ◎ |  | 機材とスキル要 |
  | キャンペーン運用 | ◎ |  | 短期集中・専門知識 |
  | 広告配信 | ◎ |  | 運用スキル要 |
  | 高度なAI活用 | ◎ |  | 最新ツール知見要 |

  2. 専任なしの最小リソース立ち上げの場合は、以下4ステップを推奨する：
     - ❶担当者選定（SNSに興味ある社内1〜2名）
     - ❷基本ルール明文化（投稿頻度／企画方向性／コメント対応ルール）
     - ❸外注先選定（撮影／デザイン／動画編集）
     - ❹段階的内製化（運用効果が出た領域から内部に巻き取る）
- 出力：外注／内製化の役割分担表

### ステップ7：戦略書を1枚にまとめる

- 入力：ステップ1〜6の全出力
- 処理：以下のテンプレートで1ページの戦略書を生成する
- 出力：SNS戦略書（テンプレートは「出力フォーマット」参照）

## 判断基準

| 状況 | 判断 | 理由 |
|---|---|---|
| 事業ゴールが「短期売上向上」 | SNS以外の施策（広告／PR）を優先 | SNSは即効性のある施策ではない |
| ターゲットが10〜20代 | TikTok優先、Instagramリール併用 | ボリュームゾーンに合致 |
| ターゲットが30〜50代BtoB | YouTube／Facebook優先、Instagram補助 | ビジネス層・高価格帯と相性 |
| リソース「専任なし・月予算20万以下」 | 1媒体に絞る | 複数媒体は品質低下の典型パターン |
| 認知獲得を最優先 | TikTok or X、Instagramリール | 拡散性が高い媒体 |
| ファン化／LTV向上が最優先 | Instagram、YouTube、LINE | リピート適性◎ |
| 検討段階の比較情報を届けたい | YouTube、Facebook | 長尺・詳細情報に強い |
| 複数媒体運用したい | メイン1＋サブ1〜2まで | 4媒体以上は失敗パターン |
| KPIが定量測定不能 | 代替の定量指標に置き換え | 検証不能なKPIは運用が形骸化する |
| エンゲージメント率高いフォロワー500人 vs 無関心5,000人 | 質を優先 | 売上に繋がるのは適合度の高いフォロワー |

## よくある失敗パターン

| NG | OK | 理由 |
|---|---|---|
| 「他社もやっているから」でSNS運用を始める | 事業ゴールに紐づけて媒体を選定 | 目的なき運用は数値に繋がらない |
| 短期的なバズだけを追求 | LTV重視の長期投資視点 | バズはフォロワー獲得には繋がるが売上に繋がらない |
| 「投資対効果」ではなく「コスト」として捉える | 「成長投資」として中長期で評価 | 短期評価で予算カットすると蓄積資産を失う |
| フォロワー数=売上と考える | エンゲージメント率の高いフォロワーの質を優先 | 無関心フォロワー5,000より関心のある500人の方が売上に繋がる |
| 4媒体以上を同時に立ち上げる | メイン1＋サブ最大2の合計3媒体まで | リソース分散による全媒体の品質低下 |
| KPIを定量測定不能なまま運用 | 必ず数値・期限・測定方法・検証頻度の4要素を付ける | 検証不能なKPIは運用が形骸化する |
| 戦略策定を外注に丸投げ | 戦略策定・KPI設計は内製、制作は外注 | 戦略は事業数値との紐付けが必要なため内部知見が必須 |
| KGIなしでKPIだけ設定 | KGIを1つ定めてからKPIを段階分解 | KPI同士の優先順位がつけられず運用が散らかる |

## 連携スキル

- `customer-journey`：ジャーニー設計の詳細フローを使う場合に併用
- `kgi-kpi-kai-design`：KPI設計をさらに精緻化したい場合
- `marketing-sales-workflow`：SNSとマーケ・営業フロー全体を接続したい場合
- `instagram-account-design`：Instagramを採用した場合の詳細設計
- `meta-ad-campaign-design` / `listing-ad-campaign-design`：広告併用時
