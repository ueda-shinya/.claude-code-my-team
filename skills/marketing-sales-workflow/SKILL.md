# マーケティング・営業フロー策定スキル

顧客が認知→購入→継続利用に至るまでの一連のプロセスを体系化し、マーケティングと営業の役割分担・KPI・タッチポイントを統合する。MQL/SQLの共通指標で部門連携を強化。

## 適用条件

- マーケと営業が分断してリード引継ぎが滞っている
- リード発生〜成約までのKPIが部門ごとにバラバラ
- 顧客接点の重複・抜け漏れを整理したい

## 入力

| 項目 | 必須 | 説明 |
|---|---|---|
| 起動フレーズ | 必須 | 上記トリガー |
| 商材 | 必須 | BtoB/BtoC・SaaS/物販等 |
| 既存組織体制 | 任意 | マーケ・IS・FS・CS の有無 |

## 実行ステップ

### ステップ1: 顧客の購買プロセスを理解する
**5段階ファネル**：認知 → 興味 → 検討 → 購入 → 継続・リピート

各段階で顧客の行動・ニーズを分析する。

### ステップ2: マーケティングと営業の役割分担を明確化

| 部門 | 役割 |
|---|---|
| **マーケティング** | 1. 見込み客（リード）獲得 2. リード育成→MQL創出 |
| **インサイドセールス（IS）** | MQLをSQLに転換する初期接触 |
| **フィールドセールス（FS）** | SQLとの商談・クロージング |
| **カスタマーサクセス（CS）** | 顧客との関係構築・リピート促進 |

※ 企業によってマーケと営業が同一 or 異なる場合がある。

### ステップ3: 各段階のタッチポイント設計

| 段階 | マーケのタッチポイント | 営業のタッチポイント |
|---|---|---|
| 認知 | 広告 | - |
| 興味 | ブログ | - |
| 育成 | メールマガジン | - |
| 検討 | - | 初回アプローチ |
| 購入 | - | 商談 |
| リピート | - | フォローアップ |

### ステップ4: KPI設定

| 段階 | KPI |
|---|---|
| **認知** | 広告のインプレッション数、ウェブサイト訪問者数 |
| **リード獲得** | メール登録数、資料ダウンロード数 |
| **商談率** | MQL→SQLの転換率 |
| **契約率** | SQL→契約数 |

### ステップ5: ツールとデータ活用

| 用途 | ツール例 |
|---|---|
| マーケ MA | HubSpot / Marketo |
| 営業 CRM | Salesforce |
| 部門間コミュニケーション | Slack / Zoom |

### ステップ6: フロー全体図の出力

**Earned / Owned / Paid × ファネル5段階のマトリクス**：

| | 認知 | 興味 | 検討期 | 問合せ | 商談 |
|---|---|---|---|---|---|
| **役割** | ニーズの顕在化 | ニーズ顕在層のリード転換 | 問合せ数・資料請求の確度向上 | 確度向上・有効リード精査 | 商談数の向上 |
| **Earned**（ブログ・SNS等） | 外部イベント/講演 | 協業・代理販売／コンサル紹介 | | | |
| **Owned**（自社メディア） | 内部セミナー／WEBサイト／SEO | 事例シェア会／資料DL | 事例・ホワイトペーパー | アンケート／資料DLフォーム／問合せフォーム | 有効リード見極め・体制構築→商談 |
| **Paid**（マス4媒体・Web広告） | 比較メディア・メルマガ | リスティング広告／Fbリード獲得広告 | リターゲティング広告 | | |

### ステップ7: プロセス×組織×KPIの整列マップ

| プロセス | リード創出 | 育成/選別 | 商談 | 成約 | 継続 |
|---|---|---|---|---|---|
| **組織** | マーケティング | IS | FS | FS | CS |
| **KPI** | リード数 | MQL/SQL | 商談数 | 成約数 | MRR |

### ステップ8: ボトルネック特定と改善

- フロー進捗を確認し、停滞部分を特定
- 試験運用→効果測定→データに基づいて調整・展開

## 判断基準

| 状況 | 判断 |
|---|---|
| マーケと営業が分断 | 共通指標（MQL/SQL）を導入 |
| 各段階のKPI曖昧 | 各段階に必ず1つ以上の定量KPI設定 |
| 顧客の購買プロセスが不明 | `customer-journey` で先行設計 |
| BtoCで MQL/SQL 概念が合わない | リード→ホット→購入の3段階に簡略化 |
