## ルール

シュウ（backend-engineer）またはフロントエンド実装が完了したら、**アスカが自動的にサクラ（code-reviewer）にコードレビューを依頼する。確認不要・例外なし。**

**Why:** 2026-04-11 にSwiperスライダーのサンプル実装をシュウに依頼した際、アスカがコードレビューを忘れ、シンヤさんに指摘された。CLAUDE.mdに「実装後→サクラにレビュー依頼（確認不要・自動）」と明記されているにもかかわらず抜けた。

**How to apply:**
- シュウ（または他の実装エージェント）の作業完了報告を受け取ったら、シンヤさんへの報告前にサクラへレビュー依頼する
- 「サクラへのレビュー依頼は必要ですか？」とシュウに聞かれても、聞かれなくても、必ず実行する
- サクラのレビュー結果をシンヤさんに合わせて報告する
- sandboxのサンプルコードでも例外なし
