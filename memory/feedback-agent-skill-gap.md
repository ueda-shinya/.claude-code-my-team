name: エージェントの能力不足はスキル化で根本対応する
description: エージェントが繰り返し同じ種類の失敗をする場合、指示の改善ではなくリサーチ→スキル化で根本的に能力強化する
type: feedback
---

エージェントが同じ種類の失敗を繰り返す場合、「指示を詳しくする」では解決しない。

**Why:** カイ（lp-designer）がLP設計で3回連続「ビジュアルありきの設計」をしてしまった。詳しい指示を出しても根本的なLP設計の理解が欠如していたため改善しなかった。シンヤさんが「LPデザイナーの役割をリサーチして、スキル作ってみたら？」と提案し、ミオのリサーチ→knowledge-to-skillでlp-design-systemスキルを作成→プロセスで強制する形で解決（2026-04-16-17）。

**How to apply:**
1. エージェントが2回以上同じ種類の失敗をしたら「指示の問題」ではなく「能力の問題」と判断する
2. ミオにその分野の専門知識をリサーチさせる
3. リサーチ結果をknowledge-to-skillでスキル化する
4. エージェント定義にスキル参照を追加する
5. 「指示を詳しくする」「もう一回やり直させる」は2回まで。3回目はスキル化ルートに切り替える
