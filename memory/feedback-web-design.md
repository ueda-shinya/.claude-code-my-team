## フォントサイズの最小値ルール

通常テキスト（本文・説明文・リスト等）は **最小16px** を守ること。
補足・注釈・出典・キャプション等は **最小10px** を守ること。

**Why:** clamp()でレスポンシブ対応する際、最小値を小さくしすぎると小さい画面や縮小表示で可読性が著しく低下するため。稲田さん案件のスライドで指摘を受けた（2026-04-07）。

**How to apply:**
- clamp(最小px, vw, 最大px) を使う場合、最小値は必ず上記ルールを守る
- Shuへの実装依頼時にも明示すること
- スライド・LP・HTMLレポート等すべてのWeb成果物に適用
