# 02. LP・ヒーローセクション

## LP 全体プロンプト（英語）

```
Using the messaging document I've uploaded, create a high-fidelity landing
page for [Product Name]. Target audience: [persona].
The page needs:
- Hero section with headline and subhead leading with [primary benefit]
- Three value prop sections with icons
- Competitor comparison table
- Customer quote section
- Demo request CTA
Tone: [tone descriptors].
```

## LP 全体プロンプト（日本語）

```
アップロードしたメッセージング文書をもとに、[製品名]のハイフィデリティな
ランディングページを作成してください。
対象読者: [ペルソナ]
必要なセクション:
- [主要ベネフィット]を前面に出したヒーロー（キャッチコピー＋サブコピー）
- アイコン付きバリュープロポジション × 3
- 競合比較表
- 顧客の声（引用）
- デモ申込CTA
トーン: [トーン形容詞群]
```

## ヒーローセクション特化（高品質実例）

```
Generate the HTML and CSS for a hero section for a [industry] company.
Mood: [mood adjectives, e.g., optimistic, modern, trustworthy].
Background: [color hex]. Accent and CTAs: [accent hex].
Layout: asymmetrical — large compelling headline on the left, SVG placeholder
on the right.
Font: [display font] for headlines, [body font] for body.
Bold H1, lighter subheading, prominent CTA with subtle hover effect.
```

## ヒーローバリエーション要求

```
Create 3 hero section variants for [product type].

Variant A: benefit-led headline, minimal layout, white background
Variant B: bold dark background, large statement type, single CTA
Variant C: split layout — copy left, product screenshot right

For all: font [display font] for headline, keep CTAs outcome-specific
(e.g., "See 40% faster onboarding" not "Get started")
```

## ブランドサイト（投資家向け・高級感）

```
Create a high-fidelity landing page designed to raise $[X] from [target investors]
for [company name].
Reference aesthetic: SpaceX-level gravitas meets Stripe's layout discipline.
Tone: trustworthy, bold, slightly editorial.
Sections: hero with founder quote, problem/solution, product demo placeholder,
team bios, investor logos, contact CTA.
Typography: Fraunces for display headings, Source Sans 3 for body.
Color: black canvas, single warm accent (#c96442 terracotta).
No stock photography — use geometric SVG illustrations instead.
```

## 単一セクション特化

### 価格表
```
Design a pricing page. 3 tiers: Starter / Growth / Enterprise.
Annual/monthly toggle. Highlight the Growth plan.
Include: feature comparison table, FAQ below the fold, enterprise CTA.
Dark mode. Font: Satoshi. Accent: electric green (#00C97A).
```

### FAQ アコーディオン
```
Build an FAQ accordion section. 8 questions, 2-column layout on desktop,
single-column on mobile. Smooth expand/collapse animation (300ms ease).
Theme: warm minimal (#f4f3ee background, #c96442 active state).
```
