## 1. 脳の3層モデルとコピー戦略

| 脳のレイヤー | 優先する刺激 | コピー戦術 |
|---|---|---|
| **爬虫類の脳（本能）** | 危機感・即時性 | 「今すぐ逃れなければならない苦痛」を提示。複雑な思考を介させない |
| **中間の脳（感情）** | ストーリー・社会的証明 | 苦痛からの解放後の快楽を描写。ブランドへの親近感を形成 |
| **新しい脳（理性）** | 統計・論理的根拠 | 感情で決めた購入を「正当化」させる材料を与える |

## 2. 9つの心理的エフェクト

| # | エフェクト | 概要 | 適用例 |
|---|---|---|---|
| ① | プラットフォール効果 | 弱みの開示が信頼を高める | 「私が犯した5つの致命的なミスを公開します」 |
| ② | ノベルティ効果 | 変化自体が一時的効果を生む | コピー・クリエイティブを定期刷新してCTR低下を防ぐ |
| ③ | プライミング効果 | 先行情報が後の行動に影響 | CTAの直前に「自信を持って」「成功者の仲間入り」を配置 |
| ④ | フォーカシング効果 | 最も目立つ情報に執着して判断 | USPを最大見出しに置き、全情報をそのレンズで解釈させる |
| ⑤ | BYAF効果 | 「自由選択」明示で承諾率倍増 | 「決めるのはもちろん皆様の自由です」と断る自由を与える |
| ⑥ | 初頭・親近効果 | 最初と最後が最も記憶に残る | 箇条書きの1番目に最強ベネフィット、最後に返金保証 |
| ⑦ | 認知流暢性 | 読みやすい情報は「真実」と錯覚 | 「CTRを最大化」→「クリックされる確率を上げる」 |
| ⑧ | 錯覚の真実効果 | 反復提示された情報を真実と認識 | コアメッセージを全媒体で一貫して繰り返す |
| ⑨ | オープンループ効果 | 未完了のものに脳が執着し続ける | 「たった一つの単語とは何か？その正体を後ほど明かします」 |

## 3. フレームワーク使い分け

| フレームワーク | 最適シーン | 組み込む心理エフェクト |
|---|---|---|
| **AIDA** | 新規獲得・認知拡大 | Attention:オープンループ → Interest:フォーカシング → Desire:ストーリー → Action:BYAF |
| **PASONA** | 悩み解決・緊急性の高い商材 | 問題:爬虫類の脳を刺激 → 解決:未来を視覚化 → 限定:希少性で本能を誘発 |
| **PREP** | メール・解説・信頼構築 | Point:結論先出し → Reason:Because → Point:再結論（親近効果） |

## 4. 3つの壁突破 15テクニック

| 壁 | 読者の心理 | 打開策 |
|---|---|---|
| **読まない壁** | 「自分には関係ない」 | ①権威性 ②パワーワード ③検索意図の網羅 ④PREP法 ⑤視覚的アイキャッチ |
| **信じない壁** | 「嘘くさい」 | ⑥社会的証明 ⑦デメリット開示 ⑧断言・言い切り ⑨Because ⑩全編一貫性 |
| **行動しない壁** | 「後でいい」 | ⑪限定性・希少性 ⑫ベネフィット訴求 ⑬BYAF ⑭ハードル低下 ⑮返金保証 |

### Becauseの力（5つのWHY）
Xerox社実験：Because を添えるだけで承諾率 60% → 94% へ飛躍。

1. **WHY Now?**：限定性・時事性・即時解決の必要性
2. **WHY This?**：他にはない独自の解決策・USP
3. **WHY You?**：圧倒的実績・専門性・共鳴するビジョン
4. **WHY This Price?**：価値に対する圧倒的ROI
5. **WHY Believe?**：社会的証明・第三者評価・デメリット開示
