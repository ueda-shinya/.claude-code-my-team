# プラグイン管理規約

自作プラグインは `~/.claude/wordpress-plugins/` で一元管理する。

## ディレクトリ構成

```
wordpress-plugins/
├── README.md              ← プラグイン一覧（ステータスつき）
├── mu-plugins/            ← MU プラグイン（常時動作・管理画面から無効化不可）
└── <plugin-name>/         ← 通常プラグイン
    ├── <plugin-name>.php  ← メインファイル
    ├── uninstall.php      ← アンインストール処理
    └── requirements.md    ← 要件定義（必須）
```

## ステータス定義

| 記号 | 意味 |
|---|---|
| ✅ 安定 | 本番で使用中・動作確認済み |
| 🔵 制作完了 | コード完成・動作確認未 |
| 🚧 制作中 | 現在開発中 |
| 🔧 整備中 | 不具合あり・修正中 |
| ⚠️ 非推奨 | 旧版・新版あり |

## requirements.md の記載項目

プラグインを新規作成する際は `requirements.md` を必ず添付する。

- **概要**：何をするプラグインか
- **背景・課題**：なぜ作るか、旧構成の問題点
- **機能要件**：機能ごとの仕様（表形式推奨）
- **管理画面要件**：UI仕様
- **セキュリティ要件**：各対策と根拠
- **非機能要件**：対応PHP・WPバージョン等
- **バージョン履歴**：変更の記録

詳細は `~/.claude/projects/.../memory/feedback-dev-workflow.md` を参照。
