# 不動産屋における顧客情報取得施策：Googleフォーム活用の整理

## 概要

来店顧客の情報登録数を増やすため、Googleフォームを活用した3チャネル対応の仕組みを検討。
紙の持ち歩き不要・機会損失の削減を目指す。

---

## 対応チャネルの構成

| チャネル | 対象 | 方法 |
|---|---|---|
| タブレット直接入力 | デジタル操作に慣れたお客様 | 店舗設置タブレットでフォーム入力 |
| QRコード → スマホ | タブレットが不慣れなお客様 | 自分のスマホからフォームにアクセス |
| 紙 | デジタルが苦手なお客様 | 従来通り紙に記入（後日スタッフが転記） |

---

## この施策の強み・弱み・リスク

### 強み
- Googleフォームは無料。タブレット代程度の初期投資で済む
- HTML・プログラミング不要。即日構築・運用開始が可能
- 回答がスプレッドシートに自動集約。紙→手入力の二度手間がなくなる
- 3チャネル対応でデジタルリテラシーに左右されない

### 弱み
- デザインのカスタマイズに限界がある
- CRM（顧客管理システム）との自動連携は標準では難しい
- 個人アカウントで運用すると、退職・凍結時にデータ消失リスクがある

### リスクと対策

| リスク | 影響度 | 対策 |
|---|---|---|
| フォーム設定ミスによる情報漏洩 | 高 | 公開前に設定チェックリストで確認 |
| 個人情報保護法違反（同意未取得） | 高 | フォーム冒頭に利用目的と同意チェック欄を設置 |
| 紙対応分のデータ入力遅延・ミス | 中 | 「当日中に転記」のルールを徹底 |
| Googleアカウント凍結によるデータ消失 | 中 | Google Workspace（法人）を利用する |

---

## Googleフォームの取得項目（案）

### 必須項目（6項目以内に抑える）
1. 氏名
2. 電話番号
3. メールアドレス
4. ご希望エリア
5. ご希望の種別（賃貸 / 売買）
6. 個人情報取扱への同意（チェックボックス・必須設定）

### 任意項目（追加推奨）
- ご予算（賃料上限 / 購入価格帯）
- ご入居希望時期
- 間取り・広さの希望
- ご家族構成
- **ご来店のきっかけ（ポータルサイト / SNS / 紹介 / 通りがかり等）** ← 必ず入れる

> **注意**：年収・勤務先は来店アンケート段階では聞かない。初対面では心理的抵抗が大きい。商談が進んでから取得する。

---

## 個人情報保護の対応（法的必須事項）

1. フォーム冒頭に利用目的を明記（「物件のご紹介・ご連絡に使用します」等）
2. 同意チェックボックスを「必須」設定にする
3. 物件オーナー・管理会社への情報共有がある場合は事前に明記
4. 店舗内に個人情報保護方針を掲示し、フォーム内にも記載

### 運用ルール

| ルール | 内容 |
|---|---|
| アカウント管理 | Google Workspace（法人アカウント）で運用 |
| フォーム設定 | 「回答の概要を表示」は必ずOFF |
| アクセス権限 | 編集権限は管理者のみ |
| 紙対応 | 当日中にフォームへ転記。原紙はシュレッダー処分 |
| バックアップ | スプレッドシートを月1回以上エクスポート |

## さらに効果を高める追加施策

| 施策 | 優先度 | 効果 |
|---|---|---|
| LINE公式アカウント連携 | 高 | メールより開封率が高く追客に強い |
| 来店前の事前アンケート送付 | 高 | 来店時の接客準備の質が上がり成約率向上 |
| 「ご来店のきっかけ」の月次集計 | 高 | 広告費の最適配分に直結 |
| QRコード付きショップカード | 中 | 帰宅後に自分のペースで登録できる導線 |
| CRM（顧客管理システム）への移行 | 中（将来） | 登録数が増えてきたら検討 |

## 推奨アクション（優先順）

1. **Google Workspace（法人アカウント）を契約する**（月額680円〜/ユーザー）
2. **Googleフォームを作成する**（必須6項目 + ご来店のきっかけ、冒頭に同意欄）
3. **1週間テスト運用**（タブレット・QRコード・紙の3パターンを実際に試す）
4. **紙対応のオペレーションをルール化**（誰が・いつまでに転記するか）
5. **月次で「ご来店のきっかけ」を集計**して広告費の見直しに活用

## フォームのコピー案（コト作成）

### タイトル（推奨）

```
まずは、あなたの希望を教えてください。
── 記入時間の目安：約3分
```

### 冒頭の説明文（推奨）

```
ご来店いただき、ありがとうございます。

このフォームにご記入いただくと、担当スタッフがあなたの
ご希望をしっかり把握した状態でご案内できます。
「毎回同じことを説明する手間」が省け、ご希望に合った
物件をよりスピーディーにご提案できます。

所要時間は約3分です。
ご入力いただいた情報は、物件のご提案にのみ使用し、
第三者への提供はいたしません。安心してご記入ください。
```

### 個人情報の利用目的

```
ご入力いただいた個人情報は、物件情報のご提案・ご案内、
お問い合わせへのご返答のみに使用します。
法令に基づく場合を除き、第三者に提供することはありません。
```

### 同意チェックボックス

```
上記の個人情報の取り扱いに同意します。
```

### 任意項目の断り書き（フォーム内に添える）

```
※「任意」の項目は未入力でも送信できます。わかる範囲でご記入ください。
```

## 入力項目の推奨順序（ミオ調査）

**根拠**：「簡単な選択式 → ニーズ → 個人情報」の順が完了率が最も高い。電話番号を必須にすると完了率が5〜10%低下するため任意推奨。

| 順番 | 項目 | 形式 | 必須/任意 |
|---|---|---|---|
| 1 | ご希望の種別（賃貸 / 売買） | 選択式 | 必須 |
| 2 | ご希望エリア | 選択式 | 必須 |
| 3 | ご予算 | 選択式 | 任意 |
| 4 | ご入居希望時期 | 選択式 | 任意 |
| 5 | 間取り・広さの希望 | 選択式 | 任意 |
| 6 | ご家族構成 | 選択式 | 任意 |
| 7 | ご来店のきっかけ | 選択式 | 任意 |
| 8 | 氏名 | 記述 | 必須 |
| 9 | メールアドレス | 記述 | 必須 |
| 10 | 電話番号 | 記述 | 任意（「ご希望の場合のみ」と添える） |
| 11 | 備考・ご要望 | 記述 | 任意 |
