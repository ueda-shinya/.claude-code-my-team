## サービス概要

エキスパ（ex-pa.jp）は、株式会社エキスパートが運営するオールインワン型マーケティングツール。メール・LINE・ショートメールの「トリプル配信」を中心に、ステップ配信・読者管理・フォーム作成・決済・サイト作成・セミナー管理を一元化できる。主にネット系個人起業家・情報商材販売者が利用するマーケ自動化システム。

## 新旧システムの違い

| 項目 | 旧システム（旧メール配信） | 新システム（トリプル配信） |
|---|---|---|
| 正式名称 | エキスパートメール / 旧LINE配信 | トリプル配信 |
| 提供開始 | サービス創業時〜2022年5月9日 | 2022年5月10日〜現在 |
| 配信チャネル | メールのみ（LINE配信は旧LINE配信として別管理） | メール・LINE・ショートメールを統合管理 |
| LINE配信 | 旧LINE配信機能として別提供 | トリプル配信内で無制限・従量課金なし |
| 管理画面 | メール・LINE・その他が個別画面 | 一元管理（統合UI） |
| 料金体系 | メール配信数ベース | 3チャネル統合で月額固定（Light / Basic / Pro） |
| 対象ユーザー | 2022年5月9日以前の既存ユーザー | 2022年5月10日以降の新規契約者 + 任意移行者 |

### 段階的廃止スケジュール

- 2022年5月10日：トリプル配信正式リリース（以降、新規契約者はトリプル配信のみ）
- **2023年9月7日**：旧LINE配信機能終了の告知。対象ユーザーはトリプル配信へ移行必須
- **旧メール配信の廃止時期**：現時点で公式廃止告知なし（当面継続可能）
- 2024年12月1日：料金プラン値上げ（Light +1,000円／Basic +1,000円／Pro +2,000円）

## 移行可否の結論

**△ 条件付き可能（手動作業が必須。自動移行ツールなし）**

| 種別 | 可否 | 根拠 |
|---|---|---|
| 読者リスト | ○ | CSV/Excelインポートで移行可能 |
| ステップメール設定 | △ | エクスポート→インポート可。直接編集すると破損リスク |
| 既存の商品・決済 | △ | 継続稼働はするが新規追加・編集は有料プラン要 |
| 配信履歴・ステップ進捗 | 要注意 | 1次情報で明示なし。引き継がれない可能性が高い（未確認） |
| 登録/解除URL | 要注意 | 専用ドメインが変わる可能性あり |

## 移行できるもの／できないもの

**移行できる**
- メルマガ読者リスト（メール／電話／住所／自由項目、CSV・Excel）
- ステップメール設定（エクスポート→インポート。編集厳禁）
- 既存商品販売・決済の継続稼働

**移行できない・失われる可能性**
- メール配信プラン**解約時にステップメール・読者データは削除される**（解約＝データ消失、完了前解約は厳禁）
- 配信停止・解除URL・登録フォームURLの構造変化（専用ドメイン差異）
- 旧システムでの配信履歴・ステップ進捗（引き継がれない可能性が高い／未確認）

## 推奨移行手順（公式情報ベース）

1. 読者データをCSV/Excelでエクスポート（最初に実施）
2. ステップメール設定をエクスポート
3. トリプル配信側でマガジン（配信リスト）を新規作成
4. 読者データをインポート
5. ステップメール設定をインポート（直接編集禁止）
6. 旧・新を並行稼働させて検証
7. LP・既存メール内のフォームURL・解除URLをすべて差し替え
8. 問題なければ旧系統を停止

## 注意点・リスク（3つ）

1. **解約＝データ削除**：メール配信プラン解約時にステップメール・読者データが削除される（即時か期限猶予かは公式未明記）。移行完了まで絶対に解約しない。
2. **URL構造変化リスク**：旧⇔新で専用ドメインが異なる可能性あり。LP・メール本文・広告LPに埋め込んだURLが旧ドメインのままだと登録解除フォームが機能しなくなる恐れ。
3. **ステップ進捗の引き継ぎ不可（可能性が高い）**：旧で「5通目配信済み」の読者が、新側で最初から配信される＝既存読者への重複・順序乱れが発生しうる。
