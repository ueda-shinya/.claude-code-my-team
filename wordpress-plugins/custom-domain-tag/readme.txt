This plugin adds a custom tag to Contact Form 7 to include the site's domain name.

== Description ==
This plugin provides a custom tag [site_domain] that you can use in Contact Form 7 to dynamically insert your site's domain name into emails.

== Installation ==

1. Upload `custom-domain-tag` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the [site_domain] tag in your Contact Form 7 email settings.
