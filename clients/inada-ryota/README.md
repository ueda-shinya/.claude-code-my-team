## 相談内容（2026-04-07受領）

### 伝えたいメッセージ（稲田さん原文）

> 採用しても、すぐに辞めてしまう。
> それは「採用」の問題ではないかもしれません。
>
> 保護者対応、職員トラブル、行政対応。
> これらが整っていない職場では、どれだけ採用しても定着しません。
>
> そして今、少子化により、
> 保育園・幼稚園は「選ばれる側」の時代に入っています。
>
> これから残っていく園に必要なのは、
> 現場の負担を減らしながら、持続的に運営できる体制です。
>
> そのために必要なのが、
> クレーム・労務・行政対応を含めた、実務に落ちる法務です。
>
> 当事務所は、こども・若者を対象とする事業者支援に注力し、
> 現場と経営の両方を見据えたサポートを行っています。

### 現状の課題認識

| # | 課題 | 詳細 |
|---|---|---|
| ① | ターゲットの明確性 | 保育園・幼稚園経営者向けとは伝わるが、「どんな課題を持つ人向けか」が一目で分かりにくい |
| ② | 課題とサービスの結びつき | サービス内容は整理されているが、「そのサービスが何の経営課題を解決するか」の因果関係が弱い |
| ③ | 業界構造の変化への言及不足 | 少子化に伴う縮小・再編局面への「選ばれ、残っていくための経営課題」視点が不足 |
| ④ | 問い合わせの心理的ハードル | 弁護士相談へのハードルを下げる要素（無料相談、気軽さの訴求）が不十分 |
| ⑤ | 強みの伝達不足 | 企業法務経験・行政関与・こども若者分野への注力が「経歴」止まりで「依頼する理由」として伝わっていない |

### 改善の方向性

| # | 方向性 | 詳細 |
|---|---|---|
| ① | 「採用・定着」と「選ばれる園づくり」の両軸導入 | 「採用しても定着しない」課題 ＋ 少子化の中で「選ばれ続ける園」という中長期課題を入口に |
| ② | 悩みベースの導線設計 | サービス紹介ではなく「こんなお悩みは？」形式でユーザーの状況に寄り添い、サービスへ接続 |
| ③ | 法務＝現場改善＋持続的経営のメッセージ明確化 | 単なるトラブル対応ではなく「現場負担軽減→離職防止→選ばれる園づくり」に寄与する法務として前面に |
| ④ | 強みの再構成 | 企業法務・行政・こども若者分野を「だから保育事業者の現在と将来に対応できる」ストーリーとして再構成 |
| ⑤ | 問い合わせ導線の強化 | 初回相談ハードル低下文言 / 「状況整理だけでも可」明示 / 無料相談・日程調整ボタン設置 |
