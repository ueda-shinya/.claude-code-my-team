## 2. 梅プラン：モニタリング（月5万円）

### 2-1. 一言キャッチ
> **Phase 1で作った資産（LP・記事・診断ページ）を"腐らせない"ための健康診断プラン。稲田先生自身が運用の手綱を握りたい方向け。**

### 2-2. 含まれる成果物・アクティビティ

**① 月次アクセス解析レポート（簡易版）**
- GA4・Search Console・LP診断ページのCVRを統合した4〜6ページのPDFレポート
- 数字の羅列ではなく「先月比で何が動いたか」を3つの示唆に絞って記載
- レポート構成：
  - サマリー（1ページ）：セッション数・問合せ件数・診断完了数・主要KPIの前月比
  - 流入分析（1ページ）：流入チャネル別、検索クエリTOP10
  - コンテンツ別パフォーマンス（1ページ）：記事別PV・滞在時間・離脱率
  - 示唆と推奨アクション（1〜2ページ）：来月やるべき3つ
  - 付録：生データCSV

**② 戦略ミーティング月1回（60分／オンライン）**
- 前半30分：レポート読み合わせ
- 後半30分：先生からの相談事項・来月の方針確認
- 議事録は当日中にNotionで共有

**③ チャット相談（営業日48時間以内返信）**
- 運用に関する軽い質問ベース（例：「この記事の修正どう思います？」「この問合せ、返信文これでOK？」）
- 月10往復程度を想定
- 法律相談は対象外（あくまで運用相談）

**④ 年2回の棚卸しレポート（半期）**
- 6ヶ月・12ヶ月時点でSEO順位変動・競合5社との差分・Phase 1資産の陳腐化チェック
- 改善が必要な箇所をリスト化して提示

### 2-3. 月間想定工数（Office Ueda側）

| 作業 | 時間 |
|---|---|
| GA4/GSCデータ抽出・分析 | 3h |
| レポート作成 | 3h |
| MTG準備・実施・議事録 | 1.5h |
| チャット対応 | 1〜2h |
| **合計** | **8〜10時間/月** |

### 2-4. 稲田先生側の負担
- 月次MTG：60分
- レポート事前読み込み：15分
- チャット返信：月30分程度
- **合計：約90分/月**

### 2-5. 納品物フォーマット
- 月次レポート：PDF（A4・4〜6ページ）
- 生データ：Googleスプレッドシート共有
- 議事録：Notion（永久保存・検索可能）
- チャット：Chatwork

### 2-6. このプランを選ぶべき稲田先生の状況
1. Phase 1を通じてWebマーケの基礎を理解し、記事制作は自分でやりたい先生
2. 月の広告費・運用費を最小限に抑えつつ、数字の見方だけ伴走してほしい先生
3. 事務所に広報担当の事務員さんがいて、実作業は内製できる体制がある先生
4. まずは半年だけ伴走してもらい、手応えを見てから竹にアップグレードしたい先生

### 2-7. 6ヶ月後の到達イメージ
- Phase 1で公開した記事5本のうち、3本以上が狙ったキーワードで検索1ページ目
- 月次レポートを先生自身が"読める"ようになり、改善判断ができる状態
- 問合せ月2〜3件ペース（Phase 1終了時の水準を維持）
- 稲田先生が「次にやるべきことが見えている」状態

### 2-8. プラン境界の明確化
- **竹との差分：** 記事制作・スポット制作・軽微なLP修正は一切含まない。全て「提案まで」で止まる
- **下位プランへの落とし先：** 梅が最下位。これ以下はスポット契約のみ
- **注意喚起：** 「Phase 1で作った資産が半年で古くなる業界ではないが、記事の追加がゼロだと競合5社に抜かれるリスクがある」ことを商談で必ず伝える

## 3. 竹プラン：運用支援（月10万円）⭐推奨

### 3-1. 一言キャッチ
> **月2本のSEO記事で"選ばれ続ける園をつくる法務設計パートナー"の情報資産を積み増し続ける、大半の先生にとっての標準プラン。顧問契約1件（LTV年120万円）取れれば元が取れる水準。**

### 3-2. 含まれる成果物・アクティビティ

**① 梅プランの全機能**
- 月次レポート・戦略MTG・チャット相談（※返信速度は24時間以内に短縮）
- 年2回の棚卸しレポート

**② SEO記事制作 月2本**
- 1本あたり3,000〜5,000字、ペルソナA「今夜も眠れない園長」の悩みに紐づくテーマ
- 記事テーマ例（ストック化）：
  - 「保育士の早期離職を防ぐ就業規則の5つの設計ポイント」
  - 「処遇改善手当の配分で失敗しない園長のための実務ガイド」
  - 「保護者クレームをこじらせない初動対応マニュアル」
  - 「労基署調査が入ったときに園長がやるべきこと／やってはいけないこと」
  - 「採用した保育士が3ヶ月で辞める園に共通する3つの法務ミス」
- **制作フロー：**
  1. 月初MTGで翌月の2本のテーマ・切り口を確定（15分）
  2. Office Ueda側で構成案作成 → 先生レビュー（Chatwork・往復1〜2回）
  3. 初稿作成 → 先生の法務監修（Googleドキュメントでコメント）
  4. 修正・入稿・公開
  5. 公開後1週間でGSC登録・内部リンク調整
- **先生の監修負担：** 1本あたり20〜30分（構成レビュー10分＋初稿監修15分）

**③ スポット制作対応 月1件**
- 対象例：
  - LP内の1セクション修正（コピー差し替え・画像追加）
  - 既存ホワイトペーパーへの追記版PDF作成
  - 診断ページの設問追加・ロジック調整
  - 新規チェックリストPDF1枚（A4・両面）
  - ステップメールの文面差し替え（1通）
- **制限：** 1件あたり工数上限5時間。超える場合は別途見積
- 繰越不可（使わなかった月の分は消滅）

**④ 軽微なLP改善実装**
- 月1回程度、CVR改善のためのマイナーチェンジ（ボタン色変更、ファーストビューコピー調整、フォーム項目削減など）
- A/Bテストまでは行わない（松プランの領域）

**⑤ 月次戦略MTG（60分）＋中間チャットチェックイン**
- 月初：戦略MTG
- 月中：Chatworkで10分程度の進捗確認（非同期OK）

### 3-3. 月間想定工数（Office Ueda側）

| 作業 | 時間 |
|---|---|
| レポート作成・分析 | 5h |
| SEO記事2本（構成・執筆・入稿・監修対応） | 10〜12h |
| スポット制作1件 | 3〜5h |
| LP軽微修正 | 1〜2h |
| MTG準備・実施・議事録 | 2h |
| チャット対応（24h以内返信） | 1〜2h |
| **合計** | **22〜28時間/月** |

※ 第12章の工数試算（運用フェーズ48〜66h）のうち、コア部分を竹プランに配分。残りは松プランへの上乗せ分。

### 3-4. 稲田先生側の負担
- 月次MTG：60分
- 記事監修：2本×25分＝50分
- スポット制作レビュー：20分
- チャット返信：月30分
- LP修正確認：10分
- **合計：約180分/月（週あたり45分）**

### 3-5. 納品物フォーマット
- 月次レポート：PDF（6〜8ページ、示唆を厚めに）
- SEO記事：Googleドキュメント → WordPress入稿済み状態
- スポット制作物：PDF／画像／HTML（案件による）
- 議事録：Notion
- 月末まとめ：「今月やったこと一覧」をChatworkで送付

### 3-6. このプランを選ぶべき稲田先生の状況
1. 問合せを"今の水準で維持"ではなく"積み増したい"と考えている先生
2. 記事執筆にかける時間がない／文章は書けるが構成設計が苦手な先生
3. 顧問契約を年3〜5件ペースで増やしたい成長フェーズの事務所
4. Instagram運用までは不要だが、Web記事での露出は継続したい先生

### 3-7. 6ヶ月後の到達イメージ
- Phase 1の5本＋竹で追加した12本＝計17本のSEO記事資産
- うち5〜7本が主要キーワードで検索1ページ目、月間オーガニック流入500〜800セッション
- 月次問合せ5〜8件、うち顧問契約化1〜2件
- **ROI：** 顧問1件獲得できれば年120万円 = 竹プラン年額と相殺。2件以上でプラス
- 「稲田先生の名前でググると関連記事が並ぶ」状態

### 3-8. プラン境界の明確化
- **梅との差分（+5万円の納得感）：**
  - 記事2本（10〜12h）= 単独発注なら1本5〜8万円が相場なので、この時点で5万円分は回収
  - スポット制作1件 = 単独なら3〜5万円
  - チャット返信速度が48h → 24h
  - **「梅は数字を見るだけ、竹は数字を動かす」** と表現すると商談で伝わりやすい
- **松との差分：**
  - Instagram運用なし
  - セミナー資料制作なし
  - LP改善は"軽微修正"まで（設計〜実装はなし）
  - MTGは月1回（松は月2回）

**ダウングレード（松→竹、竹→梅、松→梅）**
- **3ヶ月前予告必須**

- 例：1月に申し出 → 4月から新プラン適用
- 予告期間中は現プランの料金・納品物を維持

### 6-2. 解約条件

- **最低継続期間：6ヶ月**（Phase 2開始から起算）
  - 理由：SEO効果は3〜6ヶ月のラグがあり、短期解約は先生にとっても不利益
- **解約予告：2ヶ月前**
  - 例：6月末解約希望 → 4月末までに申し出
- **最低継続期間中の解約：** 残月数分の料金を一括精算（例：3ヶ月目で解約なら残3ヶ月分を支払い）
- **例外：** 稲田事務所の事業継続困難・法的トラブル等のやむを得ない事情は協議の上で免除

### 6-3. 納品物の権利帰属
- 記事・LP・Instagram投稿・セミナー資料など全ての制作物の著作権は稲田事務所に帰属
- Office Uedaは実績として「制作事例」に匿名掲載する権利を保持（先生が許可した場合のみ実名）

### 6-4. 稼働しない期間の扱い
- 年末年始（12/29〜1/3）・GW（5/3〜5/5）・お盆（8/13〜8/15）は稼働休止
- 休止期間の料金減額なし（年間スケジュールで平準化済み）

### 6-5. チャット相談の範囲
- **含まれる：** 運用・マーケ・数字・制作物に関する相談
- **含まれない：** 法律相談（そもそも逆方向）・Office Ueda以外が制作した資産に関する相談（別料金）
