## 佐々木社長の特性

- 営業経験が長く、顧客との関係構築は得意
- 組織運用・仕組み化は苦手
- **「自分が当たり前にできること＝みんなできる」という認識のズレがある**
- 仕組み化＝義務・ルールと感じやすい。「なんで言語化が必要？」となりがち
- 従業員の意識レベルが自分と異なることに気づいていない

## 支援の背景・現状

- 業務委託の営業が育ってきて社長に余裕が生まれた → **今が仕組み化の着手タイミング**
- これまで「やりたいことが従業員不足でできなかった」経緯あり
- 仕組みを作ってほしいと依頼されているが、社長自身の暗黙知が多く抵抗感も出やすい

## シンヤさんの接し方メモ

- 「義務を作る」ではなく「**社長の頭の中を再現する仕組み**」フレームで進める
- 社長を「教える側・知識を渡す側」に置くと協力的になりやすい
- ギャップは言葉で伝えず、**社長自身に発見させる**（従業員に同じ状況を聞かせる等）
- 新規事業（ホームコンシェルジュ）は「一緒にゼロから作る」感覚で進めやすい → 仕組み化の入口として活用
