# ランドプランニング / 訪問サービス事業

> **事業名（案）：** 黒瀬町「住まいの相談窓口（ホームコンシェルジュ）」

---

## 事業の目的

「便利屋的な単発作業」が目的ではない。
**「困ったらまずランドプランニングに相談しよう」と真っ先に思い出してもらえる関係性を作ること**が目的。

その関係性を活かして、最終的には**リフォーム・不動産売買の指名受注**（相見積もりなし）につなげる。

---

## ターゲット・エリア

- **エリア：** 広島県東広島市黒瀬町
- **対象：** これから売買取引を行う顧客（成約後にホームコンシェルジュを紹介する形で展開）
- **既存リスト：** なし（過去データは未デジタル化のため、新規取引から積み上げていく）

---

## テスト戦略

### フェーズ設計
| 項目 | 設定 |
|---|---|
| テスト件数 | 20〜30件 |
| テスト期間 | 3ヶ月 |
| 無料回数 | 1顧客あたり最大3回 |

### アプローチの流れ
**売買成約（接点発生）→ 成約後の挨拶・説明 → 訪問（テスト実施）**

- 成約後：「アフターサービスとして」ホームコンシェルジュを案内する
- 案内：無料テストであること・営業でないことを明示して紹介
- 訪問：無料テストを実施。初回に「住まいのカルテ」の説明も行う

### テスト終了後の移行
```
無料テスト（最大3回）
    ↓
テスト終了の挨拶訪問（4回目）
  → アンケート実施（満足度・継続希望・料金感覚）
    ↓
継続希望 → 有料プランへ移行
継続不要 → 「何かあればいつでも」で関係維持 → 定期的にDM接触
```

| プラン名 | 料金 | 内容 | 狙い |
|---|---|---|---|
| **テスト招待（無料）** | 無料（最大3回） | 既存客限定。軽作業＋住まいのカルテ作成 | 信頼関係の再構築・カルテデータの蓄積 |

| **定額プラン・プレミアム** | 未定（テスト後決定） | 月4回（週1ペース）＋会員割引 | 毎週頼りたい高齢者向け。他社流出防止 |

## サービス範囲

### やること（1回30分以内・特殊工具不要）
- 電球・蛍光灯の交換
- エアコン・洗濯機のフィルター清掃
- 買い物のおつかい（10kgまで）
- 家電の操作説明・Wi-Fi接続確認

### やらないこと（専門業者へ引き継ぎ）
- 電気・水道・ガス等の資格が必要な工事
- 屋根などの高所作業・家電の分解修理
- 不用品の直接回収
- お客様を車に乗せての送迎（白タク行為のため**絶対禁止**）

---

## 本業へのつなげ方（4ステップ）

1. **純粋な人助け** → 軽作業で訪問し、信頼を稼ぐ
2. **「住まいのカルテ」に記録** → 訪問ついでに外壁・雨樋等をプロ目線でチェック・写真記録
3. **プロからのアドバイス** → 記録をもとに「台風前に雨樋を直しませんか？」と自然に提案
4. **指名受注** → 「いつもの親身な担当者」として他社比較なしに受注

> 営業・押し売りは一切しない。信頼関係が先、提案は後。

## メッセージ方針

**核：「営業なし・押し売りなし。気が向いたときにどうぞ。」**

### 避けるべき表現
| NG | 理由 |
|---|---|
| 「今なら無料！期間限定！」 | 焦らせる表現は信頼を壊す |
| 「お得なサービスです」 | 既存客への不動産系コミュニケーションに逆効果 |
| 「リフォームや売却のご相談にも」 | 最初から目的がバレる |
| 「ご自宅の状態を確認させていただきます」 | 査定・調査目的と誤解される |

## 成功指標

| 指標 | 目標 | 備考 |
|---|---|---|
| 手紙への承諾率 | 20%以上 | 既存客との関係が生きているかの検証 |
| 3回リピート率 | 70%以上 | サービス自体の価値検証 |
| **有料移行率** | **30%以上** | **ビジネス成立の最重要指標** |
| リフォーム相談発生 | 3件以上（6ヶ月以内） | 最終ゴールへの先行指標 |

### 判断基準
- 有料移行率30%以上 → 本格展開へ
- 有料移行率10〜30% → サービス内容・料金を調整して第2テスト
- 有料移行率10%未満 → サービス設計の根本見直し
