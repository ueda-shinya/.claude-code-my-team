## 第1章 営業の心構えと準備

### 1-1. 結果を出す営業マンと出せない営業マンの決定的な違い

**📊 重要な事実:**
営業1年目で結果を出す人と出ない人の違いは、**「営業時間外の生き方」**です。

#### ✅ 結果を出す営業マンの行動
- 帰宅後も明日の商談をシミュレーション
- 営業ノウハウを自主的に学習
- 先輩のトークを録音して研究
- 部屋を整理し、勉強に集中できる環境を維持

#### ❌ 結果が出ない営業マンの行動
- 帰宅後はゲームやスマホに終始
- 何も勉強しない
- 商談の振り返りをしない
- 環境整備をしない

### 1-2. 新人が最初にやるべき2つのこと

#### 【重要】業務時間外に必ず実行すべき2つのこと

**1️⃣ 同質化 - 売れている先輩をコピーする**

**具体的な手順:**
1. 売れている先輩営業マンに商談同行をお願いする
2. 商談中の会話を録音する(許可を得て)
3. 録音を聞いて、効果的なフレーズをノートに書き出す
4. すべて暗記する
5. 何度も反復練習して自分のトークとして使えるようにする

💡 **ポイント:** 最初は完全コピーでOK。自分流にアレンジするのは結果が出てから!

**2️⃣ 環境設定 - 家での勉強環境を整える**

**やるべきこと:**
- 自室を常に整理整頓
- 勉強机の上はシンプルに
- スマホは別の部屋に置く
- 営業ノートと筆記用具をすぐ使える位置に配置

## 第3章 最初の30秒のアプローチ

### 3-1. 飛び込み・テレアポの最初の30秒が9割を決める

**📊 重要な事実:**
訪問販売は**最初の30秒のアプローチで9割方決まっている**。断りにくくなるアプローチをするだけで成約率は上がります。

### 3-2. 最初の30秒でやるべき4ステップ

#### **ステップ1: 圧倒的な第一印象**

**対面の場合:**
- 90度の深いお辞儀を3秒キープ

**電話の場合:**
- 明瞭で元気な声のトーンで自己紹介

#### **ステップ2: 気遣いと感謝を伝える**

**実践トーク例:**
```
「お忙しいところすいません」
「お手数かけて申し訳ございません」
「ありがとうございます」
```

💡 **ポイント:** 相手の時間を尊重する姿勢を示す

#### **ステップ3: "特別感"を演出する**

**実践トーク例:**
```
「お世話になっております」
```

💡 **効果:** 相手に"特別に選ばれた"という印象を与え、断りにくくさせる

#### **ステップ4: イメージしやすい自己紹介**

**❌ NG例:**
```
「株式会社〇〇の田中です」
```

**✅ 良い例:**
```
「大阪駅の第一ビルの8階で営業を行っております、
株式会社〇〇の田中と申します」
```

💡 **効果:** 地名・建物名を入れることで親近感が生まれる

### 3-3. 興味がない相手へのアプローチ3ステップ

#### **ステップ1: 当たり前のように堂々と依頼する**

**テレアポの場合:**
```
「恐れ入ります。すぐに担当者様に代わっていただけますか?」
「社長様はいらっしゃいますか?」
```

**飛び込みの場合:**
```
「突然失礼します。大切なお話がございますので、
玄関先まで失礼させていただけますか?」
```

💡 **ポイント:** 一言だけ詫びて、その後は自信満々に要求する

#### ❌ NG行動
- × 「お時間よろしいですか?」と弱腰に聞く
- × 長々と謝罪する
- × 遠慮がちな態度

#### **ステップ2: クエスチョンマークを作る**

いきなり商材説明をせず、「どういうこと?」と思わせる問いを投げかける。

**実践トーク例:**
```
「突然ですが、これ受け取られましたか?」
「助成金はおいくらもらわれましたか?」
「日本の保険会社は何社あると思いますか?」
```

💡 **効果:** 頭に"疑問符"をつくり、聞く姿勢と集中を引き出す

#### **ステップ3: 自分事だと思ってもらう**

商品の特徴ではなく、**お客様がどう変わるか**を語る。

**❌ NG例:**
```
「この商品は最新技術を使用しており、他社より優れています」
```

**✅ 良い例:**
```
「この商品を使うと、御社の業務が30%効率化され、
残業時間が大幅に削減できます」
```

💡 **ポイント:** すべて「お客様にとってのメリット」として伝える

## 第4章 信頼関係を築く心理学

### 4-1. ザイアンスの熟知の法則

**📚 理論:**
人は相手の**内面性**を知れば知るほど、興味・好意が高まり「この人には何でも話せる」と感じる。

### 4-2. 2秒の自己紹介テクニック

#### 基本原則:
長々と自己紹介するのではなく、会話の流れで自分の内面を**各2秒程度**で小出しにする。

#### 開示する順番:

**レベル1: 趣味・嗜好**
```
「私の休みの日は娘と公園に行くのが大好きなんですけど、
〇〇さんは休みの日に何をされているんですか?」
```

**レベル2: 強み・長所**
```
「私は人と話すのが得意で、毎回相手のニーズを
的確に引き出すよう心がけています」
```

**レベル3: 弱み**
```
「実は僕、子育てがめちゃくちゃ下手で、
育児も家事もほとんどできないんですよ」
```

💡 **ポイント:** 
- 各2秒程度で短く
- 会話の自然な流れで挿入
- 弱みは最後に開示

#### ❌ NG行動
- × 一度に全部話す
- × 長々と自分語りをする
- × 最初から弱みを見せる

## 第5章 売れるトーク術の基本

### 5-1. 結論から話す構成

**📌 重要原則:**
同じ話時間でも、結論を先に伝えることで「短く・集中して」聞こえる。

### 5-2. 聞くスペースを作る

#### なぜ結論から話すのか?

脳は「何の話が始まるか」を把握すると、スペースを空けて集中モードに入る。

#### ❌ グダグダトークの原因
- 何の話か分からない
- いつ終わるのか分からない
- お客様がイライラする

#### ✅ 売れるトーク構成

**6ステップ構成:**

**1. 前振り**
```
「これは100社あるリフォーム会社の中で、
なぜ皆様に選ばれているかという話なんですけど」
```

**2. 結論(要点)**
```
「実はうちの塗装は耐久年数が驚異の20年間と非常に長いんです」
```

**3. 理由説明**
```
「なぜなら先端コーティング技術を使用しているからです」
```

**4. 証拠/データ**
```
「実験結果でも20年近く剥がれないことが確認されています」
```

**5. 追加メリット/安心ポイント**
```
「しかもアフターフォローもしっかりしており、
何かあればすぐに駆けつけます」
```

**6. まとめ(再結論)**
```
「だから、100社の中からお客様が当社を選ばれるのは
当然のことなのです」
```

💡 **効果:** 聞き手は最後まで集中して聞き、納得感が高まる!

## 第6章 お客様の心を開く褒め方

### 6-1. 初対面で心の扉を開く3ステップ

#### ステップ1: 羞恥心を外す

**マインドセット:**
恥ずかしさを捨て、堂々と相手を褒められるマインドをつくる。

#### ステップ2: オーラを褒める

第一印象で感じた雰囲気をざっくり褒める。

**実践トーク例:**
```
「初めましてお会いした瞬間に感じたことなんですけど、
めちゃくちゃ誠実そうな方ですよね。
オーラめちゃくちゃ出てますね」
```

**褒めるポイント例:**
- 誠実そう
- 知的
- 優しそう
- 仕事ができそう
- 信頼できそう

#### ステップ3: 二度褒める

一度目の褒め返しを受けたら、**比較**を用いて再度褒める。

**実践トーク例:**
```
「いやこれほんまの話でね、私実は38年生きてますけど、
知的とか仕事できそうとか1回も言われたことないんですよ。
ですから羨ましいです」
```

**比較の種類:**
- (a) 自分との比較: 「私は○○ですが、〇〇さんは△△ですね」
- (b) 今までとの比較: 「今まで○○でしたが、〇〇さんは△△ですね」

💡 **効果:** 快楽成分を引き出し、心の扉を完全に開かせる!

### 6-2. 具体的なものを褒める

**実践トーク例:**
```
「ネクタイおしゃれですね。
めちゃくちゃかっこいいじゃないですか」
```

#### 褒めるポイント例:
- ネクタイ
- 腕時計
- 手入れされた花
- オフィスのレイアウト
- 趣味のアイテム

## 第7章 実践的心理テクニック5選

### 7-1. ハロー効果

**すでに第2章で解説済み**

### 7-2. 聞き手の法則

**📚 理論:**
人は利き手側に立つ人を優位と感じる。

#### 実践方法:

**ステップ1:** お客様の利き手側(日本人の約9割は右利きのため、お客様から見て左側)に立つ

**ステップ2:** パーソナルスペース(約120cm)を侵さず、斜め横から説明する

💡 **効果:** 無意識に信頼感を与える!

### 7-3. バックトラッキング(鸚鵡返し)

**📚 理論:**
相手の言ったことをそのまま繰り返すだけで、「よく聞いてくれている」と感じさせる。

#### 実践例:

**お客様:** 「昨日、娘と遊園地に行ったんですよ」
**営業:** 「遊園地に行かれたんですね!」

**お客様:** 「そうなんです。とても楽しかったです」
**営業:** 「楽しかったんですね!」

💡 **ポイント:** 
- 新人でもすぐに実践できる
- 特別な準備不要
- どの場面でも使える

### 7-4. フット・イン・ザ・ドア

**📚 理論:**
最初に小さな要求で"イエス"を取り、その後に大きな要求を出すと承諾率が上がる。

#### 実践手順:

**ステップ1: 小さな要求**
```
「まず資料だけでもご覧いただけますか?」
```

**ステップ2: 数日後に大きな要求**
```
「来週お伺いして詳しくご説明させていただけますか?」
```

💡 **効果:** 一度「イエス」と言った後は、次の要求も受け入れやすくなる!

### 7-5. ドア・イン・ザ・フェイス

**📚 理論:**
まず大きな要求を投げかけて断らせた後、本命の要求を提示すると承諾率が上がる。

#### 実践手順:

**ステップ1: 最大の提案(意図的に断らせる)**
```
「フルセットのプレミアムプランはいかがですか?
月額50万円です」
```

**ステップ2: 本命の提案**
```
「では、こちらのスタンダードプランはいかがですか?
月額10万円で、必要な機能は全て含まれています」
```

💡 **効果:** 返報性の法則により、本命の提案が通りやすくなる!

## 第8章 クロージングの基礎

### 8-1. 即決させる3つのコツ

#### コツ1: 先回り - 選択の自由を与える

**実践手順:**

**ステップ1: 断る自由を与える**
```
「〇〇様、これから商品のことをお伝えするんですが、
全然ダメだ、いらないなと思ったら、
バッサリ断ってもらって大丈夫ですから安心してくださいね」
```

↓ お客様から「はい」を取る

**ステップ2: 始める同意を取る**
```
「もし、これ自分に必要だなと思っていただけたら、
ぜひスタートしてくださいね」
```

↓ 再度「はい」を取る

**ステップ3: 本題に入る**

💡 **効果:** お客様を「自分ごと化」し、一貫性の法則を働かせる!

#### ❌ NG行動
- × 「断ってもいいけど、良かったら始めてください」を一文にまとめる
- × 先回りの順序を間違える

#### コツ2: if活用 - 想像させて感情を揺さぶる

**📚 理論:**
脳には想像と現実の区別がない。想像させればさせるほど物は売れていく。

**実践例:**
```
「もし〇〇様がこの商品をご購入いただいたら、
生活はどう変わりそうですか?」

「もしここに置くとしたら、どこが最適ですか?」
```

💡 **ポイント:** 少なくとも3~5回はif質問を挿入する

#### コツ3: 一番刺さる理由を聞き出す

**実践手順:**

**質問1:**
```
「本日お話を聞いていただいて、
どこのポイントが良かったと思っていただけましたか?」
```

**質問2:**
```
「なぜそのポイントが良いと思われたんですか?」
```

**クロージング:**
```
「〇〇様がおっしゃった○○の効果が得られるので、
ぜひ今スタートしましょう」
```

💡 **効果:** お客様自身が言った理由でクロージングするから刺さる!

### 8-2. お客様が自分でクロージングする4つの質問

**成約率88%を実現する質問パターン**

#### 質問1: どこが良かったか
```
「本日お話を聞いていただいて、
どこのポイントが良かったと思っていただけましたか?」
```

#### 質問2: なぜ良いと思ったか
```
「なぜそのポイントが良いと思われたんですか?」
```

#### 質問3: 生活がどう変わるか
```
「それが実現したら、〇〇様の生活は
具体的にどう変わりそうですか?」
```

#### 質問4: 気分はどうか
```

```

💡 **効果:** お客様自身に「買う理由」と「買う目的」を言語化してもらうことで、心理的リアクタンス(抵抗感)を回避し、自然に契約に導く!

- □ 明るく元気な声

### 【最初の30秒チェックリスト】

#### アプローチ
- □ 気遣いと感謝を伝えた
- □ 「お世話になっております」で特別感演出
- □ 地名・建物名を入れた自己紹介
- □ 当たり前のように堂々と依頼した

#### 興味喚起
- □ クエスチョンマークを作る問いかけをした
- □ お客様にとってのメリットを語った
- □ 商品の特徴ではなく、お客様の変化を伝えた

### 【信頼関係構築チェックリスト】

#### 自己開示
- □ 趣味・嗜好を2秒で開示
- □ 強み・長所を2秒で開示
- □ 弱みを2秒で開示(最後に)

#### 褒め方
- □ オーラを褒めた
- □ 二度褒めた(比較を使って)
- □ 具体的なものを褒めた

### 【トーク術チェックリスト】

#### 構成
- □ 前振りをした
- □ 結論から話した
- □ 理由を説明した
- □ データ・証拠を示した
- □ 事例を紹介した
- □ まとめで再結論

#### 心理テクニック
- □ バックトラッキング(鸚鵡返し)を使った
- □ お客様の左側に立った(聞き手の法則)
- □ フット・イン・ザ・ドアを使った
- □ ドア・イン・ザ・フェイスを使った

### 【クロージングチェックリスト】

#### 先回り
- □ 断る自由を与えた
- □ 始める同意を取った
- □ 2段階の「はい」を確実に取った

#### if活用
- □ 3~5回のif質問を挿入した
- □ お客様に想像させた

#### 刺さる理由
- □ どこが良かったかを聞いた
- □ なぜ良いと思ったかを聞いた
- □ 生活がどう変わるかを聞いた
- □ 気分はどうかを聞いた
- □ お客様が言った理由でクロージングした

### 【業務時間外チェックリスト】

#### 同質化
- □ 先輩に商談同行をお願いした
- □ 商談を録音した(許可を得て)
- □ 有効なフレーズを書き出した
- □ 暗記した
- □ 反復練習した

#### 環境設定
- □ 自室を整理整頓した
- □ 勉強机をシンプルにした
- □ スマホを別の部屋に置いた
- □ 営業ノートを準備した

## まとめ

このマニュアルは、即決営業チャンネルの人気動画トップ10から抽出した実践的なノウハウをまとめたものです。

### 新人営業マンが最初の1ヶ月で身につけるべき必須スキル:

1. **第一印象の作り方** - 90度のお辞儀×3秒キープ
2. **最初の30秒のアプローチ** - 特別感×親近感
3. **信頼関係を築く心理学** - 2秒の自己紹介
4. **売れるトーク術** - 結論から話す構成
5. **心の扉を開く褒め方** - 羞恥心を外す→オーラを褒める→二度褒める
6. **実践的心理テクニック** - ハロー効果、聞き手の法則、バックトラッキング、フット・イン・ザ・ドア、ドア・イン・ザ・フェイス
7. **クロージングの基礎** - 先回り、if活用、刺さる理由を聞く
8. **業務時間外の過ごし方** - 同質化と環境設定

### 最重要ポイント:

✅ **1日1章ずつ読み、その日のうちに実践する**
✅ **トーク例は声に出して練習する**
✅ **チェックリストで毎日自己評価する**
✅ **NG行動を必ず確認する**
✅ **結果が出るまで先輩のトークを完全コピーする**
