**競合環境:**

## 4. 仮説（現時点での推測）

### 仮説1: 「集客の量」が最大のボトルネック
コンテンツの質（LP滞在時間2分30秒）やサービスの強みは十分にある。しかし「そもそも見込み客がたどり着いていない」のが現状の最大課題。

**根拠**: LP 7日間のセッション数がわずか9件 / Instagram広告からの流入が0件 / 東広島市の事業所数6,836件に対してリーチできていない

### 仮説2: SEOブログの読者と問い合わせターゲットにミスマッチがある
技術ブログの読者は同業者やエンジニアが多く、「ホームページを作りたい・改善したい」中小企業経営者層とは異なる。

**根拠**: サイト改善レポートで既に指摘済み / 経営者平均年齢62.3歳 → 技術記事ではなく「経営課題の解決策としてのWeb」という切り口が刺さりやすい

### 仮説3: 紹介・口コミ頼みの集客構造になっている可能性
デジタルチャネルよりも紹介・既存顧客からの口コミで受注が回っており、再現性がなく受注量が安定しない。

**根拠**: GA4でdirect/none（直接流入）が最多 → 「紹介された人がURLを直接入力する」パターンと整合

### 仮説4: 「補助金 × ホームページ制作」の訴求が十分に活かされていない
小規模事業者持続化補助金（2026年第19回: 4/30締切）は強い動機づけになるが、SNS・LP・ブログで前面に出ていない可能性。

**根拠**: 広島県で97%の中小企業が支援制度を活用できていない / 「WHY Now?」の訴求として補助金の締切は最強のトリガー

### 仮説5: チャネルが分散しすぎて、どれも中途半端になっている
Instagram事業・個人・X事業・個人・Threads・ブログ・GBP・LPと、個人事業主のリソースに対してチャネル数が多すぎる。

**根拠**: 個人事業主が制作業務と並行してマーケティングを回す時間は限られる / フォーカシング効果の観点からも絞った方が成果が出やすい

## 5. 明日の議論に向けた補足メモ

**マーケティングの基本順序（この順番で議論する）:**
```
「誰に」（ターゲット） → 「何を」（メッセージ・価値） → 「どう届けるか」（チャネル・施策）
```
「Instagramをどう運用するか」から入ると手段が目的化する。**まず「誰に届けたいか」を固める**のが最優先。

**次のステップ（明日の議論後の想定）:**
1. 質問リストの回答をもとに「課題定義シート」を完成させる
2. ターゲット（ペルソナ）を1〜2パターンに絞る
3. チャネル優先順位を決定し、最初の2週間でテストする施策を1つ決める
