# カラーミー × GMC 自動同期ツール

## プロジェクト概要

カラーミーショップの商品情報をGoogle Merchant Center（GMC）に自動同期し、
Googleショッピングタブへの無料リスティング掲載を実現するツール。

officeueda が導入設定代行・継続フォローサービスとして提供する。

## ビジネスモデル

```
ツール（非公開・無料）
  ↓ officeueda が導入セットアップ
カラーミーユーザー（EC事業者）
  ↓ 月次フォロー契約
officeueda の収益
```

## 方針

- カラーミー・GMC ともに無料プランの範囲内で完結
- カラーミーアプリストアへの公開は行わない（officeueda 経由で導入）
- まず US-SAIJO さん（2026年4月ECオープン）で実証

## 機能スコープ（初版）

| 機能 | 内容 |
|---|---|
| 商品フィード生成 | カラーミーAPIから商品情報を取得してGMC用フィードを生成 |
| GMC自動登録 | GMC Content APIに商品データを送信 |
| 定期同期 | 商品追加・更新・削除をGMCに反映（30日更新義務の自動化） |
