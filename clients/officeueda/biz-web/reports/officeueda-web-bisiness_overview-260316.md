# オフィスウエダ 事業内容

## 1. ポジショニング
「あ、ちょっと来てよ」で呼べる、**東広島・呉・広島の“顔が見える”Web担当者**です。対面で事業理解を深め、難しいIT用語や更新作業は丸投げで対応し、隣で成長を支えるスタイルをとっています。

## 2. 使命・方針（MVV）
*   **Mission**：自分たちが心から良いと思えるサービスをつくり、その価値を自信を持って届ける
*   **Vision**：愛するものだからこそ、お客様の幸せと成功を本気で支える
*   **Values**：誇れるものだけを提供／お客様の成功を心から願う／関わるすべての人が愛せる環境

## 3. 提供領域（事業内容）
Web制作を中核に、集客・採用・運用まで**“成果前提”で設計・実装・保守を支援**しています。
*   **コーポレート/店舗サイト制作**：信頼性や会社情報発信の基盤構築
*   **採用サイト・LP制作**：応募・購入など単一目的の導線を最適化
*   **集客強化リニューアル**：問い合わせ増を目的とした全面的な改修
*   **WordPressトラブル対応**：表示不具合やログイン不可等の復旧（緊急トラブルに即応する「WP復旧24」）
*   **保守・管理**：更新、バックアップ、問い合わせ対応、Zoom支援など

## 4. 対象顧客
**東広島・呉・広島エリアの事業者**を中心に、対面相談や訪問対応を強みとして支援しています。相談テーマは「制作」にとどまらず、運用や改善、トラブルの復旧まで対応可能です。

## 5. 選ばれる理由（3つの強み）
*   **地域密着**：顔を見て話せる安心感を提供し、現場の空気を踏まえた提案を行います。
*   **丸投げOK**：専門用語は不要。面倒な手続きや更新作業はすべて代行します。
*   **成果重視**：単に「作る」のではなく、「売れる／応募が来る」ことを前提に設計します。
