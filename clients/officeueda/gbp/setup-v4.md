## 0. 概要・目的

### 位置づけ
Stage 1でブランディング（名刺・自己紹介・FVコピー）が確定したため、Stage 2でオンライン上の「もうひとつの名刺」であるGBPを v4サービス設計＋Stage1ブランディング と完全整合させる。

**本ドキュメントのゴール：**
- シンヤさんがGBP管理画面にコピペするだけで、v4訴求軸（A日常言語・B電話・C広島中小企業・D丸ごと）が完全反映された状態になる
- 景品表示法対策（駆けつけるNG・訪問は専属プランのみ併記）を全文で遵守
- 月1-2本の投稿・写真追加で運用が回る状態を作る

### 現状→更新後の差分（想定）

### 1-2. カテゴリ

#### 主カテゴリ（1つ）
```
コンピューター サポート＆サービス
```
**選定理由**（2026-04-21 シンヤさん最終判断でC→Bに変更）：
1. 検索ボリュームが大（「パソコンサポート」「PC修理」等、地方中小社長が実際に検索する語彙）
2. v4主力「ITまるごとサポート」（月額27,500円で電話・メール・トラブル対応）の実態と一致
3. MEO（ローカルパック3枠表示）で有利（「○○市 パソコンサポート」系検索で地図表示されやすい）
4. v4訴求「電話1本で、ITのことなら何でも」＝手を動かすサポート業態と整合（「コンサルタント」は話を聞く業態）

#### 副カテゴリ（2つに絞る）

1. **コンピューター コンサルタント**（第1副）
   - **理由**：v4の「IT相談窓口・もうひとりのIT担当」ポジションを保持。"相談もできる"という上位機能も拾う。サポート（手）＋コンサルタント（頭）で幅をカバー。

2. **ウェブデザイナー**（第2副）
   - **理由**：Web集客パッケージ（55-88万）はアップセル商品だが、「ホームページ 東広島」で検索される入口として保持価値あり。主カテゴリにしないことで"制作会社"に落ちるのを回避。

**シンヤさん確認ポイント**：GBP管理画面で上記3カテゴリの正式名称（スペース位置等）を確認。Googleは日本語表記を改定することがあるので、管理画面のプルダウン表示通りに選択。

### 文字数カウント
- 全角749文字（GBP上限750文字内、安全マージン1文字）
- GBP管理画面で貼り付け時、改行は残す（段落ごとの可読性優先）

### SEO語の自然埋め込みチェックリスト
- ✅「広島」×5回／「東広島」×2回／「呉」×1回／「広島市」×2回
- ✅「中小企業」×4回
- ✅「ホームページ」×3回
- ✅「パソコン」×3回
- ✅「IT」×13回（相談・担当・サポート等の派生含む）
- ✅「Googleマップ」×1回／「GBP」×1回
- ✅「月額」「無料」「診断」（ロングテール用）

### 景表法セルフチェック
- ✅「駆けつける」「即対応」「すぐ行きます」「24時間365日」「必ず」→ **未使用**
- ✅ 訪問言及時に「専属プランのみ月1回」を併記済
- ✅「24時間365日対応ではありません」を明示（有利誤認防止）
- ✅「売り込みはいたしません」は行動約束として適正（景表法対象外）

### v4訴求4軸カバレッジ
- A日常言語：◎「ちょっとパソコン見て」「誰に聞いたらいいか分からん」「おらんけぇ困っとる」（方言入り）
- B電話：◎「電話1本で」「まずはお電話ください」
- C広島・中小企業：◎「広島の中小企業」「東広島」「広島市」「呉」
- D丸ごと：◎「丸ごとお引き受け」「丸ごと対応」「専属のIT窓口」

### 補足：方言「おらんけぇ」を入れる判断理由
- 地方中小企業社長の日常言語＝Stage1ブランディング A軸強化
- GBP説明文は「人間味」で差別化が効く（他社は事務的な文章が大半）
- リスク：標準語圏からは違和感の可能性 → **広島エリア限定発信なので問題なし**
- 1箇所のみ挿入（過剰な方言は読みづらい）

## 3. サービス登録

### 3-1. 登録方針

v4の全構造をGBPに載せると情報過多。**GBPでは「分かりやすさ優先」で再構成**：

| v4正本の構造 | GBP登録での扱い |
|---|---|
| IT困りごと診断（無料30分） | **独立サービスとして最上段に登録**（入口） |
| 安心プラン（月額16,500円） | 「ITまるごとサポート」として1本に統合 |
| まるっとおまかせ（月額27,500円） | 同上（価格は「月額16,500円〜」表記） |
| 専属パートナー（月額66,000円） | 同上 |
| デジタル経営顧問（月額110,000円） | **非表示**（営業場面のみ提示） |
| Web集客パッケージ | 独立サービスとして登録 |
| 業務デジタル化サポート | 独立サービスとして登録 |
| ITかけつけサポート（スポット） | 独立サービスとして登録 |

**非表示の理由**：
- 経営顧問はv4正本で「初回営業では出さない／LPで見た相手から聞かれた場合のみ」方針。GBPは不特定多数が閲覧するため非表示が整合。
- 3プラン内訳を出さないのはStage1 Q3決定「価格帯の詳細記載は見送り。関心喚起後に提示」に準拠。
