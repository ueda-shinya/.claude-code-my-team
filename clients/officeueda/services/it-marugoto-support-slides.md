# 後回しにしたITの困りごと、放っておくと…

<br>

| 放置していること | 起きること |
|---|---|
| ホームページが古いまま | → お客さんが競合サイトへ流れていく |
| WordPressの更新を放置 | → ある日突然、サイトが真っ白に |
| セキュリティ未対応 | → 知らないうちにウイルス感染・データ消失 |
| 相談相手がいない | → 高額な業者に言われるがまま契約してしまう |

<br>

> どれも「気づいたときにはもう遅い」ケースです。<br>でも、聞ける人がいれば、全部防げます。
