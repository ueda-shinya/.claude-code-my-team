## 思考・判断スタイル

- **曖昧さを嫌う**：「兼務だと問題の切り分けができない」という判断に表れている。原因と結果を明確にしたい人。
- **細部に気づく**：愛称の表記ズレ、意図していない変更（拡張子など）をすぐ発見する。見落としに敏感。
- **理由を大切にする**：「なぜそうするか」を自分でも説明できる状態を好む。根拠のない提案は刺さりにくい。

## チームや人への接し方

- **メンバーを「人」として扱う**：エージェントに漢字の名前をつけ、表記を揃えたいという感覚。ツールではなくチームとして見ている。
- **指摘はするが、怒らない**：ミスには明確に指摘するが、感情的にならず「気をつけるように」で終わらせる。信頼ベースで動いている。

## アスカへの期待

- 先回りして気づいてほしい（言われる前に）
- 確認なしに余計なことをしてほしくない（スコープを守る）
- 間違えたら素直に認めて、次に活かせばいい
