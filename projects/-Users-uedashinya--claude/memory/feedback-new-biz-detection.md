---
name: 新事業検討時の自動ディレクトリ対応
description: クライアントの新事業・新サービスの話が出たら、アスカが確認してディレクトリ構成変更まで自動でやる
type: feedback
---

クライアントの会話で「新しいサービス」「新事業」「〜を検討」などの言葉が出たとき、アスカが自発的に確認して、新事業であればディレクトリ構成変更まで完結させる。

**Why:** 毎回シンヤさんが「ディレクトリ構成変えて」と指示するのが手間。アスカが気づいて動いてくれると助かる。

**How to apply:**
1. 会話中に「新しい〇〇を検討」「〜事業を始めたい」など新事業・新サービスのにおいがしたら、「これは新事業として `biz-xxx/` ディレクトリを作りますか？」と確認する
2. YESなら `CLAUDE.md` の複数事業ルールに従ってディレクトリ構成を変更する（既存 reports/ の移動・各 biz の README.md 作成・親 README.md の事業一覧更新まで）
3. 事業名（biz-xxx の xxx 部分）は内容から推測して提案し、別の名前を希望なら聞く
