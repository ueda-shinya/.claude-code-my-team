## スキル・カスタマイズ共有リポジトリ

- **awesome-claude-code** (hesreallyhim): https://github.com/hesreallyhim/awesome-claude-code — スキル・フック・エージェント横断のAwesomeリスト。最重要
- **awesome-claude-skills** (travisvn): https://github.com/travisvn/awesome-claude-skills — スキル特化のキュレーションリスト
- **awesome-claude-code-toolkit** (rohitg00): https://github.com/rohitg00/awesome-claude-code-toolkit — 135エージェント、35+スキル、42コマンドなど最大規模
- **claude-code-templates** (davila7): https://github.com/davila7/claude-code-templates — WebUIで閲覧・インストール可能な100+構成
- **awesome-claude-code-plugins** (ccplugins): https://github.com/ccplugins/awesome-claude-code-plugins — プラグイン特化リスト

## CLAUDE.md テンプレート

- **claude-md-templates** (abhishekray07): https://github.com/abhishekray07/claude-md-templates — 技術スタック別テンプレート
- **Anthropic公式ベストプラクティス**: https://code.claude.com/docs/en/best-practices

## MCP サーバーディレクトリ

- **公式 MCP Registry**: https://registry.modelcontextprotocol.io/ — Anthropic公認。10,000+件
- **PulseMCP**: https://www.pulsemcp.com/servers — 最大級の非公式ディレクトリ。11,000+件
- **awesome-mcp-servers** (wong2): https://github.com/wong2/awesome-mcp-servers — 代表的なGitHubリスト
- **mcpservers.org**: https://mcpservers.org/ — awesome-mcp-serversのWeb版

## 使い方

- 全体像を把握したい → awesome-claude-code（横断的なAwesomeリスト）
- 具体的にスキルやエージェントを探したい → awesome-claude-code-toolkit（量が最大）→ awesome-claude-skills
- 既存のものがあればベースにカスタマイズするのが最速
