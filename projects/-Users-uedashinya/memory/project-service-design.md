## 背景

2026-04-12のラウンドテーブルで事業戦略を策定後、同日にサービス再設計を実施。
「何ページいくら」の制作会社メニュー → 「課題を丸ごと引き受ける」IT実務パートナーメニューへ転換。

**Why:** 下請け単価6万の構造から脱却し、直受けで月商100万を目指すには、サービスの見せ方自体を変える必要がある。制作会社として比較されるとコモディティ化する。

**How to apply:** 自社サイト・LP・名刺・営業トーク全てをこの新体系に合わせて統一する。

## 新サービス体系（2026-04-12 改訂：メイン/アップセル構造に変更）

**設計方針：ITまるごとサポート（月額）をメインの看板にし、Web集客パッケージはアップセルとして提案する。**

**理由：**
- 月額は即断できる価格帯（月2.2万）→ 営業ハードルが低い
- 毎月接点があるから信頼が積み上がり、紹介も生まれやすい
- 「中小企業のIT担当」ポジショニングと完全一致
- 信頼がある状態で55万のWeb制作を提案する方が成約率が高い

### 入口（無料）：IT困りごと診断（30分）
- 対象：「ITに困っとるけど何から手をつけたらいいかわからん」中小企業
- 内容：Web・GBP・IT環境をチェックし、課題を3つ整理して1枚で渡す
- 目的：信頼獲得。売り込まない。「この人は分かってる」と思わせるだけ

### ★メインサービス（月額2.2〜5.5万）：ITまるごとサポート
- 対象：ITの困りごとを抱えているが相談先がない中小企業
- 内容：WordPress保守+月次アクセスレポート+IT相談し放題
- 価値：「保守」ではなく「いつでも聞ける安心感」「御社のIT担当」
- 営業のしやすさ：月2.2万は経営者が即断できる。商談1-2回で契約
- 目標：12ヶ月で20社（月額ストック70万）

### アップセル①（55〜88万）：Web集客パッケージ
- 対象：月額サポート中に「ホームページちゃんとしたい」と相手から言ってきた企業
- 内容：サイト制作+GBP最適化+基本SEO+アクセス解析設定
- 売り方：信頼関係ができた状態で提案（ゼロから55万を売るのとは成約率が全然違う）
- タイミング：月額契約3ヶ月目以降。月次レポートで課題を可視化→相手から相談が来る

### アップセル②（33〜66万）：業務デジタル化サポート
- 対象：「紙・Excel・FAXでなんとかやってるけど限界」の企業
- 内容：業務フロー整理+ツール選定・導入（kintone、Google Workspace等）+社内定着サポート
- AI事業への自然な導線（「まずデジタル化→次にAIで自動化」の階段）

### 上位月額（11〜22万）：デジタル経営顧問
- 対象：「ITのことを丸ごと任せたい」従業員10名以上の企業
- 内容：月2回訪問+Web運用管理+IT相談無制限+改善提案
- ITまるごとサポートからのアップグレード先
- 6ヶ月契約〜

### スポット（1.1〜6.6万/件）：ITかけつけサポート
- 既存のWordPressトラブル対応を統合
- 接点維持が目的。ここから月額サポートへの合流導線もあり

## アセンションモデル（営業導線）

```
交流会で「ITの困りごと」を聞く
  → IT困りごと診断（無料30分）
    → ★ITまるごとサポート（月額2.2-5.5万）← メインはここ
      → 毎月の接点で信頼構築（月次レポート・相談対応）
        → Web集客パッケージ（55-88万）← 相手から相談が来る
          → デジタル経営顧問（月11-22万）← アップグレード
            → （将来）AI活用提案
```

別ルート：ITかけつけサポート（スポット）→ 信頼獲得 → 月額サポートへ合流

名刺交換→月額契約まで最短2週間〜1ヶ月。

## 営業フロー（交流会起点）

1. 交流会当日：「パソコンとかITで困ってることあります？」→ 100%話が出る。その場で1つ解決
2. 翌日：お礼メッセージ＋「診断やりますよ」
3. 1週間以内：IT困りごと診断（30分無料）→課題3つを1枚で渡す
4. 診断後すぐ：「月2万で全部面倒見れますよ」→ 月額サポート提案
5. 月額契約3ヶ月目〜：月次レポートでWebの課題を可視化→「ホームページちゃんとしたい」が出たらWeb集客パッケージ提案

## 価格の見せ方

- 比較対象を「他のWeb制作会社」ではなく「人件費」にする
- 「IT担当者を1人雇うと月25万。うちは月2.2万。年間26万で社員1人の1ヶ月分」
- Web集客パッケージは「月3件問い合わせ増→1件受注で元が取れる」のROIで提示

- LP「セルフプラン22万」は廃止（「IT実務パートナー」と「セルフでどうぞ」は矛盾）
