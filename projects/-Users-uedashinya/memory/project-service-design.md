## 新サービス体系（v4：2026-04-21 改訂）

**設計方針：ITまるごとサポート（月額）をメインの看板にし、Web集客パッケージはアップセルとして提案する。**
**主力プラン：まるっとおまかせ（竹）に誘導する松竹梅構造。**

**理由：**
- 月額は即断できる価格帯 → 営業ハードルが低い
- 毎月接点があるから信頼が積み上がり、紹介も生まれやすい
- 「中小企業のIT担当」ポジショニングと完全一致
- 信頼がある状態で55万のWeb制作を提案する方が成約率が高い

### 入口（無料）：IT困りごと診断（30分）
- 対象：「ITに困っとるけど何から手をつけたらいいかわからん」中小企業
- 内容：Web・GBP・IT環境をチェックし、課題を3つ整理して1枚で渡す
- 目的：信頼獲得。売り込まない。「この人は分かってる」と思わせるだけ

#### 「相談」と「作業」の定義

**「相談」** ＝ 口頭・メール・チャットによる質問への回答、ツールや手順の案内、画面共有での操作説明、および「やるべきか・やらないべきか」の判断助言。
**「作業」** ＝ クライアントの環境にログインし、設定変更・データ操作・ソフトウェアのインストールや設定・コードの記述や修正など、成果物が残る操作を実行すること。
IT全般相談に含まれるのは「相談」のみ。「作業」が発生する場合は月額プラン内の作業枠から充当、または超過料金。

#### 月間対応上限の運用

- 「件数」と「時間」のダブルキャップ。先に到達した方が上限
- 超過時は事前連絡の上、超過料金で対応
- 相談1回あたりの目安上限：30分。超えたら「これは作業に入りますね」と伝える

#### プラン別ターゲット

- **安心プラン**：WPサイトはあるが放置状態。まずは保守だけでも。年に数回「ちょっと聞きたい」がある
- **★まるっとおまかせ**：ITの困りごとを月額で全部任せたい。電話で「今すぐ教えて」ができると助かる
- **専属パートナー**：従業員10名以上で、月1訪問で全部見てほしい本気の企業

#### 契約条件補足

- 早期解約時の請求：最低契約期間内に解約した場合、初月特典の割引分を請求
  - 安心：無料分16,500円 / まるっと：割引分13,750円 / 専属：割引分33,000円
- 訪問対応：東広島市内は交通費込み / 広島市・呉市は別途1,100円 / 県外は実費
- スポット訪問：安心・まるっと=11,000円/回 / 専属=月1回込み、2回目以降5,500円/回

## アセンションモデル（営業導線）

```
交流会で「ITの困りごと」を聞く
  → IT困りごと診断（無料30分）
    → ★まるっとおまかせ（月2.75万）← 主力・ここに誘導
      → 毎月の接点で信頼構築（月次レポート・相談対応）
        → Web集客パッケージ（55-88万）← 相手から相談が来る
          → 専属パートナー or デジタル経営顧問 ← アップグレード
            → （将来）AI活用提案
```

別ルート：ITかけつけサポート（スポット）→ 信頼獲得 → 月額サポートへ合流
別ルート：安心プラン（入口）→ 3ヶ月後にまるっとへアップセル

名刺交換→月額契約まで最短2週間〜1ヶ月。

## 営業フロー（交流会起点）

1. 交流会当日：「パソコンとかITで困ってることあります？」→ 100%話が出る。その場で1つ解決
2. 翌日：お礼メッセージ＋「診断やりますよ」
3. 1週間以内：IT困りごと診断（30分無料）→課題3つを1枚で渡す
4. 診断後すぐ：「月2.75万で全部面倒見れますよ」→ まるっとおまかせプラン提案
5. 月額契約3ヶ月目〜：月次レポートでWebの課題を可視化→「ホームページちゃんとしたい」が出たらWeb集客パッケージ提案

## 提示戦略（LP vs 営業）

| 場 | 提示内容 |
|---|---|
| LP | 入口3プラン並列表示（安心・まるっと・専属）＋下部に「さらに手厚く」ブロックでデジタル経営顧問を別枠表示 |
| 初回営業（交流会→診断後） | **2択提示**（安心 vs まるっと）。専属はLPで見たと相手から聞かれた場合のみ説明 |
| 契約3ヶ月目以降 | 超過料金発生時に専属へのアップセル提案 |
| 専属契約3ヶ月目以降 | 超過or要望発生時にデジタル経営顧問へのアップグレード提案 |

**2択提示の理由（タク答申より）：**
- 3プラン同時提示は「持ち帰り検討」を生みやすい
- まるっとが「真ん中の妥協案」に見えると誘導力が弱まる
- 地方中小企業は選択肢が多いと意思決定が遅れる

## 価格の見せ方

- 比較対象を「他のWeb制作会社」ではなく「人件費」にする
- 「IT担当者を1人雇うと月25万。うちは月2.75万。年間33万で社員1人の1ヶ月分ちょっと」
- Web集客パッケージは「月3件問い合わせ増→1件受注で元が取れる」のROIで提示
- まるっとへの誘導：「安心プランとの差額たった1.1万で、電話OK・当日回答・レポート付き・IT全般相談OK」

## 営業トークのポイント（まるっと誘導）

1. **安心 vs まるっとの2択にする**（専属は最初から見せない）
2. 差額1.1万の小ささを強調：「1日370円で電話もレポートもIT全般もOK」
3. 専属は「聞かれたら出す」程度に留める
4. 初月半額13,750円で「まず試してみて」のリスクリバーサル
5. **最強差分「電話OK」と「IT全般相談OK」を前面に**（レン答申：地方中小企業社長にとって"文字で説明できない不安"の解消が最大の価値）
6. **「まず3ヶ月試してみて」フレーミング**（タク答申：6ヶ月縛りは初回に言わず、契約書段階で説明）

## 再検証トリガー（v4新規）
