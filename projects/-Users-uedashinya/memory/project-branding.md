## 景品表示法対策 運用ルール（2026-04-21 新設）

### 禁止表現
- 「駆けつける」「即対応」「すぐ行きます」「24時間365日」「必ず」をマーケティング媒体で使用禁止
- 訪問サービスを説明する際は、必ず「専属プランのみ／月1回訪問」を併記

### 許容表現
- 口頭の挨拶・雑談・ピッチでの「すぐ駆けつけます」は許容（記録が残らず景表法対象外）
- 「電話1本で、ITのことなら何でも」はOK（訪問を断定していない）
- 「電話・リモート・訪問の組み合わせ」はOK（サービス内容の正確な説明）

### LPでの必須表記
- v4正本の営業時間記載（平日9:00-18:00）をFAQセクションに転記
- 「24時間365日対応ではない」「休日・深夜は保証外」を明示

### 監視主体・承認フロー
- **作成**：コト（copywriter）・レン（marketing-planner）が作成
- **事前承認**：アスカが景表法観点で事前承認。NGワード検出時はコト・レンに差戻し
- **月次監査**：アスカが月次で全媒体を1回全件洗い出し（LP・名刺データ・SNSプロフィール・営業資料）

### サービス名除外ルール
- サービス名としての **「ITかけつけサポート」** は除外（ひらがな表記・固有名詞扱い・既定サービスメニューとして継続使用可）
- ただし同サービスを説明する際は「スポット対応・都度見積」を必ず併記し、「無条件即時駆けつけ」の誤認を防ぐ
